# Todo Packages

這個資料夾放「已明確要保留、但暫不開工」的整理清單。

用途：

- 收斂下一次要檢查的範圍。
- 避免把聊天中的焦慮、靈感或補強點散落在對話裡。
- 讓下一輪 AI 能快速知道哪些是待檢查項，哪些不是授權立即開工。

規則：

- 這裡不是流水帳，不記錄每日工作紀錄。
- 每份 todo package 都要有明確目的、範圍、非目標與收口方式。
- todo package 只代表候選待辦；Nick 沒明確指定前，不自動執行 flow Step、重寫履歷或大規模改 KB。
- 若 todo package 變成正式履歷 / 面試素材，才回填 `04 / 05 / 08 / 17` 或對應 flow 的 `career-interview.md`。
