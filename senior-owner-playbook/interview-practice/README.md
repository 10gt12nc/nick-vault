# Interview Practice

本資料夾記錄 Nick 的 Senior Java Backend 面試練習結果。

它不是聊天逐字稿，也不是流水帳。每輪只保留可複習、可追蹤進步的內容：

- 題目與場景。
- Nick 原始回答摘要。
- AI 評分與能力判斷。
- 會被追問打穿的點。
- 修正版回答。
- 下次要追問或補強的主題。

## 使用規則

- 每次 Nick 進行面試問答練習後，都要把有價值的結果回填到本資料夾。
- 不記錄情緒發洩或完整對話逐字稿，只記錄可複習的弱點、修正版與下一輪追問。
- 若某輪回答形成可正式使用的 90 秒 / 3 分鐘說法，再回填到 `04-interview-casebook.md` 或對應 flow 的 `career-interview.md`。
- 若某輪暴露基本功缺口，優先補到本資料夾；只有當缺口變成通用規則或題庫結構，才回填 `19-interview-coaching-question-bank.md`。
