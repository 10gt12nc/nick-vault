| 文法結構 | 中文意思 | 口語例句 | 文章 / 考試例句 |
| --- | --- | --- | --- |
| 名詞＋は | 提出主題、對比 | 私は台湾人です。（我是台灣人） | 今日は暑いです。（今天很熱） |
| 名詞＋が | 主語焦點、現象、能力 / 感情對象 | 雨が降ってるね。（下雨了呢） | 机の上に本があります。（桌上有書） |
| 名詞＋も | 也、同樣加入 | 私も行くよ。（我也去喔） | コーヒーも注文しました。（也點了咖啡） |
| 名詞＋です / だ | 是…；斷定 | これは私の本です。（這是我的書） | 彼は学生だ。（他是學生） |
| 名詞＋じゃありません / ではありません | 不是… | 私は学生じゃありません。（我不是學生） | これは私のかばんではありません。（這不是我的包包） |
| 名詞＋の＋名詞 | 所屬、內容、性質 | これは私の本です。（這是我的書） | 駅の前に銀行があります。（車站前有銀行） |
| 名詞＋を | 動作直接對象 | 朝ご飯を食べます。（吃早餐） | 図書館で本を読みました。（在圖書館看了書） |
| 場所＋で | 動作發生的場所 | カフェで勉強します。（在咖啡廳讀書） | 公園で子どもが遊んでいます。（孩子在公園玩） |
| 名詞（手段）＋で | 工具、交通、方法 | 電車で行こう。（坐電車去吧） | このペンで名前を書いてください。（請用這支筆寫名字） |
| 材料名詞＋で | 用…材料做成 | 紙で箱を作りました。（用紙做了盒子） | この机は木でできています。（這張桌子是木頭做的） |
| 理由名詞＋で | 因客觀原因 | 風邪で休みました。（因為感冒請假） | 台風で電車が止まりました。（因颱風電車停駛） |
| 狀態名詞＋で | 在…狀態 / 條件下 | 一人で行けるよ。（我可以一個人去） | この人数で入れますか。（這個人數能進去嗎） |
| 場所 / 時間 / 對象＋に | 到達點、存在點、時間點、對象 | 七時に起きます。（七點起床） | 先生に質問しました。（向老師提問） |
| 方向＋へ | 往…方向 | 明日、日本へ行きます。（明天去日本） | 駅へ向かいます。（往車站方向去） |
| 人＋と | 和…一起、互動對象 | 友だちとご飯を食べた。（和朋友吃飯） | 母と電話で話しました。（和媽媽講了電話） |
| 名詞＋と＋名詞 | 和；完整列舉 | コーヒーと紅茶をください。（請給我咖啡和紅茶） | 机の上に本とノートがあります。（桌上有書和筆記本） |
| 名詞＋や＋名詞（＋など） | …和…等；部分舉例 | パンや牛乳を買ってきて。（買麵包、牛奶之類的） | 休みの日に映画やドラマなどを見ます。（休假看電影、戲劇等等） |
| 名詞＋か＋名詞 / 普通形＋か＋普通形 | 或者；是否 | コーヒーかお茶、どっちにする？（咖啡或茶要哪個？） | 行くか行かないか、早く決めてください。（請快決定去不去） |
| 句子＋か | 疑問句尾 | これは何ですか。（這是什麼？） | 明日、学校へ行きますか。（明天去學校嗎） |
| 疑問詞＋が / を / に / で | 誰、什麼、哪裡、何時等疑問 | 何を食べる？（要吃什麼？） | 誰が来ましたか。（誰來了？） |
| なに / なん＋名詞 / 助詞 | 何的讀音分化 | 何人ですか。（幾個人？） | 何を買いましたか。（買了什麼？） |
| 起點＋から / 終點＋まで | 從…到… | 九時から働きます。（從九點開始工作） | 駅から家まで歩きました。（從車站走到家） |
| 場所に物があります | 某地有某物 | 机の上に本があります。（桌上有書） | 近くにコンビニがあります。（附近有便利商店） |
| 場所に人 / 動物がいます | 某地有人 / 動物 | 公園に猫がいます。（公園裡有貓） | 教室に先生がいます。（教室裡有老師） |
| 名詞は場所にあります / います | A 在某地 | トイレは二階にあります。（廁所在二樓） | 田中さんは会議室にいます。（田中先生在會議室） |
| 名詞があります / います | 有…；存在 | 今日は時間があります。（今天有時間） | 私には兄がいます。（我有哥哥） |
| 動詞ます形 | 禮貌現在、未來、習慣 | 毎日日本語を勉強します。（每天讀日文） | 明日、友だちに会います。（明天見朋友） |
| 動詞ません形 | 禮貌否定 | 今日は行きません。（今天不去） | この店ではカードを使いません。（這家店不使用卡片） |
| 動詞ました / ませんでした | 禮貌過去 / 過去否定 | 昨日、映画を見ました。（昨天看了電影） | 朝ご飯を食べませんでした。（沒有吃早餐） |
| 動詞て形＋ください | 請… | もう一度言ってください。（請再說一次） | ここに名前を書いてください。（請在這裡寫名字） |
| 動詞ない形＋でください | 請不要… | 忘れないでください。（請不要忘記） | ここで写真を撮らないでください。（請不要在這裡拍照） |
| 動詞て形＋います | 正在…；持續狀態；習慣 | 今、昼ご飯を食べています。（現在正在吃午餐） | 台北に住んでいます。（住在台北） |
| 動詞て形＋動詞 / 句子 | 動作連接、依序敘述 | 朝起きて、シャワーを浴びます。（早上起床後洗澡） | 名前を書いて、先生に出してください。（寫名字後交給老師） |
| 動詞て形＋もいいです | 可以… | ここに座ってもいいですか。（可以坐這裡嗎） | このアプリを使ってもいいです。（可以使用這個 App） |
| 動詞て形＋はいけません | 不可以… | ここでタバコを吸ってはいけません。（這裡不能抽菸） | この部屋に入ってはいけません。（不可進入這個房間） |
| 動詞ます形＋たいです | 想做… | ラーメンを食べたいです。（想吃拉麵） | 日本へ行きたいです。（想去日本） |
| 名詞＋がほしいです | 想要某物 | 新しいスマホがほしいです。（想要新手機） | もっと時間がほしいです。（想要更多時間） |
| 動詞辞書形＋ことができます | 能夠… | 日本語を少し話すことができます。（會說一點日文） | このカードを使うことができます。（可以使用這張卡） |
| 動詞た形＋ことがあります | 曾經…過 | 日本へ行ったことがあります。（去過日本） | 納豆を食べたことがありません。（沒吃過納豆） |
| 動詞辞書形 / ない形＋つもりです | 打算… / 不打算… | 来年、日本へ行くつもりです。（明年打算去日本） | 今日は外に出ないつもりです。（今天不打算出門） |
| 動詞ない形＋なければなりません | 必須… | 薬を飲まなければなりません。（必須吃藥） | 明日までに宿題を出さなければなりません。（明天前必須交作業） |
| 動詞ない形＋なくてもいいです | 不…也可以 | 明日は来なくてもいいです。（明天不來也可以） | 全部食べなくてもいいです。（不用全部吃完也可以） |
| 動詞ませんか | 要不要…；邀約 | 一緒に昼ご飯を食べませんか。（要不要一起吃午餐？） | 今週の日曜日、映画を見ませんか。（這週日要不要看電影？） |
| 動詞ましょう | 一起…吧；提議 | 行きましょう。（走吧） | 休み時間に少し話しましょう。（休息時間聊一下吧） |
| 動詞ましょうか | 我來…嗎；主動提議 | 手伝いましょうか。（我來幫忙嗎？） | 窓を開けましょうか。（要我開窗嗎？） |
| 動詞ます形＋に行く / 来る | 去 / 來做… | ご飯を食べに行こう。（去吃飯吧） | 図書館へ本を借りに行きます。（去圖書館借書） |
| 動詞辞書形＋まえに | 在…之前 | 寝るまえに歯を磨きます。（睡前刷牙） | 旅行のまえにホテルを予約します。（旅行前預約旅館） |
| 動詞た形＋あとで | 做完…之後 | ご飯を食べたあとで散歩します。（吃完飯後散步） | 宿題をしたあとでテレビを見ます。（寫完作業後看電視） |
| 動詞て形＋から | 做完…之後再… | 手を洗ってから食べます。（洗手後再吃） | 仕事が終わってから電話します。（工作結束後打電話） |
| 普通形＋から | 因為…所以… | 雨だから、家にいる。（因為下雨，所以待在家） | 時間がありませんから、急いでください。（因為沒有時間，請快一點） |
| 普通形＋ので | 因為…所以…；較客觀柔和 | 今日は仕事なので、早く寝ます。（今天要上班，所以早睡） | 電車が遅れたので、遅刻しました。（因電車誤點而遲到） |
| 普通形＋けど / が | 雖然…；但是…；鋪墊 | 高いけど、おいしいです。（雖然貴，但是好吃） | 行きたいですが、時間がありません。（想去，但是沒有時間） |
| 動詞たり＋動詞たりします | 有時…有時…；列舉活動 | 週末は映画を見たり本を読んだりします。（週末看電影、看書等等） | 休みの日は掃除したり買い物したりします。（休假會打掃、買東西等等） |
| 動詞ます形＋ながら | 一邊…一邊… | 音楽を聞きながら勉強します。（一邊聽音樂一邊讀書） | 歩きながら電話しないでください。（請不要邊走邊講電話） |
| い形容詞＋です | 很…；是…的 | この店はおいしいです。（這家店很好吃） | 今日は暑いです。（今天很熱） |
| い形容詞＋くないです / かったです | 不… / 過去很… | この服は高くないです。（這件衣服不貴） | 昨日は寒かったです。（昨天很冷） |
| な形容詞＋です / な＋名詞 | 很…；…的名詞 | この町は静かです。（這個城鎮很安靜） | 静かなカフェで勉強します。（在安靜的咖啡廳讀書） |
| い形容詞 / な形容詞＋名詞 | 修飾名詞 | 安い店を知ってる？（你知道便宜的店嗎？） | 便利な駅の近くに住んでいます。（住在方便的車站附近） |
| 形容詞＋て / で | 而且…；因為… | この店は安くておいしいです。（這家店便宜又好吃） | 部屋は静かで明るいです。（房間安靜又明亮） |
| い形＋くなる / な形＋になる | 變得… | 最近、寒くなりました。（最近變冷了） | 日本語が少し上手になりました。（日文稍微變好了） |
| い形＋くする / な形＋にする / 名詞＋にする | 使…變得…；決定為… | 音を小さくしてください。（請把聲音調小） | コーヒーにします。（我要咖啡） |
| AはBより〜です | A 比 B 更… | 電車はバスより速いです。（電車比公車快） | 東京は大阪より人が多いです。（東京比大阪人多） |
| AよりBのほうが〜です | 比起 A，B 更… | コーヒーよりお茶のほうが好きです。（比起咖啡，更喜歡茶） | 駅より家のほうが静かです。（比起車站，家裡更安靜） |
| AとBとどちらが〜ですか | A 和 B 哪個更… | 犬と猫とどちらが好き？（狗和貓喜歡哪個？） | 日本語と英語とどちらが難しいですか。（日文和英文哪個難？） |
| 範圍＋で＋いちばん | 在…之中最… | 季節の中で春がいちばん好きです。（季節中最喜歡春天） | この店でこの料理がいちばん人気です。（這家店這道菜最受歡迎） |
| あまり＋否定 | 不太… | 辛い食べ物はあまり好きじゃありません。（不太喜歡辣的食物） | 最近あまり運動していません。（最近不太運動） |
| 全然＋否定 | 完全不… | 全然分かりません。（完全不懂） | 今日は全然寒くないです。（今天完全不冷） |
| もっと | 更；再多一點 | もっとゆっくり話してください。（請說慢一點） | もっと日本語を勉強したいです。（想更努力學日文） |
| もう＋動詞 / もう＋否定 | 已經…；不再… | もう食べたよ。（已經吃了） | もう時間がありません。（已經沒有時間了） |
| まだ＋動詞 / まだ＋否定 | 還…；還沒… | まだ起きてる？（還醒著嗎？） | まだ宿題が終わっていません。（作業還沒做完） |
| 名詞 / 數量＋だけ | 只有…；只… | これだけください。（只要這個） | 一つだけ質問があります。（只有一個問題） |
| 數量＋ぐらい / くらい | 大約… | 十分ぐらい待って。（等大約十分鐘） | 駅まで歩いて五分ぐらいです。（走到車站大約五分鐘） |
| 〜ね / 〜よ / 〜よね | 共鳴、提醒、確認 | 今日はいい天気ですね。（今天天氣很好呢） | 明日来ますよね。（明天會來吧） |
| 疑問詞＋でも | 無論…都 | 誰でも参加できます。（誰都可以參加） | 何でも聞いてください。（什麼都可以問） |
| 疑問詞＋も＋否定 | 全面否定 | 部屋には誰もいません。（房間裡誰都不在） | 今日は何も食べていません。（今天什麼都沒吃） |
| 名詞＋しか＋否定 | 只有…；強調少 | 千円しかありません。（只有一千日圓） | 今日は一時間しか勉強できません。（今天只能讀一小時） |
| 普通形＋と思います | 我覺得…；認為… | 明日は雨だと思います。（我覺得明天會下雨） | この方法はいいと思います。（我認為這個方法好） |
| 普通形＋と言います | 說…；叫做… | 彼は明日行くと言いました。（他說明天要去） | これは日本語で何と言いますか。（這個用日文怎麼說？） |
| 普通形＋んです / のです | 說明背景、追問理由 | どうしたんですか。（怎麼了？） | 今日は少し疲れているんです。（今天有點累） |
| 普通形＋でしょう | 大概…吧；對吧 | 明日は晴れるでしょう。（明天大概會放晴吧） | 駅はあちらでしょう。（車站大概在那邊吧） |
| 動詞た形 / ない形＋ほうがいいです | 最好… / 最好不要… | 早く寝たほうがいいです。（最好早點睡） | 無理しないほうがいいです。（最好不要勉強） |
| 場所＋を＋移動動詞 | 經過 / 離開某地 | 朝、公園を歩きます。（早上走過公園） | 七時に家を出ました。（七點離開家） |
| 動詞ても / 形容詞ても | 即使…也… | 雨が降っても行きます。（即使下雨也去） | 高くても買いたいです。（即使貴也想買） |
| 名詞 / な形＋なら | 如果是…；承接話題 | 明日なら大丈夫です。（如果是明天就可以） | 静かな店なら行きたいです。（如果是安靜的店就想去） |
| 普通形＋時 | …的時候 | 食べる時、連絡してください。（吃的時候請聯絡） | 日本へ行った時、写真をたくさん撮りました。（去日本時拍了很多照片） |
