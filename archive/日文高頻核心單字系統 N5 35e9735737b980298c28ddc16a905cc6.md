# 日文高頻核心單字系統 N5

分層原則

| 層級 | 目標 | 學習方式 |
| --- | --- | --- |
| A 主動輸出核心 | 生存、聊天、旅行、日常必用 | 要會聽、會說、會造句 |
| B 高頻理解核心 | 文章、標示、生活資訊常見 | 要能讀懂、聽懂 |
| C 閱讀辨識 | N5 內較低頻或情境限定 | 先認得，不急著大量輸出 |

標籤

日常高頻：聊天、生活、旅行、購物最常見。

書面高頻：通知、文章、說明、教材常見。

新聞常見：基礎新聞或生活資訊可見。

偏低頻／閱讀辨識：N5 可見，但主動輸出優先度較低。
