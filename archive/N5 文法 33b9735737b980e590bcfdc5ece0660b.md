# N5 文法

# 基礎結構（句子骨幹）

## ～が（主語／對象）

```
◆ 語感例句結構
・焦點主語標記
・名詞＋が＋述語
・雨が降ってるから、今日は家にいよう。（因為在下雨，今天待在家）
・彼が来た瞬間、空気が変わった。（他一來，氣氛就變了）

◆ 混淆比較
・は／を
・が＝新資訊焦點；は＝主題對比；を＝動作對象
・彼が来た（是「他」來了） vs 彼は来た（他來了〈對比他人〉）
・水が飲みたい（想喝水〈水是焦點〉） vs 水を飲みたい（單純想喝水）

◆ 錯誤陷阱
・已知主題卻誤用が（應用は）
・私は学生がです（×）
・私は学生です（✓ 我是學生）

◆ 變化補充
・疑問詞搭配
・誰／何＋が → 必用が，答案對應焦點
・實戰建議：看到「誰／何」先選が，再看句尾
```

## ～は（主題）

```
◆ 語感例句結構（必出）
・語感：拿來當話題
・結構：N は ＋ 評論／描述
・今日は暑い。→ 今天很熱（今天當話題）
・私はコーヒーは飲まない。→ 我不喝咖啡（我這個人）

◆ 混淆比較（必出｜選項核心）
・對手文法：が／も
・一句話差異：は＝對比話題；が＝新資訊主語；も＝也包含
・私は行く。vs 私が行く。→ 我會去（話題）vs 是我去（強調誰）
・コーヒーは飲む。vs コーヒーも飲む。→ 咖啡我會喝 vs 咖啡我也喝

◆ 錯誤陷阱（必出｜考試重點）
・NG：把「誰做」誤用は（應用が）
・× 誰は来た？
・✓ 誰が来た？（誰來了？）

◆ 變化補充（可省略）
・分類：對比用は（隱含比較）
・影響答案的理由：句中常有「其他不一樣」
・實戰建議：看到對比或轉折，優先選は
```

## ～も（也）

```
◆ 語感例句結構（必出）
・語感：追加同類
・結構：名詞／句子＋も
・私も行くよ。（我也去喔）
・コーヒーも頼んだ。（咖啡也點了）

◆ 混淆比較（必出｜選項核心）
・は／が
・も＝加進同類；は＝對比提示；が＝新資訊主語
・私は行くけど、彼は行かない。（我去，但他不去）
　→私も行く。（我也去）
・誰が来る？→田中さんが来る。（誰來？→田中來）
　→田中さんも来る。（田中也來）

◆ 錯誤陷阱（必出｜考試重點）
・已有主題再疊加「は＋も」錯
・× 私はも行く。
・✓ 私も行く。（我也去）

◆ 變化補充（可省略）
・否定句
・も可搭配否定形成「全部不」
・誰も来ない（誰都沒來）→注意語氣是全否
```

## 名詞＋です／だ

```
◆ 語感例句結構
・斷定＝直接下結論
・名詞＋です／だ
・これは私の本です。（這是我的書）
・彼はエンジニアだ。（他是工程師）

◆ 混淆比較
・～です／だ vs ～である／～になります
・です＝單純斷定；である＝書面強調；になります＝變化結果
・彼は先生です。（他是老師） vs 彼は先生になります。（他會成為老師）
・これは事実だ。（這是事實） vs これは事実である。（此為事實／較正式）

◆ 錯誤陷阱
・名詞句卻用動詞形（多餘或錯）
・彼は学生します。（×）
・彼は学生です。（✓ 他是學生）

◆ 變化補充
・體裁差異
・だ（普通體）／です（禮貌體）會影響句尾選擇
・考試看到敬體句尾，優先選です
```

# 格助詞（核心用法）

## 名詞＋を（目的語）

```
◆ 語感例句結構（必出）
・動作直接對象
・名詞＋を＋動詞
・コーヒーを飲む。
　（喝咖啡）
・メールを送った。
　（寄了郵件）

◆ 混淆比較（必出｜選項核心）
・が／に
・功能差：を＝受詞；が＝主語；に＝對象/到達點
・私は寿司を食べる。
　（我吃壽司）vs 寿司が好きだ。
　（喜歡壽司）
・彼に会う。
　（見他）vs 彼を呼ぶ。
　（叫他）

◆ 錯誤陷阱（必出｜考試重點）
・自動詞誤用：不可亂用を
・× 電車を着く。
・✓ 電車に着く。（到達用に）

◆ 變化補充（可省略）
・特殊用法
・場所を＋移動動詞（道を歩く）
・看到「動作影響對象」優先選
```

## 場所＋に（存在／到達／時間點）

```
◆ 語感例句結構（必出）
・定位到達時點
・場所／時間＋に＋存在／移動動詞
・駅に着いた。
　（到達車站）
・部屋に猫がいる。
　（房間裡有貓）

◆ 混淆比較（必出｜選項核心）
・で／へ
・功能差：に＝到達/存在點；で＝動作地；へ＝方向
・公園で遊ぶ。
　（在公園玩）vs 公園に行く。
　（去公園）
・学校へ行く。
　（往學校）vs 学校に着く。
　（到達學校）

◆ 錯誤陷阱（必出｜考試重點）
・誤用動作地：不可用に表「做事地點」
・× カフェに勉強する。
・✓ カフェで勉強する。（在咖啡廳讀書）

◆ 變化補充（可省略）
・時間用法
・時間點也用に（3時に会う）
・看到「到達／存在／時間點」優先選
```

## 場所＋で（動作發生地）

```
◆ 語感例句結構（必出）
・動作發生地點
・場所＋で＋動作動詞
・カフェで勉強する。
　（在咖啡廳讀書）
・公園で子どもたちが遊んでいる。
　（孩子們在公園玩）

◆ 混淆比較（必出｜選項核心）
・に／を
・功能差：で＝做事地點；に＝存在/到達；を＝移動經過
・部屋に猫がいる。
　（存在）vs 部屋で勉強する。
　（做事地點）
・道を歩く。
　（沿路走）vs 道で立ち話する。
　（在路上聊天）

◆ 錯誤陷阱（必出｜考試重點）
・存在誤用：不可用で表「存在」
・× 部屋で猫がいる。
・✓ 部屋に猫がいる。（存在用に）

◆ 變化補充（可省略）
・語意關鍵
・強調「行為發生的位置」
・看到「做某事的地方」優先選
```

## 場所・方向＋へ（に）

```
◆ 語感例句結構（必出）
・方向目標移動
・場所／方向＋へ（に）＋移動動詞
・学校へ行く。
　（往學校去）
・家に帰る。
　（回家）

◆ 混淆比較（必出｜選項核心）
・に／で
・語感差：へ＝方向；に＝到達；で＝動作地點
・駅へ向かう。
　（朝車站方向）vs 駅に着く。
　（到車站）
・公園で遊ぶ。
　（在公園玩）vs 公園へ行く。
　（往公園去）

◆ 錯誤陷阱（必出｜考試重點）
・誤用動作地：不可用へ表「做事地點」
・× カフェへ勉強する。
・✓ カフェで勉強する。（在咖啡廳讀書）

◆ 變化補充（可省略）
・互換差異
・へ↔に多可互換（行く／帰る）
・強調「方向」優先へ
```

## 對象（人／物）＋に（對象／授受）

```
◆ 語感例句結構（必出）
・對象接收作用
・人／物＋に＋動詞（あげる／くれる／もらう 等）
・友だちにプレゼントをあげた。
　（送禮物給朋友）
・先生に質問した。
　（向老師提問）

◆ 混淆比較（必出｜選項核心）
・を／へ
・功能差：に＝作用對象；を＝直接受詞；へ＝方向
・彼に電話する。
　（打給他）vs 電話をかける。
　（打電話這個動作）
・友だちへ手紙を書く。
　（往朋友寫信）vs 友だちに手紙を書く。
　（寫信給朋友）

◆ 錯誤陷阱（必出｜考試重點）
・授受混亂：對象不可用を
・× 彼をプレゼントをあげた。
・✓ 彼にプレゼントをあげた。（給他禮物）

◆ 變化補充（可省略）
・語意延伸
・に也可表「對象反應」（彼に怒る）
・看到「作用對象／給誰」優先選
```

## 起點（人）＋から

```
◆ 語感例句結構（必出）
・來源起點給予
・人＋から＋動詞（もらう／聞く 等）
・友だちからプレゼントをもらった。
　（從朋友那裡收到禮物）
・先生から大事な話を聞いた。
　（從老師那裡聽到重要的事）

◆ 混淆比較（必出｜選項核心）
・に／より
・來源差：から＝一般來源；に＝授受對象（くれる）；より＝書面來源
・先生から本をもらった。
　（從老師得到）vs 先生に本をもらった。
　（向老師得到）
・会社よりご連絡いたします。
　（書面）vs 会社から連絡します。
　（一般說法）

◆ 錯誤陷阱（必出｜考試重點）
・授受混亂：給予方不可用に
・× 友だちにプレゼントをもらった。
・✓ 友だちからプレゼントをもらった。（從朋友那裡收到）

◆ 變化補充（可省略）
・語境差異
・正式通知偏用より
・看到「來源／從誰」優先選から
```

## 離開點＋を

```
◆ 語感例句結構（必出）
・離開通過場所
・場所＋を＋移動動詞（出る／離れる／通る）
・家を出る。
　（離開家）
・公園を通って駅に行く。
　（穿過公園去車站）

◆ 混淆比較（必出｜選項核心）
・から／で
・語感差：を＝離開/經過；から＝起點來源；で＝動作地點
・家から出る。
　（從家出來）vs 家を出る。
　（離開家）
・公園で遊ぶ。
　（在公園玩）vs 公園を通る。
　（穿過公園）

◆ 錯誤陷阱（必出｜考試重點）
・存在誤用：不可用於靜態存在
・× 公園をいる。
・✓ 公園にいる。（存在用に）

◆ 變化補充（可省略）
・語意延伸
・也可表「經過路線」
・看到「離開／通過」優先選
```

## 通過・移動＋を＋自動詞

```
◆ 語感例句結構（必出）
・經過路線移動
・場所＋を＋自動詞（歩く／渡る／通る）
・橋を渡って学校に行く。
　（過橋去學校）
・道を歩いて帰った。
　（沿著路走回家）

◆ 混淆比較（必出｜選項核心）
・で／に
・語感差：を＝路線經過；で＝動作地；に＝到達點
・公園で走る。
　（在公園跑）vs 公園を走る。
　（穿過公園跑）
・駅に着く。
　（到車站）vs 駅を通る。
　（經過車站）

◆ 錯誤陷阱（必出｜考試重點）
・他動詞混淆：不可誤判為受詞
・× 道を行く（當成受詞理解）
・✓ 道を歩く。（表示經過路線）

◆ 變化補充（可省略）
・關鍵辨識
・多搭配移動類自動詞
・看到「經過路線」優先選
```

## ～から～まで

```
◆ 語感例句結構（必出）
・起點到終點範圍
・N／時間＋から～まで
・9時から5時まで働く。
　（從9點工作到5點）
・家から駅まで歩いた。
　（從家走到車站）

◆ 混淆比較（必出｜選項核心）
・から／まで／までに
・範圍差：から～まで＝整段範圍；まで＝終點；までに＝期限
・9時から5時まで勉強する。
　（整段時間）vs 5時まで勉強する。
　（做到5點為止）
・5時までに終わらせる。
　（5點前完成）vs 5時まで終わらせる。
　（做到5點）

◆ 錯誤陷阱（必出｜考試重點）
・期限混用：不可把まで當截止時間
・× 5時まで宿題を出してください。
・✓ 5時までに宿題を出してください。（5點前交）

◆ 變化補充（可省略）
・語意延伸
・場所／時間皆可用
・看到「範圍（起～終）」優先選
```

## 名詞＋の（修飾／所屬）

```
◆ 語感例句結構（必出）
・名詞連接修飾
・名詞＋の＋名詞
・これは私の本です。
　（這是我的書）
・日本の文化が好きだ。
　（喜歡日本的文化）

◆ 混淆比較（必出｜選項核心）
・が／を
・功能差：の＝修飾/所屬；が＝主語；を＝受詞
・私の本です。
　（我的書）vs 私が読む本です。
　（我讀的書）
・彼の写真を見る。
　（看他的照片）vs 写真を見る。
　（看照片）

◆ 錯誤陷阱（必出｜考試重點）
・誤當主語：不可用の取代が（一般句）
・× 私の行く。
・✓ 私が行く。（我去）

◆ 變化補充（可省略）
・語意延伸
・可表多種關係（所屬／性質／內容）
・看到「名詞修飾名詞」優先選
```

# 手段・狀態・理由

## 方法・手段＋で

```
◆ 語感例句結構（必出）
・語感：工具手段
・結構：名詞（方法・手段）＋で
・このペンで書くのが一番楽だよ。（用這支筆寫最輕鬆）
・電車で行けば、30分で着くよ。（用電車去30分鐘就到）

◆ 混淆比較（必出｜選項核心）
・對手文法：で vs に / を使って
・一句話差異：で＝手段本身／に＝到達點／を使って＝刻意強調「使用」
・電車で行く。（手段）vs 駅に行く。（目的地）
　用電車去 vs 去車站
・ナイフで切る。（自然手段）vs ナイフを使って切る。（刻意強調用刀）
　用刀切 vs 使用刀子來切

◆ 錯誤陷阱（必出｜考試重點）
・NG用法說明：把「場所」或「對象」誤用成手段
・学校で行く（×）
・学校に行く（✓ 去學校）

◆ 變化補充（可省略）
・分類：抽象手段也可用
・影響答案的理由：是否可視為「工具／方式」
・實戰建議：看到交通、工具、方法幾乎優先選で
```

## 材料＋で

```
◆ 語感例句結構（必出）
・語感：材料構成
・結構：材料名詞＋で
・この机は木でできている。（這張桌子是用木頭做的）
・紙で箱を作ったよ。（用紙做了盒子）

◆ 混淆比較（必出｜選項核心）
・對手文法：で vs から
・一句話差異：で＝材料外觀仍可辨／から＝完全變質
・木でできた家。（看得出是木頭）vs ワインはぶどうから作られる。（變成另一種東西）
　木頭做的房子 vs 葡萄釀成葡萄酒
・紙で作った花。（還是紙）vs チーズは牛乳からできる。（已非牛奶）
　紙做的花 vs 牛奶做成起司

◆ 錯誤陷阱（必出｜考試重點）
・NG用法說明：已完全變質卻誤用で
・ワインはぶどうで作られる（×）
・ワインはぶどうから作られる（✓ 葡萄釀成酒）

◆ 變化補充（可省略）
・分類：できている常搭配
・影響答案的理由：是否保留原材料形態
・實戰建議：看得出原料→で；完全變化→から
```

## 理由＋で

```
◆ 語感例句結構（必出）
・語感：客觀原因
・結構：理由名詞＋で
・台風で電車が止まった。（因為颱風電車停了）
・仕事で遅くなった、ごめん。（因為工作晚了）

◆ 混淆比較（必出｜選項核心）
・對手文法：で vs から / ので
・一句話差異：で＝客觀結果原因／から＝主觀理由／ので＝較禮貌解釋
・事故で遅れた。（客觀事故）vs 寝坊したから遅れた。（自己原因）
　因事故遲到 vs 因睡過頭遲到
・雨で中止になった。（自然結果）vs 雨なので中止します。（禮貌說明）
　因雨取消 vs 因為下雨所以取消（較客氣）

◆ 錯誤陷阱（必出｜考試重點）
・NG用法說明：主觀意志句誤用で
・疲れで帰ります（×）
・疲れたので帰ります（✓ 因為累所以要回去）

◆ 變化補充（可省略）
・分類：多接不可控制原因
・影響答案的理由：句尾是否為意志／命令
・實戰建議：有「我要／請」→不用で，改用から／ので
```

## 狀態・情況＋で

```
◆ 語感例句結構（必出）
・語感：狀態條件
・結構：狀態名詞＋で
・一人で全部やったよ。（一個人把全部做完）
・この人数で入れるかな？（這個人數能進去嗎）

◆ 混淆比較（必出｜選項核心）
・對手文法：で vs に / のまま
・一句話差異：で＝條件下行動／に＝結果狀態／のまま＝維持不變
・一人で行く。（在一人狀態下行動）vs 一人になる。（變成一個人）
　一個人去 vs 變成一個人
・そのままでいいよ。（維持原樣）vs この状態で進める。（用這狀態做）
　保持原樣 vs 用這狀態繼續

◆ 錯誤陷阱（必出｜考試重點）
・NG用法說明：把「結果變化」誤用で
・静かでなった（×）
・静かになった（✓ 變安靜了）

◆ 變化補充（可省略）
・分類：數量／人數常見
・影響答案的理由：是否描述「在此條件下做」
・實戰建議：能接「做某事」→で；描述變化→に
```

## 名詞＋と＋名詞

```
◆ 語感例句結構（必出）

◆ 混淆比較（必出｜選項核心）

・一句話差異：と＝全部列舉完整／や・とか＝部分舉例未列完

```

## 對象＋と（一起／比較）

```
◆ 語感例句結構（必出）

## 引用內容＋と

```

# 副助詞・限定

## だけ

```
◆ 語感例句結構
・語感：就這個量
・結構：名詞＋だけ／普通形＋だけ
・水だけ飲んでる。／我只喝水。
・言いたいことだけ言って帰った。／他只把想說的講完就回去了。

◆ 混淆比較
・對手文法：しか〜ない／のみ
・一句話差異：だけ＝單純限定；しか〜ない＝少到帶不足感；のみ＝更書面
・千円だけ持ってる。vs 千円しか持ってない。／我只有一千圓。前者中性，後者有「就這麼少」的感覺。
・参加者だけ入れます。vs 参加者のみ入れます。／只有參加者能進。後者更像公告或規則。

◆ 錯誤陷阱
・NG用法說明：だけ後面不要再硬接否定來表「只有」
・× 水だけしか飲まない。
・✓ 水しか飲まない。／我只喝水。　✓ 水だけ飲む。／我只喝水。

◆ 變化補充
・分類：だけで／だけあって
・影響答案的理由：だけで是「光靠～就」；だけあって是「不愧是～」
・實戰建議：看到「只、僅」先想だけ；看到「少到可惜」再選しか〜ない
```

## しか＋否定

```
◆ 語感例句結構
・語感：少到可惜
・結構：名詞＋しか＋否定（ない）
・千円しか持ってない。／我只有一千圓（太少）。
・水しか飲まない。／我只喝水（別的都不喝）。

◆ 混淆比較
・對手文法：だけ／ばかり
・一句話差異：しか〜ない＝帶不足感；だけ＝中性限定；ばかり＝偏多/偏向
・千円しかない。vs 千円だけある。／只有一千圓。前者嫌少，後者中性。
・甘い物ばかり食べてる。vs 甘い物しか食べてない。／前者是「一直吃很多」，後者是「只吃這個」。

◆ 錯誤陷阱
・NG用法說明：しか必須搭配否定（不能用肯定）
・× 千円しかある。
・✓ 千円しかない。／只有一千圓（太少）。

◆ 變化補充
・分類：しか〜ない（固定搭配）
・影響答案的理由：句尾一定是否定，看到肯定直接排除
・實戰建議：題目有「只有而且嫌少」優先選しか〜ない
```

## ずつ

```
◆ 語感例句結構
・語感：平均分配
・結構：數量詞＋ずつ
・一人に二枚ずつ配った。／每人分到兩張。
・毎日30分ずつ勉強してる。／每天一點一點讀30分鐘。

◆ 混淆比較
・對手文法：ごとに／おきに
・一句話差異：ずつ＝平均分給每個/每次；ごとに＝每隔就發生；おきに＝隔一個間隔
・一人に二枚ずつ配る。vs 一人ごとに配る。／前者強調「每人兩張」，後者只強調「每個人都發」。
・5分ずつ休む。vs 5分おきに休む。／前者是「每次休5分」，後者是「每隔5分休一次」。

◆ 錯誤陷阱
・NG用法說明：ずつ不能直接接名詞（要接數量詞）
・× 人ずつ来る。
・✓ 二人ずつ来る。／每次來兩個人。

◆ 變化補充
・分類：時間／人數分配
・影響答案的理由：看是否有「每個人/每次平均量」的概念
・實戰建議：看到「每人…幾個」「每次…多少」直接選ずつ
```

## ぐらい／くらい

```
◆ 語感例句結構
・語感：大約程度
・結構：名詞／數量詞／普通形＋ぐらい（くらい）
・あと10分ぐらいで着く。／大概再10分鐘會到。
・泣きたいくらい疲れてる。／累到想哭的程度。

◆ 混淆比較
・對手文法：ほど／だけ
・一句話差異：ぐらい＝主觀大概；ほど＝客觀程度比較；だけ＝明確限定
・10分ぐらい待った。vs 10分ほど待った。／都可說約10分，但ほど較正式/客觀。
・死ぬくらい痛い。vs 死ぬほど痛い。／後者更強烈、更誇張。

◆ 錯誤陷阱
・NG用法說明：ぐらい不能用來表明確上限（不能當「最多」）
・× 3人ぐらいしか入れない。（想表最多3人）
・✓ 3人までしか入れない。／最多只能進3人。

◆ 變化補充
・分類：くらい＝書面感稍強
・影響答案的理由：兩者基本可互換，不影響選項
・實戰建議：口語隨便選ぐらい，正式文可用くらい
```

## ～には、～へは、～とは

```
◆ 語感例句結構
・語感：對比強調點
・結構：名詞＋には／へは／とは
・日本には行ったことあるけど、韓国にはない。／日本去過，但韓國沒有。
・彼とは話すけど、あの人とは話さない。／會跟他聊，但不跟那個人聊。

◆ 混淆比較
・對手文法：に／へ／と（無は）
・一句話差異：加は＝拉出來對比；不加＝單純敘述
・日本に行く。vs 日本には行く。／前者只是去，後者有「別的地方不一定去」的對比。
・彼と話す。vs 彼とは話す。／前者單純描述，後者有「至少會跟他」的對比。

◆ 錯誤陷阱
・NG用法說明：は會改變語氣（變對比），不是可有可無
・× 先生には会いました。（想單純說見過老師）
・✓ 先生に会いました。／我見過老師（單純事實）。

◆ 變化補充
・分類：主題化＋對比
・影響答案的理由：題目有「但是／反而／至少」常暗示用は
・實戰建議：看到句子有對比或轉折，優先考慮加は
```

## ～にも、～からも、～でも

```
◆ 語感例句結構
・語感：也連同
・結構：名詞＋にも／からも／でも
・彼にも連絡した。／也聯絡了他。
・友達からもプレゼントをもらった。／也從朋友那邊收到了禮物。

◆ 混淆比較
・對手文法：に／から／で（無も）
・一句話差異：も＝加上「也」的並列感；不加＝單純來源/對象
・彼に連絡した。vs 彼にも連絡した。／前者只聯絡他，後者是「他也有被聯絡」。
・店で買った。vs 店でも買った。／前者單純在店買，後者是「店裡也有買（其他地方也買）」。

◆ 錯誤陷阱
・NG用法說明：も會暗示「不只一個」，不能用在唯一對象
・× 日本にも一つしかない。／（矛盾：又說唯一又說也）
・✓ 日本に一つしかない。／日本只有一個。

◆ 變化補充
・分類：追加／並列
・影響答案的理由：句中若有「也、連～都」語感會觸發も
・實戰建議：看到前面已出現一個對象，後面再補就用も
```

# 疑問與不定

## なに／なん

```
◆ 語感例句結構
・語感：發音避拗
・結構：なに／なん＋名詞／助詞
・何を食べる？（なに）／你要吃什麼？
・何人ですか？（なんにん）／幾個人？

◆ 混淆比較
・對手文法：なに vs なん
・一句話差異：なん＝後面接N行或為了好發音；なに＝基本原形
・何をする？（なに）vs 何人？（なんにん）／助詞前多用なに，名詞前多用なん
・何で行く？（なんで）vs 何を買う？（なにを）／で前用なん，を前用なに

◆ 錯誤陷阱
・NG用法說明：不能全部都用なに（會發音卡）
・× なに人ですか。
・✓ なん人ですか。／幾個人？

◆ 變化補充
・分類：音便規則
・影響答案的理由：看到後面是N行（に・の・な等）多半用なん
・實戰建議：考試先看後面音：順口＝なん，不順＝なに
```

## だれ／どなた

```
◆ 語感例句結構
・語感：禮貌等級差
・結構：だれ／どなた
・あの人はだれ？／那個人是誰？
・どなた様ですか？／請問您是哪位？

◆ 混淆比較
・對手文法：だれ vs どなた
・一句話差異：どなた＝對對方更禮貌；だれ＝一般用
・だれが来るの？vs どなたがいらっしゃいますか？／前者隨便問，後者對客戶或長輩
・あの人はだれ？vs あの方はどなたですか？／後者更尊敬、正式

◆ 錯誤陷阱
・NG用法說明：對長輩/客戶用だれ會失禮
・× 社長はだれですか。
・✓ 社長はどなたですか。／社長是哪位？

◆ 變化補充
・分類：敬語層級
・影響答案的理由：句子有「ですか／いらっしゃる」等敬語，通常搭どなた
・實戰建議：看到正式場合（店員、客戶），直接選どなた
```

## いつ

```
◆ 語感例句結構
・語感：時間未知問
・結構：いつ＋動詞
・いつ来るの？／你什麼時候來？
・次の休みはいつ？／下次休假是什麼時候？

◆ 混淆比較
・對手文法：何時（なんじ）／どのくらい
・一句話差異：いつ＝問時間點（廣泛）；なんじ＝問幾點；どのくらい＝問時間長度
・いつ来る？vs 何時に来る？／前者不限範圍，後者是問幾點
・どのくらい待つ？vs いつまで待つ？／前者問多久，後者問到什麼時候

◆ 錯誤陷阱
・NG用法說明：いつ不能直接表「多久」
・× いつ時間かかる？
・✓ どのくらい時間かかる？／要花多久？

◆ 變化補充
・分類：疑問詞（時間）
・影響答案的理由：看到「幾點」用なんじ，「多久」用どのくらい
・實戰建議：不確定具體時間→いつ；精確時間→なんじ
```

## どこ

```
◆ 語感例句結構
・語感：地點未知問
・結構：どこ＋助詞（に／で／へ 等）
・どこに行くの？／你要去哪裡？
・どこで買ったの？／你在哪裡買的？

◆ 混淆比較
・對手文法：どちら／どのへん
・一句話差異：どこ＝單純問地點；どちら＝較禮貌或二選一；どのへん＝大概區域
・どこに住んでる？vs どちらにお住まいですか？／後者更禮貌
・どこ？vs どのへん？／前者要精確位置，後者是大概哪一帶

◆ 錯誤陷阱
・NG用法說明：助詞要配動詞（に＝目的地／で＝動作地）
・× どこに買ったの？
・✓ どこで買ったの？／在哪裡買的？

◆ 變化補充
・分類：疑問詞（地點）
・影響答案的理由：看到「移動」選に／へ，「行為」選で
・實戰建議：先判斷動詞是「去」還是「做」，再選助詞
```

## いくつ

```
◆ 語感例句結構
・語感：數量未知問
・結構：いくつ（＋名詞）
・りんご、いくつ買う？／蘋果要買幾個？
・子どもはいくつ？／小孩幾歲？

◆ 混淆比較
・對手文法：何個（なんこ）／何歳（なんさい）
・一句話差異：いくつ＝泛用（數量/年齡）；なんこ/なんさい＝明確單位
・りんご、いくつ？vs りんご、何個？／前者口語泛用，後者更明確「個數」
・子どもはいくつ？vs 子どもは何歳？／兩者都可，後者較正式

◆ 錯誤陷阱
・NG用法說明：大數量或精確場合不適合用いくつ
・× 人数はいくつですか。（正式統計）
・✓ 人数は何人ですか。／人數是多少？

◆ 變化補充
・分類：疑問詞（數量/年齡）
・影響答案的理由：需要精確單位時改用「何＋量詞」
・實戰建議：日常聊天用いくつ；考試看到單位→優先選何＋量詞
```

## いくら

```
◆ 語感例句結構
・語感：金額未知問
・結構：いくら（＋ですか）
・これ、いくらですか？／這個多少錢？
・全部でいくら払った？／總共付了多少？

◆ 混淆比較
・對手文法：いくつ／どのくらい
・一句話差異：いくら＝金額；いくつ＝數量；どのくらい＝程度/大概量
・これ、いくら？vs りんご、いくつ？／前者問價格，後者問數量
・どのくらいかかる？vs いくらかかる？／前者問時間/程度，後者問費用

◆ 錯誤陷阱
・NG用法說明：いくら不能拿來問數量
・× りんごはいくらある？
・✓ りんごはいくつある？／有幾個蘋果？

◆ 變化補充
・分類：疑問詞（金額）
・影響答案的理由：句子出現「錢／費用」一定用いくら
・實戰建議：看到「多少錢」直接反射いくら
```

## どう／いかが

```
◆ 語感例句結構
・語感：感受詢問差
・結構：どう／いかが（ですか）
・これ、どう？／這個怎麼樣？
・お味はいかがですか？／味道如何呢？

◆ 混淆比較
・對手文法：どう vs いかが
・一句話差異：いかが＝更禮貌；どう＝一般用
・この服どう？vs この服はいかがですか？／後者用在店員或正式場合
・調子どう？vs ご体調はいかがですか？／後者更尊敬對方

◆ 錯誤陷阱
・NG用法說明：對客戶/長輩用どう會顯得不夠禮貌
・× お味はどうですか。（服務業對客人）
・✓ お味はいかがですか。／味道如何呢？

◆ 變化補充
・分類：疑問詞（程度/狀態）
・影響答案的理由：句子出現敬語（お〜／ご〜）通常搭いかが
・實戰建議：正式場合一律選いかが，日常對話用どう
```

## どんな

```
◆ 語感例句結構
・語感：性質類型問
・結構：どんな＋名詞
・どんな映画が好き？／你喜歡什麼類型的電影？
・どんな人がタイプ？／你喜歡什麼樣的人？

◆ 混淆比較
・對手文法：どの／なに
・一句話差異：どんな＝問種類/性質；どの＝從已知中選；なに＝問內容本身
・どんな店？vs どの店？／前者問「什麼類型的店」，後者是「哪一家」
・どんな料理？vs 何の料理？／前者問風格，後者問是哪國/哪種料理

◆ 錯誤陷阱
・NG用法說明：どんな不能直接接具體名詞做選擇
・× どんな店に行く？（已知幾家店在選）
・✓ どの店に行く？／要去哪一家店？

◆ 變化補充
・分類：疑問詞（性質/類型）
・影響答案的理由：句子在問「什麼樣的」→選どんな
・實戰建議：先判斷是「選哪個」還是「問類型」，避免混用
```

## なぜ／どうして

```
◆ 語感例句結構
・語感：理由追問差
・結構：なぜ／どうして＋句子
・なぜ来なかった？／為什麼沒來？
・どうして泣いてるの？／你怎麼在哭？

◆ 混淆比較
・對手文法：なぜ vs どうして
・一句話差異：なぜ＝偏理性原因；どうして＝帶情緒/意外
・なぜ失敗したのか説明して。vs どうして来なかったの？／前者要分析理由，後者像責問或關心
・なぜこの方法を選んだ？vs どうしてそんなことしたの？／前者理性選擇，後者帶情緒

◆ 錯誤陷阱
・NG用法說明：正式說明用どうして會太口語/情緒化
・× レポートで「どうして〜」を書く。
・✓ レポートで「なぜ〜」を書く。／報告用なぜ

◆ 變化補充
・分類：疑問詞（原因）
・影響答案的理由：看到「說明/分析」偏なぜ；對話情緒偏どうして
・實戰建議：考試看到論述句→選なぜ；對話句→選どうして
```

## 疑問詞＋か

```
◆ 語感例句結構
・語感：不確定某個
・結構：疑問詞＋か
・誰か来たよ。／有人來了。
・何か食べる？／要吃點什麼嗎？

◆ 混淆比較
・對手文法：疑問詞＋も／疑問詞＋でも
・一句話差異：～か＝某個不確定；～も＝全部都；～でも＝隨便都可
・誰か来た。vs 誰も来なかった。／前者有人來，後者一個都沒來
・何か食べる？vs 何でも食べる。／前者隨便吃點什麼，後者什麼都吃

◆ 錯誤陷阱
・NG用法說明：誰も、何も要搭配否定（不能用肯定）
・× 誰も来た。
・✓ 誰も来なかった。／誰都沒來

◆ 變化補充
・分類：不定指
・影響答案的理由：句子有「某個／有人／什麼東西」就用～か
・實戰建議：看到不確定存在→選疑問詞＋か
```

## 疑問詞＋も＋否定

```
◆ 語感例句結構
・語感：一個都無
・結構：疑問詞＋も＋否定
・誰も来なかった。／誰都沒來。
・何も分からない。／什麼都不懂。

◆ 混淆比較
・對手文法：疑問詞＋か／疑問詞＋でも
・一句話差異：～も＋否定＝全否定；～か＝某個；～でも＝隨便都可（多肯定）
・誰も来ない。vs 誰か来る。／前者一個都沒有，後者有人會來
・何も食べない。vs 何でも食べる。／前者什麼都不吃，後者什麼都吃

◆ 錯誤陷阱
・NG用法說明：疑問詞＋も必須搭配否定
・× 誰も来た。
・✓ 誰も来なかった。／誰都沒來。

◆ 變化補充
・分類：全否定
・影響答案的理由：看到「完全沒有／一個都不」直接選～も＋否定
・實戰建議：句尾若是否定，優先檢查能不能用疑問詞＋も
```

# 指示詞

## これ／それ／あれ／どれ

```
◆ 語感例句結構
・語感：距離三分法
・結構：これ／それ／あれ／どれ
・これ、めっちゃいいね。／這個超不錯。
・あれは何？／那個（遠的）是什麼？

◆ 混淆比較
・對手文法：この／その／あの／どの
・一句話差異：～れ＝單獨代名詞；～の＝後面一定要接名詞
・これは高い。vs この服は高い。／前者直接指「這個」，後者要接名詞
・どれがいい？vs どの店がいい？／前者在選物，後者在選哪一家店

◆ 錯誤陷阱
・NG用法說明：これそれあれ不能直接接名詞
・× これ本を買う。
・✓ この本を買う。／買這本書。

◆ 變化補充
・分類：指示詞（こ・そ・あ・ど）
・影響答案的理由：看距離（近自己＝こ／近對方＝そ／遠＝あ／疑問＝ど）
・實戰建議：先判斷距離，再決定用哪一組
```

## この／その／あの／どの

```
◆ 語感例句結構
・語感：限定名詞指
・結構：この／その／あの／どの＋名詞
・この店、めっちゃ安い。／這家店很便宜。
・あの人、見たことある。／那個人（遠的）好像看過。

◆ 混淆比較
・對手文法：これ／それ／あれ／どれ
・一句話差異：～の＝一定接名詞；～れ＝單獨使用
・この本を買う。vs これを買う。／前者指定「這本書」，後者直接說「這個」
・どの店がいい？vs どれがいい？／前者選店，後者選物

◆ 錯誤陷阱
・NG用法說明：この／その／あの不能單獨用
・× このがいい。
・✓ これがいい。／這個比較好。

◆ 變化補充
・分類：こそあど＋名詞限定
・影響答案的理由：句子後面有名詞→只能用この／その／あの／どの
・實戰建議：看到名詞直接排除「～れ」系列
```

## ここ／そこ／あそこ／どこ

```
◆ 語感例句結構
・語感：地點三分指
・結構：ここ／そこ／あそこ／どこ
・ここで待ってて。／在這裡等我。
・あそこにコンビニあるよ。／那邊（遠）有便利商店。

◆ 混淆比較
・對手文法：こっち／そっち／あっち
・一句話差異：ここ系＝具體地點；こっち系＝方向/較口語
・ここに座って。vs こっちに座って。／前者指這個位置，後者偏「這邊來」
・あそこに行く。vs あっちに行く。／前者明確地點，後者偏方向

◆ 錯誤陷阱
・NG用法說明：助詞要配動作（に＝存在/目的地；で＝動作地）
・× ここに食べる。
・✓ ここで食べる。／在這裡吃。

◆ 變化補充
・分類：指示詞（場所）
・影響答案的理由：看距離（我＝こ／你＝そ／都遠＝あ／疑問＝ど）
・實戰建議：先判斷是「在哪做事」還是「去哪裡」，再選助詞
```

# 形容詞

## い形容詞（現在／過去／否定）

```
◆ 語感例句結構
・語感：狀態直接變
・結構：い形容詞：い→い（現在）／かった（過去）／くない（否定）／くなかった（過去否定）
・この店は安い。昨日は安かった。／這家店便宜，昨天也很便宜。
・今日は寒くない。昨日も寒くなかった。／今天不冷，昨天也不冷。

◆ 混淆比較
・對手文法：な形容詞（だ系）
・一句話差異：い形＝自己變形；な形＝靠だ變化
・安い→安くない。vs 便利だ→便利じゃない。／前者直接改，後者用じゃない
・高かった。vs 静かだった。／い形用かった，な形用だった

◆ 錯誤陷阱
・NG用法說明：い形容詞不能加だ
・× 安いだ。
・✓ 安い。／很便宜。

◆ 變化補充
・分類：基本活用
・影響答案的理由：看到い結尾直接套四種變化
・實戰建議：先判斷い形/な形，再決定變化方式
```

## い形容詞くて

```
◆ 語感例句結構
・語感：並列原因接
・結構：い形容詞（去い）＋くて
・この店は安くておいしい。／這家店又便宜又好吃。
・疲れていて、眠くて寝た。／因為很累又很想睡就去睡了。

◆ 混淆比較
・對手文法：～し／～から
・一句話差異：くて＝自然並列/弱原因；し＝列舉理由；から＝明確原因
・安くて買った。vs 安いから買った。／後者原因更明確
・便利だし安い。vs 便利で安い。／し是「還有」，で是單純並列

◆ 錯誤陷阱
・NG用法說明：くて不能接「意志/命令」句
・× 安くて買おう。
・✓ 安いから買おう。／因為便宜所以要買。

◆ 變化補充
・分類：て形連接
・影響答案的理由：後面是敘述結果→くて；是意志/命令→不用くて
・實戰建議：看到「打算/命令」直接排除くて
```

## い形容詞く＋動詞

```
◆ 語感例句結構
・語感：變成那樣
・結構：い形容詞（去い）＋く＋動詞（なる／する 等）
・寒くなった。／變冷了。
・音を小さくした。／把聲音調小了。

◆ 混淆比較
・對手文法：～くて／～に（な形）
・一句話差異：く＋動詞＝狀態變化/操作；くて＝並列/原因；に＝な形專用
・寒くなった。vs 寒くて外に出ない。／前者變冷，後者因為冷
・静かにする。vs 静かくする（×）／な形用に，不能用く

◆ 錯誤陷阱
・NG用法說明：な形容詞不能用く（要用に）
・× 便利くする。
・✓ 便利にする。／變方便。

◆ 變化補充
・分類：狀態變化（なる／する）
・影響答案的理由：看到「變得／把～變」就用く＋動詞
・實戰建議：先判斷是描述原因還是變化，避免跟くて混
```

## い形容詞＋名詞

```
◆ 語感例句結構
・語感：直接修飾名
・結構：い形容詞＋名詞
・安い店を探してる。／在找便宜的店。
・面白い映画を見た。／看了一部有趣的電影。

◆ 混淆比較
・對手文法：な形容詞＋名詞
・一句話差異：い形＝直接接；な形＝要加な
・安い店。vs 便利な店。／前者不用な，後者一定要な
・おいしい料理。vs 有名な料理。／同樣規則

◆ 錯誤陷阱
・NG用法說明：い形容詞不能加な
・× 安いな店。
・✓ 安い店。／便宜的店。

◆ 變化補充
・分類：連體修飾
・影響答案的理由：看到い結尾直接接名詞，不加な
・實戰建議：先判斷い形/な形，再決定要不要加な
```

## い形容詞＋の

```
◆ 語感例句結構
・語感：狀態名詞化
・結構：い形容詞＋の
・甘いのが好き。／喜歡甜的。
・高いのは買わない。／貴的不買。

◆ 混淆比較
・對手文法：～こと／～もの
・一句話差異：の＝具體事物/感覺；こと＝抽象行為；もの＝實體名詞
・走るのが好き。vs 走ることが好き。／前者偏感覺，後者較抽象
・白いの。vs 白いもの。／前者像「那個白的」，後者是「白色的東西（泛指）」

◆ 錯誤陷阱
・NG用法說明：の不能直接接名詞（會重複名詞功能）
・× 甘いのケーキ。
・✓ 甘いケーキ。／甜的蛋糕。

◆ 變化補充
・分類：名詞化
・影響答案的理由：句子需要「那個～的」時用の
・實戰建議：看到省略名詞（如東西/人）就用の
```

## な形容動詞で

◆ 語感例句結構

## な形容動詞に＋動詞

```
◆ 語感例句結構
・語感：變成那樣
・結構：な形容詞＋に＋動詞（なる／する 等）
・部屋がきれいになった。／房間變乾淨了。
・設定を簡単にした。／把設定變簡單了。

◆ 混淆比較
・對手文法：～で／～く（い形）
・一句話差異：に＋動詞＝狀態變化/操作；で＝並列/原因；く＝い形用
・静かになった。vs 静かで寝やすい。／前者變安靜，後者因為安靜
・高くする。vs 静かにする。／い形用く，な形用に

◆ 錯誤陷阱
・NG用法說明：な形容詞不能用く
・× 便利くする。
・✓ 便利にする。／變方便。

◆ 變化補充
・分類：狀態變化（なる／する）
・影響答案的理由：看到「變得／把～變」直接選に＋動詞
・實戰建議：先判斷詞性，避免跟い形く混用
```

## な形容動詞な＋名詞

```
◆ 語感例句結構
・語感：必加な修飾
・結構：な形容詞＋な＋名詞
・静かな部屋が好き。／喜歡安靜的房間。
・便利なアプリを使ってる。／在用方便的App。

◆ 混淆比較
・對手文法：い形容詞＋名詞
・一句話差異：な形＝一定要な；い形＝直接接
・便利な店。vs 安い店。／前者要な，後者不用
・有名な人。vs 面白い人。／同樣規則

◆ 錯誤陷阱
・NG用法說明：な不能省略
・× 静か部屋。
・✓ 静かな部屋。／安靜的房間。

◆ 變化補充
・分類：連體修飾
・影響答案的理由：看到な形修飾名詞一定補な
・實戰建議：先判斷是不是な形，再決定要不要加な
```

## な形容動詞な＋の

```
◆ 語感例句結構
・語感：那種狀態的
・結構：な形容詞＋な＋の
・静かなのが好き。／喜歡安靜的（那種）。
・便利なのを選んだ。／選了方便的那個。

◆ 混淆比較
・對手文法：～の／～こと
・一句話差異：な＋の＝狀態＋具體對象；の＝直接名詞化（多接動詞/い形）；こと＝抽象行為
・静かなのがいい。vs 走るのがいい。／前者是狀態，後者是行為
・便利なの。vs 便利なこと。／前者指東西，後者偏概念

◆ 錯誤陷阱
・NG用法說明：な形容詞名詞化不能省略な
・× 静かのが好き。
・✓ 静かなのが好き。／喜歡安靜的。

◆ 變化補充
・分類：名詞化（な形）
・影響答案的理由：看到な形＋「的那個」一定要な＋の
・實戰建議：先補な再接の，兩步不能少
```

# 動詞（基本）

## 動詞（現在／過去／否定）

```
◆ 語感例句結構
・語感：時態直接變
・結構：動詞：辞書形（現在）／た形（過去）／ない形（否定）／なかった（過去否定）
・毎日運動する。昨日は運動した。／每天運動，昨天也運動了。
・今日は行かない。昨日も行かなかった。／今天不去，昨天也沒去。

◆ 混淆比較
・對手文法：い形／な形
・一句話差異：動詞＝表示動作變化；形容詞＝表示狀態
・食べる。vs おいしい。／前者是動作，後者是狀態
・行かなかった。vs 高くなかった。／前者動詞否定過去，後者形容詞否定過去

◆ 錯誤陷阱
・NG用法說明：動詞不能用形容詞變化
・× 行くくない。
・✓ 行かない。／不去。

◆ 變化補充
・分類：基本活用
・影響答案的理由：看到動作就用動詞變化
・實戰建議：先分辨是不是動作，再套對應變形
```

## 動詞（基本形）

```
◆ 語感例句結構
・語感：原始動作形
・結構：動詞辞書形（～る／～く 等）
・毎日コーヒーを飲む。／每天喝咖啡。
・明日映画を見る。／明天看電影。

◆ 混淆比較
・對手文法：ます形／ない形
・一句話差異：辞書形＝最原始；ます形＝禮貌；ない形＝否定
・食べる。vs 食べます。／前者普通，後者禮貌
・行く。vs 行かない。／前者肯定，後者否定

◆ 錯誤陷阱
・NG用法說明：辞書形不能直接接です
・× 行くです。
・✓ 行きます。／要去。

◆ 變化補充
・分類：基本形（所有變化起點）
・影響答案的理由：看到原形句或接續（如前面直接接動詞）常用辞書形
・實戰建議：先認出動詞原形，其他變化全部從這裡推
```

## 動詞＋名詞

```
◆ 語感例句結構
・語感：動作修飾名
・結構：動詞（普通形）＋名詞
・昨日見た映画は面白かった。／昨天看的電影很好看。
・日本に行く人が増えてる。／去日本的人變多了。

◆ 混淆比較
・對手文法：名詞の＋名詞／い形＋名詞
・一句話差異：動詞＋名詞＝做過/要做的動作；の＝所屬；い形＝性質
・読む本。vs 日本の本。／前者是「要讀的書」，後者是「日本的書」
・買う店。vs 安い店。／前者是「要去買的店」，後者是「便宜的店」

◆ 錯誤陷阱
・NG用法說明：不能用ます形修飾名詞
・× 食べます店。
・✓ 食べる店。／吃東西的店。

◆ 變化補充
・分類：連體修飾（動詞）
・影響答案的理由：看到「做～的○○」直接用動詞普通形
・實戰建議：先想動作，再直接接名詞，不要加の或ます
```

## 動詞＋て

```
◆ 語感例句結構
・語感：動作接續用
・結構：動詞て形
・ご飯を食べて寝た。／吃完飯就睡了。
・ドアを開けて入ってください。／請開門進來。

◆ 混淆比較
・對手文法：た形／ます形
・一句話差異：て形＝接後面動作/請求；た形＝完成；ます形＝禮貌
・食べて寝た。vs 食べた。／前者有連續動作，後者只表完成
・待ってください。vs 待ちます。／前者請求，後者單純敘述

◆ 錯誤陷阱
・NG用法說明：て形不能單獨當完整句（需接續）
・× 食べて。
・✓ 食べてください。／請吃。

◆ 變化補充
・分類：接續形
・影響答案的理由：句子有「然後／請～」幾乎都用て形
・實戰建議：看到連續動作或請求，優先用て形
```

## 動詞＋ています（進行／狀態／習慣）

```
◆ 語感例句結構
・語感：正在或已成
・結構：動詞て形＋います
・今ご飯を食べています。／正在吃飯。
・結婚しています。／已經結婚（維持狀態）。

◆ 混淆比較
・對手文法：ている vs た
・一句話差異：ている＝進行/狀態/習慣；た＝一次完成
・窓が開いている。vs 窓が開いた。／前者開著（狀態），後者開了（動作）
・毎日走っている。vs 昨日走った。／前者習慣，後者單次

◆ 錯誤陷阱
・NG用法說明：結果動詞用た會少了「現在狀態」
・× 結婚した。（想表現在已婚）
・✓ 結婚している。／已婚（現在狀態）

◆ 變化補充
・分類：進行／結果狀態／習慣
・影響答案的理由：看是「正在做」還是「做完後維持」
・實戰建議：有「現在持續」感就用ている，不要只用た
```

## 動詞＋ないで

```
◆ 語感例句結構
・語感：不做接續
・結構：動詞ない形＋で
・朝ご飯を食べないで出かけた。／沒吃早餐就出門了。
・何も言わないで帰った。／什麼都沒說就回去了。

◆ 混淆比較
・對手文法：～なくて
・一句話差異：ないで＝前後動作連續；なくて＝原因/理由
・食べないで寝た。vs 食べなくて寝た。／前者沒吃就睡，後者因為沒吃所以睡（不自然）
・行かないでください。vs 行かなくて困る。／前者禁止，後者原因

◆ 錯誤陷阱
・NG用法說明：ないで不能表原因
・× 雨が降らないで中止になった。
・✓ 雨が降らなくて中止になった。／因為沒下雨而取消（語意合理）

◆ 變化補充
・分類：否定接續／禁止
・影響答案的理由：看到「沒做A就做B」用ないで
・實戰建議：先判斷是「連續動作」還是「原因」，避免跟なくて混
```

## 動詞＋なくて

```
◆ 語感例句結構
・語感：因沒而怎樣
・結構：動詞ない形＋なくて
・お金がなくて行けない。／因為沒錢去不了。
・昨日寝なくて、めっちゃ眠い。／因為沒睡覺很睏。

◆ 混淆比較
・對手文法：～ないで
・一句話差異：なくて＝原因/理由；ないで＝不做A就做B（連續）
・食べなくて寝た。vs 食べないで寝た。／前者因沒吃而睡（怪），後者沒吃就睡
・行かなくて困る。vs 行かないで帰る。／前者原因，後者連續動作

◆ 錯誤陷阱
・NG用法說明：なくて不能接命令/請求
・× 行かなくてください。
・✓ 行かないでください。／請不要去。

◆ 變化補充
・分類：否定原因
・影響答案的理由：句子有「因為沒～」就用なくて
・實戰建議：看到後面是結果/情緒，多半用なくて
```

## 動詞＋たり〜たりします

```
◆ 語感例句結構
・語感：列舉動作們
・結構：動詞た形＋り、動詞た形＋りします
・週末は映画を見たり、ゲームしたりします。／週末會看看電影、玩玩遊戲。
・休みの日は寝たり出かけたりする。／休假日會睡覺、也會出門。

◆ 混淆比較
・對手文法：て形／～し
・一句話差異：たり〜たり＝舉例不完整；て形＝順序連續；し＝理由列舉
・食べて寝る。vs 食べたり寝たりする。／前者先後動作，後者只是舉例
・安いし便利だ。vs 安かったり便利だったりする。／し是理由，たり是特徵舉例

◆ 錯誤陷阱
・NG用法說明：たり後面不能用原形（要た形）
・× 食べるり、寝るりする。
・✓ 食べたり、寝たりする。／吃吃、睡睡。

◆ 變化補充
・分類：動作列舉（非完整）
・影響答案的理由：看到「～之類的」直接用たり〜たり
・實戰建議：句尾幾乎一定是「～たりする／します」
```

# 自他動詞

## が＋自動詞

```
◆ 語感例句結構
・語感：主體自然發
・結構：名詞＋が＋自動詞
・ドアが開いた。／門開了（自己開）。
・電気が消えた。／燈關了（自然/自己熄）。

◆ 混淆比較
・對手文法：を＋他動詞
・一句話差異：が自動＝主體自己變；を他動＝有人去做
・ドアが開いた。vs ドアを開けた。／前者門自己開，後者有人開門
・電気がついた。vs 電気をつけた。／前者燈亮了，後者有人開燈

◆ 錯誤陷阱
・NG用法說明：自動詞不能用を
・× ドアを開いた。
・✓ ドアが開いた。／門開了。

◆ 變化補充
・分類：自他動對應
・影響答案的理由：看句子有沒有「施事者」
・實戰建議：沒有人在做→用が＋自動詞
```

## を＋他動詞

```
◆ 語感例句結構
・語感：對象被操作
・結構：名詞＋を＋他動詞
・ドアを開けた。／把門打開了。
・電気を消した。／把燈關掉了。

◆ 混淆比較
・對手文法：が＋自動詞
・一句話差異：を他動＝有人去做；が自動＝自然發生
・ドアを開けた。vs ドアが開いた。／前者有人開門，後者門自己開
・電気をつけた。vs 電気がついた。／前者有人開燈，後者燈自己亮

◆ 錯誤陷阱
・NG用法說明：他動詞不能用が（會變成自動語感）
・× ドアが開けた。
・✓ ドアを開けた。／把門打開了。

◆ 變化補充
・分類：自他動對應
・影響答案的理由：看到「有人做某動作」就用を＋他動詞
・實戰建議：句子有操作者→直接選を
```

## 自動詞＋ています

```
◆ 語感例句結構
・語感：結果維持中
・結構：自動詞て形＋います
・ドアが開いています。／門開著。
・電気がついています。／燈亮著。

◆ 混淆比較
・對手文法：他動詞＋ている／た
・一句話差異：自動ている＝狀態結果；他動ている＝正在做；た＝單次完成
・ドアが開いている。vs ドアを開けている。／前者門是開的，後者正在開門
・電気がついている。vs 電気をつけた。／前者亮著，後者開了（動作）

◆ 錯誤陷阱
・NG用法說明：想表現在狀態不能用た
・× ドアが開いた。（想說現在開著）
・✓ ドアが開いている。／門現在開著。

◆ 變化補充
・分類：結果狀態
・影響答案的理由：看到「現在維持某狀態」用自動詞＋ている
・實戰建議：先判斷是動作還是狀態，避免跟他動混
```

## 他動詞＋てあります

```
◆ 語感例句結構
・語感：刻意準備好
・結構：名詞＋を＋他動詞て形＋あります
・席を予約してあります。／座位已經先訂好了。
・ドアを閉めてあります。／門已經被關好（有人刻意）。

◆ 混淆比較
・對手文法：自動詞＋ている
・一句話差異：てある＝有人刻意做並維持；ている＝自然/單純狀態
・ドアを閉めてある。vs ドアが閉まっている。／前者有人關好，後者只是關著
・資料を用意してある。vs 資料が用意されている。／前者強調事前準備，後者偏結果

◆ 錯誤陷阱
・NG用法說明：てある前面一定用を（不能用が）
・× ドアが閉めてある。
・✓ ドアを閉めてある。／門已經被關好。

◆ 變化補充
・分類：結果＋意圖（準備）
・影響答案的理由：句子有「先做好準備」就用てある
・實戰建議：看到「為了之後」這種語感直接選てある
```

# 請求・授受・提議

## 名詞＋をください

```
◆ 語感例句結構
・語感：直接要東西
・結構：名詞＋をください
・水をください。／請給我水。
・このケーキをください。／請給我這個蛋糕。

◆ 混淆比較
・對手文法：～てください／お願いします
・一句話差異：をください＝要東西；てください＝請做動作；お願いします＝更禮貌委託
・水をください。vs 水を持ってきてください。／前者拿水，後者請人去拿
・これをください。vs これお願いします。／後者更禮貌自然

◆ 錯誤陷阱
・NG用法說明：ください不能直接接動詞原形
・× 食べるください。
・✓ 食べてください。／請吃。

◆ 變化補充
・分類：請求（物品）
・影響答案的理由：句子是「拿某個東西」→用をください
・實戰建議：先判斷是要「東西」還是要「動作」
```

## 動詞＋てください

```
◆ 語感例句結構
・語感：請你去做
・結構：動詞て形＋ください
・ここに名前を書いてください。／請在這裡寫名字。
・ちょっと待ってください。／請稍等一下。

◆ 混淆比較
・對手文法：～て／～てもいい
・一句話差異：てください＝請求；て＝單純接續；てもいい＝允許
・ドアを開けてください。vs ドアを開けて入る。／前者請求，後者連續動作
・入ってください。vs 入ってもいい。／前者叫你進來，後者允許你進來

◆ 錯誤陷阱
・NG用法說明：ください不能接動詞原形
・× 書くください。
・✓ 書いてください。／請寫。

◆ 變化補充
・分類：請求（動作）
・影響答案的理由：看到「請～」且是動作就用てください
・實戰建議：先判斷是請對方做事，不是拿東西
```

## 動詞＋ないでください

```
◆ 語感例句結構
・語感：請不要做
・結構：動詞ない形＋でください
・ここでタバコを吸わないでください。／請不要在這裡抽菸。
・遅れないでください。／請不要遲到。

◆ 混淆比較
・對手文法：～なくてください／～てはいけない
・一句話差異：ないでください＝禮貌禁止；なくてください＝錯；てはいけない＝較強禁止
・入らないでください。vs 入ってはいけない。／前者較柔和，後者規則禁止
・食べないでください。vs 食べなくてください（×）／後者錯誤

◆ 錯誤陷阱
・NG用法說明：不能用なくてください
・× 来なくてください。
・✓ 来ないでください。／請不要來。

◆ 變化補充
・分類：否定請求
・影響答案的理由：看到「請不要～」直接用ないでください
・實戰建議：考試看到禁止但語氣不強，優先選ないでください
```

## 動詞＋てくださいませんか

```
◆ 語感例句結構
・語感：更客氣請求
・結構：動詞て形＋くださいませんか
・もう一度説明してくれませんか。／可以再說明一次嗎？
・ここに名前を書いてくださいませんか。／可以請您在這裡寫名字嗎？

◆ 混淆比較
・對手文法：～てください／～てもらえませんか
・一句話差異：くださいませんか＝更禮貌請求；てください＝普通請求；てもらえませんか＝委婉請對方幫忙
・待ってください。vs 待ってくださいませんか。／後者更客氣
・手伝ってもらえませんか。vs 手伝ってくださいませんか。／前者偏「請你幫我」，後者更尊敬

◆ 錯誤陷阱
・NG用法說明：不能接動詞原形
・× 来るくださいませんか。
・✓ 来てくださいませんか。／可以請您來嗎？

◆ 變化補充
・分類：敬語請求
・影響答案的理由：句子越正式/對象越高位→選くださいませんか
・實戰建議：看到客戶/長輩→直接升級這個句型
```

## ～をもらいます

```
◆ 語感例句結構
・語感：我收到東西
・結構：人に＋物を＋もらう
・友達にプレゼントをもらった。／從朋友那裡收到了禮物。
・先生にアドバイスをもらいました。／從老師那裡得到了建議。

◆ 混淆比較
・對手文法：あげる／くれる
・一句話差異：もらう＝我拿到；あげる＝我給別人；くれる＝別人給我
・友達に本をもらった。vs 友達に本をあげた。／前者我拿，後者我給
・先生が本をくれた。vs 先生に本をもらった。／兩者都我拿，但主語不同

◆ 錯誤陷阱
・NG用法說明：來源要用に（不能用を）
・× 友達をプレゼントをもらった。
・✓ 友達にプレゼントをもらった。／從朋友那裡收到禮物。

◆ 變化補充
・分類：授受動詞（受）
・影響答案的理由：句子重點在「我得到」就用もらう
・實戰建議：先判斷主語是誰拿東西，再選動詞
```

## 動詞＋ましょう

```
◆ 語感例句結構
・語感：一起做吧
・結構：動詞ます形＋ましょう
・一緒に帰りましょう。／一起回去吧。
・ちょっと休みましょう。／稍微休息一下吧。

◆ 混淆比較
・對手文法：～ましょうか／～よう
・一句話差異：ましょう＝提議一起做；ましょうか＝主動幫忙；よう＝普通形提議
・手伝いましょう。vs 手伝いましょうか。／前者一起做，後者我幫你
・帰りましょう。vs 帰ろう。／前者禮貌，後者隨便

◆ 錯誤陷阱
・NG用法說明：ましょう不能接動詞原形
・× 行くましょう。
・✓ 行きましょう。／一起去吧。

◆ 變化補充
・分類：意志／提議（丁寧）
・影響答案的理由：句子有「一起～吧」就用ましょう
・實戰建議：對話中想拉對方一起行動，優先用這個
```

## 動詞＋ませんか

```
◆ 語感例句結構
・語感：要不要一起
・結構：動詞ます形＋ませんか
・一緒にご飯に行きませんか？／要不要一起去吃飯？
・ちょっと休みませんか？／要不要休息一下？

◆ 混淆比較
・對手文法：～ましょう／～ましょうか
・一句話差異：ませんか＝詢問對方意願；ましょう＝直接提議；ましょうか＝主動幫忙
・行きませんか？vs 行きましょう。／前者問你要不要，後者直接說走吧
・手伝いましょうか？vs 手伝いませんか？／前者我幫你，後者一起做

◆ 錯誤陷阱
・NG用法說明：ませんか不是否定句（不是「不做嗎」）
・× 行きませんか。（理解成不去嗎）
・✓ 行きませんか？／要不要去？

◆ 變化補充
・分類：邀請
・影響答案的理由：句子是「詢問對方要不要一起」就用ませんか
・實戰建議：想讓對方有選擇空間，比ましょう更柔和
```

## 動詞＋ましょうか

```
◆ 語感例句結構
・語感：我來幫吧
・結構：動詞ます形＋ましょうか
・手伝いましょうか？／要我幫忙嗎？
・荷物を持ちましょうか？／要我幫你拿行李嗎？

◆ 混淆比較
・對手文法：～ましょう／～ませんか
・一句話差異：ましょうか＝我幫你；ましょう＝一起做；ませんか＝邀你一起
・手伝いましょうか？vs 手伝いましょう。／前者我幫你，後者一起做
・行きませんか？vs 行きましょうか？／前者邀你，後者偏我來做（少用於邀請）

◆ 錯誤陷阱
・NG用法說明：ましょうか不是單純提議（會變成「我來做」）
・× 一緒に行きましょうか。（想說一起去）
・✓ 一緒に行きましょう。／一起去吧。

◆ 變化補充
・分類：提議（主動幫忙）
・影響答案的理由：句子有「要不要我來？」就用ましょうか
・實戰建議：對方有負擔時，用這個最自然
```

# 希望・意志・原因

## 名詞＋がほしい

```
◆ 語感例句結構
・語感：想要東西
・結構：名詞＋がほしい
・新しいスマホがほしい。／想要新手機。
・休みがほしい。／想要休假。

◆ 混淆比較
・對手文法：動詞たい
・一句話差異：ほしい＝要名詞；たい＝想做動作
・水がほしい。vs 水を飲みたい。／前者要水，後者想喝水
・お金がほしい。vs お金を稼ぎたい。／前者要錢，後者想賺錢

◆ 錯誤陷阱
・NG用法說明：ほしい不能接動詞
・× 食べるほしい。
・✓ 食べたい。／想吃。

◆ 變化補充
・分類：願望（物）
・影響答案的理由：看到「想要某個東西」用がほしい
・實戰建議：先判斷是要「物」還是要「動作」
```

## 動詞＋たい

```
◆ 語感例句結構
・語感：我想去做
・結構：動詞ます形＋たい
・日本に行きたい。／想去日本。
・ラーメンを食べたい。／想吃拉麵。

◆ 混淆比較
・對手文法：がほしい／～たがる
・一句話差異：たい＝我想做；ほしい＝要東西；たがる＝他人想做（觀察）
・水がほしい。vs 水を飲みたい。／前者要水，後者想喝
・子どもが遊びたがる。vs 子どもが遊びたい。／前者觀察到他想，後者不自然（他人內心）

◆ 錯誤陷阱
・NG用法說明：たい通常不用來直接說他人的願望
・× 彼は行きたい。
・✓ 彼は行きたがっている。／他看起來想去。

◆ 變化補充
・分類：願望（動作）
・影響答案的理由：主語是「自己」→用たい
・實戰建議：看到想做某動作，直接用たい
```

## つもり

```
◆ 語感例句結構
・語感：已決定打算
・結構：動詞普通形＋つもりだ
・来年日本に行くつもりだ。／打算明年去日本。
・今日は早く寝るつもりです。／今天打算早點睡。

◆ 混淆比較
・對手文法：～ようと思う／～予定だ
・一句話差異：つもり＝主觀已決定；ようと思う＝剛想到；予定だ＝客觀安排
・行くつもりだ。vs 行こうと思う。／前者已決定，後者剛打算
・旅行する予定だ。vs 旅行するつもりだ。／前者有行程安排，後者只是自己想

◆ 錯誤陷阱
・NG用法說明：つもり不能用於不受自己控制的事
・× 雨が降るつもりだ。
・✓ 雨が降りそうだ。／看起來要下雨。

◆ 變化補充
・分類：意志（主觀）
・影響答案的理由：句子有「已經決定要做」就用つもり
・實戰建議：先判斷是計畫還是臨時想法
```

## ～から

```
◆ 語感例句結構
・語感：主觀原因說
・結構：普通形＋から
・雨が降っているから、行かない。／因為下雨所以不去。
・疲れたから、もう寝る。／因為累了所以要睡。

◆ 混淆比較
・對手文法：ので／て
・一句話差異：から＝主觀理由；ので＝客觀柔和；て＝弱原因/接續
・疲れたから帰る。vs 疲れたので帰る。／前者較直接，後者較禮貌
・お腹がすいて、食べた。vs お腹がすいたから食べた。／て較輕，から較明確

◆ 錯誤陷阱
・NG用法說明：から後面不能接命令/請求（在考試常當陷阱）
・× 雨だから、行ってください。
・✓ 雨なので、行ってください。／因為下雨，請去（較自然）

◆ 變化補充
・分類：原因（主觀）
・影響答案的理由：看到「因為我覺得/我判斷」就用から
・實戰建議：日常對話幾乎優先用から
```

## ～ので

```
◆ 語感例句結構
・語感：客觀理由說
・結構：普通形＋ので（名詞／な形：＋なので）
・雨が降っているので、行きません。／因為在下雨，所以不去。
・静かなので、ここにします。／因為很安靜，所以選這裡。

◆ 混淆比較
・對手文法：から／て
・一句話差異：ので＝客觀柔和；から＝主觀直接；て＝弱原因
・疲れたので帰ります。vs 疲れたから帰る。／前者較禮貌客觀
・お腹がすいて食べた。vs お腹がすいたので食べた。／後者原因更明確

◆ 錯誤陷阱
・NG用法說明：な形容詞＋ので前要加な
・× 静かので帰る。
・✓ 静かなので帰る。／因為很安靜所以回去。

◆ 變化補充
・分類：原因（客觀）
・影響答案的理由：句子偏說明/禮貌→用ので
・實戰建議：對客戶/正式場合優先用ので
```

# 比較・程度

## ～は～より

```
◆ 語感例句結構
・語感：A比B
・結構：AはBより～
・今日は昨日より寒い。／今天比昨天冷。
・この店はあの店より安い。／這家店比那家便宜。

◆ 混淆比較
・對手文法：～ほど／～よりも
・一句話差異：より＝單純比較；ほど＝否定比較（沒那麼）；よりも＝更強調比較
・今日は昨日ほど寒くない。vs 今日は昨日より寒い。／前者沒那麼冷，後者更冷
・これよりいい。vs これよりもいい。／後者語氣更強

◆ 錯誤陷阱
・NG用法說明：比較句不能少比較對象
・× 今日はより寒い。
・✓ 今日は昨日より寒い。／今天比昨天冷。

◆ 變化補充
・分類：比較句
・影響答案的理由：句子有「比～更～」就用より
・實戰建議：先找A/B兩個對象，再套より
```

## より～ほう

```
◆ 語感例句結構
・語感：與其A不如B
・結構：AよりBのほうが～
・電車よりバスのほうが安い。／比起電車，公車比較便宜。
・外で食べるより家で食べるほうがいい。／比起外食，在家吃比較好。

◆ 混淆比較
・對手文法：AはBより～
・一句話差異：より～ほう＝強調「B比較好」；AはBより＝單純比較
・電車よりバスのほうがいい。vs バスは電車よりいい。／前者強調選B，後者中性比較
・行くよりやめるほうがいい。vs やめるは行くよりいい（×）／後者結構不自然

◆ 錯誤陷阱
・NG用法說明：ほうが不能少
・× 電車よりバスが安い。（少了比較焦點）
・✓ 電車よりバスのほうが安い。／公車比較便宜。

◆ 變化補充
・分類：比較（偏好）
・影響答案的理由：句子有「比較推薦/比較好」就用ほう
・實戰建議：想表「選哪個比較好」優先用這個句型
```

## 名詞＋とおなじ

```
◆ 語感例句結構
・語感：完全一樣
・結構：名詞＋と同じ（おなじ）
・彼は私と同じ学校です。／他跟我同一間學校。
・この服は前と同じだ。／這件衣服跟之前的一樣。

◆ 混淆比較
・對手文法：～と似ている／～と違う
・一句話差異：同じ＝完全一樣；似ている＝相似；違う＝不同
・私と同じ考えだ。vs 私と似ている考えだ。／前者一樣，後者只是像
・日本と違う文化。vs 日本と同じ文化。／完全不同 vs 完全相同

◆ 錯誤陷阱
・NG用法說明：不能用に（要用と）
・× 私に同じ。
・✓ 私と同じ。／跟我一樣。

◆ 變化補充
・分類：比較（相同）
・影響答案的理由：看到「一樣」直接用と同じ
・實戰建議：比較句固定搭配記「と同じ」
```

## あまり～ない

```
◆ 語感例句結構
・語感：不太那樣
・結構：あまり＋否定
・あまり食べない。／不太吃。
・この店はあまり安くない。／這家店不太便宜。

◆ 混淆比較
・對手文法：全然～ない／あまり～肯定
・一句話差異：あまり～ない＝程度不高；全然～ない＝完全沒有；肯定不可接あまり
・あまり分からない。vs 全然分からない。／前者不太懂，後者完全不懂
・あまり好きじゃない。vs とても好きだ。／不能說「あまり好きだ」

◆ 錯誤陷阱
・NG用法說明：あまり不能接肯定
・× あまり好きだ。
・✓ あまり好きじゃない。／不太喜歡。

◆ 變化補充
・分類：程度否定
・影響答案的理由：看到「不太～」直接用あまり＋ない
・實戰建議：句尾若是否定，優先檢查あまり
```

# 時間關係

## 動詞＋てから

```
◆ 語感例句結構
・語感：做完再做
・結構：動詞て形＋から
・ご飯を食べてから出かける。／吃完飯再出門。
・仕事が終わってから飲みに行った。／工作結束後才去喝酒。

◆ 混淆比較
・對手文法：～たら／～あとで
・一句話差異：てから＝先後順序強；たら＝條件；あとで＝單純時間後
・食べてから行く。vs 食べたら行く。／前者一定先吃，後者偏條件
・食べてから寝る。vs 食べたあとで寝る。／前者順序更強

◆ 錯誤陷阱
・NG用法說明：前後主語通常要一致
・× 彼が来てから、私は帰った。（主語不一致不自然）
・✓ 彼が来てから帰った。／他來了之後（我）才回去。

◆ 變化補充
・分類：順序接續
・影響答案的理由：看到「先A再B」直接用てから
・實戰建議：強調順序一定用てから，不用其他
```

## 動詞＋たあとで／たあと

```
◆ 語感例句結構
・語感：做完之後
・結構：動詞た形＋あと（で）
・ご飯を食べたあとで出かける。／吃完飯之後出門。
・映画を見たあと、カフェに行った。／看完電影後去咖啡廳。

◆ 混淆比較
・對手文法：～てから／～たら
・一句話差異：たあと＝單純時間後；てから＝順序強；たら＝條件
・食べたあとで寝る。vs 食べてから寝る。／後者更強調「一定先吃」
・終わったら帰る。vs 終わったあと帰る。／前者偏條件，後者單純之後

◆ 錯誤陷阱
・NG用法說明：不能用て形接あと
・× 食べてあとで行く。
・✓ 食べたあとで行く。／吃完之後去。

◆ 變化補充
・分類：時間順序
・影響答案的理由：句子只是「之後」不用強調順序→選たあと
・實戰建議：沒有強調順序就用たあと，比較中性
```

## 名詞＋の＋あとで／あと

```
◆ 語感例句結構
・語感：名詞之後
・結構：名詞＋の＋あと（で）
・会議のあとで食事に行く。／會議之後去吃飯。
・授業のあと、図書館に行った。／下課後去圖書館。

◆ 混淆比較
・對手文法：動詞＋たあと／～てから
・一句話差異：名詞のあと＝事件名詞後；たあと＝動作後；てから＝順序強
・会議のあとで帰る。vs 会議が終わったあと帰る。／名詞 vs 動詞表達
・食べてから行く。vs 食事のあとで行く。／前者順序強，後者單純之後

◆ 錯誤陷阱
・NG用法說明：名詞前一定要加の
・× 会議あとで帰る。
・✓ 会議のあとで帰る。／會議之後回去。

◆ 變化補充
・分類：時間（名詞）
・影響答案的理由：前面是名詞事件→只能用のあと
・實戰建議：先判斷前面是動詞還是名詞
```

## 動詞＋まえに

```
◆ 語感例句結構
・語感：做之前先
・結構：動詞辞書形＋前に
・寝る前に歯を磨く。／睡前先刷牙。
・出かける前に連絡して。／出門前先聯絡。

◆ 混淆比較
・對手文法：～たあと／～てから
・一句話差異：前に＝還沒做；たあと＝做完後；てから＝先後順序
・食べる前に薬を飲む。vs 食べたあとで薬を飲む。／前者吃之前，後者吃之後
・行く前に準備する。vs 行ってから準備する。／時間順序完全相反

◆ 錯誤陷阱
・NG用法說明：前に不能用た形（會變已做完）
・× 食べた前に寝る。
・✓ 食べる前に寝る。／吃之前睡。

◆ 變化補充
・分類：時間（之前）
・影響答案的理由：看到「還沒做就先」→用辞書形＋前に
・實戰建議：前に＝未發生，直接用原形
```

## 名詞＋の＋まえに

```
◆ 語感例句結構
・語感：名詞之前
・結構：名詞＋の＋前に
・会議の前に資料を確認する。／會議前先確認資料。
・食事の前に手を洗う。／吃飯前先洗手。

◆ 混淆比較
・對手文法：動詞＋前に／～たあと
・一句話差異：名詞の前に＝事件名詞前；動詞前に＝動作前；たあと＝之後
・会議の前に行く。vs 会議が始まる前に行く。／名詞 vs 動詞表達
・食べる前に薬を飲む。vs 食べたあとで飲む。／前後時間相反

◆ 錯誤陷阱
・NG用法說明：名詞前不能省略の
・× 会議前に行く。
・✓ 会議の前に行く。／會議前去。

◆ 變化補充
・分類：時間（之前）
・影響答案的理由：前面是名詞事件→一定用の前に
・實戰建議：先判斷前面是不是名詞
```

## 動詞＋ながら

```
◆ 語感例句結構
・語感：同時進行
・結構：動詞ます形去ます＋ながら
・音楽を聞きながら勉強する。／一邊聽音樂一邊讀書。
・歩きながら電話してる。／一邊走一邊講電話。

◆ 混淆比較
・對手文法：～て／～つつ
・一句話差異：ながら＝同時兩動作；て＝先後；つつ＝書面同時
・食べて寝る。vs 食べながら寝る（×）／て是順序，ながら不能接不可能同時
・考えつつ話す。vs 考えながら話す。／意思近，つつ較書面

◆ 錯誤陷阱
・NG用法說明：ながら兩動作要能同時
・× 寝ながら勉強する。
・✓ 音楽を聞きながら勉強する。／邊聽音樂邊讀書。

◆ 變化補充
・分類：同時動作
・影響答案的理由：看到「一邊～一邊～」直接用ながら
・實戰建議：先判斷兩動作能不能同時發生
```

## ～とき

```
◆ 語感例句結構
・語感：在那時候
・結構：普通形＋とき（名詞：の＋とき）
・日本に行ったとき、寿司を食べた。／去日本的時候吃了壽司。
・暇なとき、ゲームする。／有空的時候會打遊戲。

◆ 混淆比較
・對手文法：～たら／～ながら
・一句話差異：とき＝時間點/期間；たら＝條件/之後；ながら＝同時
・家に帰ったとき電話した。vs 帰ったら電話した。／前者強調時間點，後者偏順序
・勉強するとき音楽を聞く。vs 勉強しながら音楽を聞く。／前者泛指時候，後者同時進行

◆ 錯誤陷阱
・NG用法說明：時態會影響前後關係（很常考）
・× 食べたとき薬を飲む。（想表吃之前）
・✓ 食べるとき薬を飲む。／吃的時候（前/中）吃藥。

◆ 變化補充
・分類：時間（點/期間）
・影響答案的理由：前面用た形＝動作後；原形＝動作前/中
・實戰建議：考試重點：先判斷動作發生順序
```

## 時間＋に

```
◆ 語感例句結構
・語感：精準時間點
・結構：時間名詞＋に
・7時に起きる。／7點起床。
・明日3時に会おう。／明天3點見。

◆ 混淆比較
・對手文法：～（時間）／～ごろ
・一句話差異：に＝明確時間點；不加＝相對時間；ごろ＝大概時間
・昨日行った。vs 3時に行った。／前者不用に，後者要
・3時に行く。vs 3時ごろ行く。／前者準時，後者大約

◆ 錯誤陷阱
・NG用法說明：相對時間不能用に
・× 明日に行く。
・✓ 明日行く。／明天去。

◆ 變化補充
・分類：時間（點）
・影響答案的理由：看到具體時刻→一定用に
・實戰建議：時間越精確，越需要に
```

## ごろ／ころ

```
◆ 語感例句結構
・語感：大概那時
・結構：時間＋ごろ／動詞・名詞＋ころ
・3時ごろ帰る。／大概3點回去。
・子どものころよく遊んだ。／小時候常常玩。

◆ 混淆比較
・對手文法：に／くらい
・一句話差異：ごろ＝具體時間點大約；ころ＝時期/那段時間；に＝精準時間
・3時ごろ行く。vs 3時に行く。／前者大約，後者準時
・学生のころ。vs 学生のごろ（×）／時期要用ころ

◆ 錯誤陷阱
・NG用法說明：ごろ不能接抽象時期
・× 子どものごろ。
・✓ 子どものころ。／小時候。

◆ 變化補充
・分類：時間（大概）
・影響答案的理由：時間點→ごろ；人生階段→ころ
・實戰建議：看到「幾點」用ごろ，「～時候」用ころ
```

## ～中（じゅう／ちゅう）

```
◆ 語感例句結構
・語感：整段期間內
・結構：名詞＋中（じゅう／ちゅう）
・世界中を旅行したい。／想環遊全世界。
・会議中なので、静かにしてください。／正在開會中，請安靜。

◆ 混淆比較
・對手文法：～間／～ている
・一句話差異：中＝整體範圍/期間；間＝持續時間內；ている＝進行狀態
・一日中寝ていた。vs 一日の間寝ていた。／前者強調整天，後者只是期間
・会議中。vs 会議している。／前者名詞化狀態，後者動作進行

◆ 錯誤陷阱
・NG用法說明：じゅう／ちゅう讀法不能亂用
・× 世界ちゅう。
・✓ 世界じゅう。／全世界。

◆ 變化補充
・分類：範圍／期間
・影響答案的理由：空間/整體→じゅう；動作中→ちゅう
・實戰建議：看到「整個～」「正在～中」優先用中
```

## すぎ／まえ

```
◆ 語感例句結構
・語感：前後時間點
・結構：時間＋すぎ／まえ
・3時すぎに来た。／3點過後來了。
・5分まえに着いた。／提前5分鐘到了。

◆ 混淆比較
・對手文法：～あと／～ごろ
・一句話差異：すぎ＝剛過那時間；まえ＝在那之前；あと＝之後（較廣）；ごろ＝大概
・3時すぎに出る。vs 3時ごろ出る。／前者過3點，後者大約3點
・会議のまえに準備する。vs 会議のあとで準備する。／前後相反

◆ 錯誤陷阱
・NG用法說明：まえ不能接た形（會變已發生）
・× 食べたまえに寝る。
・✓ 食べるまえに寝る。／吃之前睡。

◆ 變化補充
・分類：時間（前後）
・影響答案的理由：看到「過～點」用すぎ，「～之前」用まえ
・實戰建議：先判斷是「過了」還是「還沒」
```

# 變化表現

## 形容詞く＋なります

```
◆ 語感例句結構
・語感：自然變那樣
・結構：い形容詞（去い）＋く＋なります
・寒くなりました。／變冷了。
・安くなったので買った。／因為變便宜就買了。

◆ 混淆比較
・對手文法：～くする／～になる（な形）
・一句話差異：くなる＝自然變；くする＝人為改；にする/なる＝な形用
・暑くなった。vs 暑くした。／前者天氣變熱，後者（人為）讓它變熱
・静かになる。vs 静かにする。／前者自然變安靜，後者讓它安靜

◆ 錯誤陷阱
・NG用法說明：い形容詞不能用に
・× 高いにになる。
・✓ 高くなる。／變高。

◆ 變化補充
・分類：狀態變化（自然）
・影響答案的理由：句子沒有「人為操作」就用くなる
・實戰建議：看到「變得」先判斷是自然還是人為
```

## 形容動詞に＋なります

```
◆ 語感例句結構
・語感：自然變那樣
・結構：な形容詞＋に＋なります
・部屋が静かになりました。／房間變安靜了。
・操作が簡単になった。／操作變簡單了。

◆ 混淆比較
・對手文法：～にする／～くなる
・一句話差異：に＋なる＝自然變化；にする＝人為改變；くなる＝い形用
・静かになる。vs 静かにする。／前者自然變安靜，後者讓它安靜
・高くなる。vs 便利になる。／い形用く，な形用に

◆ 錯誤陷阱
・NG用法說明：な形容詞不能用く
・× 便利くなる。
・✓ 便利になる。／變方便。

◆ 變化補充
・分類：狀態變化（自然）
・影響答案的理由：看到な形＋變化→一定用に＋なる
・實戰建議：先判斷詞性，再決定用く還是に
```

## 名詞に＋なります

```
◆ 語感例句結構
・語感：身分改變成
・結構：名詞＋に＋なります
・医者になりました。／變成醫生了。
・来年、社会人になります。／明年會成為社會人。

◆ 混淆比較
・對手文法：～にする／～になる（形容詞）
・一句話差異：名詞に＝身分/狀態改變；にする＝人為決定；く/に＋なる＝形容詞變化
・先生になる。vs 先生にする。／前者變成老師，後者選老師（當…）
・高くなる。vs 医者になる。／前者性質變化，後者身分變化

◆ 錯誤陷阱
・NG用法說明：名詞變化不能用く
・× 医者くなる。
・✓ 医者になる。／變成醫生。

◆ 變化補充
・分類：身分變化
・影響答案的理由：看到職業/身份變化→用名詞＋に
・實戰建議：先判斷是不是「變成某個人/角色」
```

## 形容詞く＋します

```
◆ 語感例句結構
・語感：我去改變
・結構：い形容詞（去い）＋く＋します
・音を小さくします。／把聲音調小。
・値段を安くしました。／把價格調便宜了。

◆ 混淆比較
・對手文法：～くなる／～にする
・一句話差異：くします＝人為改變；くなる＝自然變；にする＝な形用
・暑くする。vs 暑くなる。／前者讓它變熱，後者自然變熱
・静かにする。vs 高くする。／な形用に，い形用く

◆ 錯誤陷阱
・NG用法說明：い形容詞不能用に
・× 安いにする。
・✓ 安くする。／把它變便宜。

◆ 變化補充
・分類：人為變化
・影響答案的理由：句子有「把～變成」就用く＋する
・實戰建議：先判斷是不是「有人操作改變」
```

## 形容動詞に＋します

```
◆ 語感例句結構
・語感：我把它變
・結構：な形容詞＋に＋します
・部屋を静かにします。／把房間變安靜。
・設定を簡単にしました。／把設定變簡單了。

◆ 混淆比較
・對手文法：～にする／～くする
・一句話差異：にします＝な形人為改變；くする＝い形用；にする（名詞）＝選擇/決定
・静かにする。vs 高くする。／な形用に，い形用く
・コーヒーにする。vs 静かにする。／前者是「選咖啡」，後者是「變安靜」

◆ 錯誤陷阱
・NG用法說明：な形容詞不能用く
・× 便利くする。
・✓ 便利にする。／把它變方便。

◆ 變化補充
・分類：人為變化
・影響答案的理由：看到な形＋「把～變成」→一定用に＋する
・實戰建議：先分辨詞性，避免和い形混用
```

## 名詞に＋します

```
◆ 語感例句結構
・語感：決定選這個
・結構：名詞＋に＋します
・コーヒーにします。／我決定點咖啡。
・このプランにしました。／我選了這個方案。

◆ 混淆比較
・對手文法：～になります／～にする（形容詞）
・一句話差異：にします＝自己決定；になります＝自然/結果變化；にする（形）＝把～變
・医者になります。vs 医者にします。／前者變成醫生，後者（不自然）選醫生
・静かにする。vs コーヒーにする。／前者改變狀態，後者做選擇

◆ 錯誤陷阱
・NG用法說明：不能用く（名詞不用く）
・× コーヒーくする。
・✓ コーヒーにする。／選咖啡。

◆ 變化補充
・分類：選擇／決定
・影響答案的理由：句子有「我決定選哪個」→用名詞にする
・實戰建議：看到點餐/選方案幾乎都是這個
```

## もう＋肯定／否定

```
◆ 語感例句結構
・語感：已經／不再
・結構：もう＋肯定／もう＋否定
・もう食べた。／已經吃了。
・もう行かない。／不再去了。

◆ 混淆比較
・對手文法：まだ～ていない／まだ
・一句話差異：もう＝已完成/不再；まだ＝還沒/還在
・もう終わった。vs まだ終わっていない。／已經結束 vs 還沒結束
・もう行かない。vs まだ行っていない。／不再去 vs 還沒去

◆ 錯誤陷阱
・NG用法說明：もう＋否定是「不再」，不是「還沒」
・× もう食べていない。（想表還沒吃）
・✓ まだ食べていない。／還沒吃。

◆ 變化補充
・分類：時間副詞
・影響答案的理由：句子有「已經」或「不再」就用もう
・實戰建議：看到「還沒」一律選まだ，不要用もう
```

## まだ＋肯定／否定

```
◆ 語感例句結構
・語感：還在／還沒
・結構：まだ＋肯定／まだ＋否定（～ていない）
・まだ雨が降っている。／還在下雨。
・まだ宿題をしていない。／還沒做作業。

◆ 混淆比較
・對手文法：もう＋肯定／否定
・一句話差異：まだ＝未完成/持續；もう＝已完成/不再
・まだ食べていない。vs もう食べた。／還沒吃 vs 已經吃了
・まだ行く。vs もう行かない。／還會去 vs 不再去

◆ 錯誤陷阱
・NG用法說明：まだ＋否定要用ていない（不用普通ない）
・× まだ食べない。
・✓ まだ食べていない。／還沒吃。

◆ 變化補充
・分類：時間副詞
・影響答案的理由：看到「還在/還沒」就用まだ
・實戰建議：未完成狀態幾乎固定搭ていない
```

# 語氣・句尾

## ～ね

```
◆ 語感例句結構
・語感：對吧共感
・結構：句子＋ね
・今日は寒いね。／今天很冷呢。
・この店、おいしいね。／這家店很好吃吧。

◆ 混淆比較
・對手文法：～よ／～かな
・一句話差異：ね＝求共鳴；よ＝告知對方；かな＝自言自語
・寒いね。vs 寒いよ。／前者在共感，後者在告知
・いいね。vs いいかな。／前者一起覺得好，後者在猶豫

◆ 錯誤陷阱
・NG用法說明：ね不能用在對方不知道的資訊（會怪）
・× 明日休みだね。（對方不知道）
・✓ 明日休みだよ。／明天放假喔。

◆ 變化補充
・分類：終助詞
・影響答案的理由：句子在「確認/共感」就用ね
・實戰建議：想拉對方一起認同，用ね
```

## ～よ

```
◆ 語感例句結構
・語感：告知提醒
・結構：句子＋よ
・明日休みだよ。／明天放假喔。
・それ、間違ってるよ。／那個是錯的喔。

◆ 混淆比較
・對手文法：～ね／～かな
・一句話差異：よ＝給新資訊；ね＝求共感；かな＝自言自語
・寒いよ。vs 寒いね。／前者在告知，後者在共感
・行くよ。vs 行くかな。／前者決定去，後者在猶豫

◆ 錯誤陷阱
・NG用法說明：對方已知內容用よ會怪（太像在講給不知道的人）
・× これ、おいしいよ。（兩人都在吃）
・✓ これ、おいしいね。／這個很好吃吧。

◆ 變化補充
・分類：終助詞
・影響答案的理由：句子在「告訴對方」就用よ
・實戰建議：有資訊差→用よ；有共識→用ね
```

## ～か（疑問）

```
◆ 語感例句結構
・語感：提出疑問
・結構：句子＋か
・これは何ですか。／這是什麼？
・行きますか。／要去嗎？

◆ 混淆比較
・對手文法：～ね／～かな
・一句話差異：か＝正式提問；ね＝確認；かな＝自問/猶豫
・行きますか。vs 行きますね。／前者在問，後者在確認
・大丈夫かな。vs 大丈夫ですか。／前者自言自語，後者問對方

◆ 錯誤陷阱
・NG用法說明：普通形疑問不一定用か（口語常省略）
・× 行くか？（正式場合）
・✓ 行きますか。／要去嗎？

◆ 變化補充
・分類：疑問句（丁寧）
・影響答案的理由：看到禮貌問句→用か
・實戰建議：正式問句幾乎都加か
```

## 句子＋か、句子＋か

```
◆ 語感例句結構
・語感：A還是B
・結構：句子＋か、句子＋か
・行くか、行かないかまだ決めてない。／要去還是不去還沒決定。
・買うか買わないか迷ってる。／在猶豫要不要買。

◆ 混淆比較
・對手文法：～かどうか／～か、～か（名詞）
・一句話差異：か、か＝兩個明確選項；かどうか＝是否（不限定兩個）
・行くか行かないか。vs 行くかどうか。／前者二選一，後者單純「要不要」
・コーヒーかお茶か。vs コーヒーかどうか。／前者選項，後者語意不通

◆ 錯誤陷阱
・NG用法說明：兩邊句型要對稱
・× 行くか、帰るかない。
・✓ 行くか行かないか。／要去還是不去。

◆ 變化補充
・分類：選擇疑問
・影響答案的理由：看到「A還是B」→用～か、～か
・實戰建議：兩邊動詞要用同一動詞做肯定/否定對比
```

# 補充

## ～のだ

```
◆ 語感例句結構
・語感：說明背景
・結構：普通形＋のだ（口語：んだ）
・どうして来なかったのだ。／為什麼沒來（在說明理由）？
・実は忙しかったんだ。／其實是因為很忙。

◆ 混淆比較
・對手文法：～から／～よ
・一句話差異：のだ＝補充說明/背景；から＝直接原因；よ＝單純告知
・遅れたんだ。vs 遅れたから。／前者在解釋背景，後者直接給理由
・大丈夫だよ。vs 大丈夫なんだ。／前者告知，後者帶說明語感

◆ 錯誤陷阱
・NG用法說明：名詞／な形前要接「な」
・× 学生のだ。
・✓ 学生なのだ。／是學生（說明）。

◆ 變化補充
・分類：說明語氣
・影響答案的理由：句子在「解釋／補充背景」就用のだ
・實戰建議：口語幾乎都用「んだ」更自然
```

## という＋名詞

```
◆ 語感例句結構
・語感：稱作那個
・結構：句子／名詞＋という＋名詞
・「鬼滅の刃」というアニメが好き。／喜歡叫做《鬼滅之刃》的動畫。
・彼は天才という人だ。／他是所謂天才的人。

◆ 混淆比較
・對手文法：～と＋名詞／～って
・一句話差異：という＝定義/說明名稱；と＝直接引用；って＝口語簡化
・「東京」と言う。vs 「東京」という都市。／前者說「東京」，後者叫東京的城市
・これって何？vs これという問題。／って口語，という偏說明

◆ 錯誤陷阱
・NG用法說明：不能只用と接名詞做說明
・× 「鬼滅の刃」とアニメ。
・✓ 「鬼滅の刃」というアニメ。／叫鬼滅之刃的動畫。

◆ 變化補充
・分類：定義／說明
・影響答案的理由：看到「叫做～的」→用という
・實戰建議：名稱＋類別幾乎固定用という
```

## ～でしょう

```
◆ 語感例句結構
・語感：應該吧
・結構：普通形＋でしょう
・明日は雨でしょう。／明天應該會下雨。
・彼は来ないでしょう。／他應該不會來。

◆ 混淆比較
・對手文法：～だろう／～かもしれない
・一句話差異：でしょう＝禮貌推測；だろう＝普通推測；かもしれない＝可能性較低
・行くでしょう。vs 行くだろう。／前者禮貌，後者普通
・来るでしょう。vs 来るかもしれない。／前者較有把握，後者不太確定

◆ 錯誤陷阱
・NG用法說明：な形／名詞前要接「だ」
・× 静かでしょう。
・✓ 静かだでしょう。→（實際用法）静かでしょう。／安靜吧（口語常省略だ）

◆ 變化補充
・分類：推測（丁寧）
・影響答案的理由：句子在「應該～吧」→用でしょう
・實戰建議：正式場合優先用でしょう，比だろう更自然
```

## に～があります／います

```
◆ 語感例句結構
・語感：在某有什麼
・結構：場所に＋名詞が＋あります／います
・部屋に猫がいます。／房間裡有貓。
・公園にベンチがあります。／公園裡有長椅。

◆ 混淆比較
・對手文法：は～にある／いる
・一句話差異：にが＝新出現（有某物）；はに＝已知位置說明
・部屋に猫がいる。vs 猫は部屋にいる。／前者「有貓」，後者「貓在房間」
・ここに店がある。vs 店はここにある。／存在 vs 說明位置

◆ 錯誤陷阱
・NG用法說明：人用います，物用あります
・× 部屋に人があります。
・✓ 部屋に人がいます。／房間裡有人。

◆ 變化補充
・分類：存在句
・影響答案的理由：看到「某地有～」→用に＋が
・實戰建議：先判斷是「提出存在」還是「說明位置」
```

## は～にあります／います

```
◆ 語感例句結構
・語感：已知在哪
・結構：名詞は＋場所に＋あります／います
・猫は部屋にいます。／貓在房間裡。
・店は駅の近くにあります。／店在車站附近。

◆ 混淆比較
・對手文法：に～がある／いる
・一句話差異：はに＝已知主題的位置；にが＝某地有新東西
・猫は部屋にいる。vs 部屋に猫がいる。／前者說貓在哪，後者說房間有貓
・店はここにある。vs ここに店がある。／說明位置 vs 提出存在

◆ 錯誤陷阱
・NG用法說明：主題已知時不能用が（語感會變）
・× 猫が部屋にいる。（想表「貓在房間」）
・✓ 猫は部屋にいる。／貓在房間。

◆ 變化補充
・分類：位置句
・影響答案的理由：句子在「某物在哪裡」→用はに
・實戰建議：先判斷是介紹新東西還是說明已知位置
```
