# N1 文法

---

- **◎：一定要精熟**
- **○：至少看到就知道意思、語感、使用場景**
- **△：只做辨識，不深背**
- **✕：真的放掉**

# 時間・起點・範圍

## ○にして

```
◆ 語感例句結構
・極端才成立
・名詞＋にして
・彼は10歳にして、もう大学レベルの数学ができる。（才10歲就會大學數學）
・この店は安さ（やすさ）にして、この味はすごい。（這麼便宜卻有這種味道）

◆ 混淆比較
・で／にもかかわらず
・で＝單純狀態／にもかかわらず＝反差／にして＝極端條件成立
・彼は10歳で、天才だ。（只是狀態）
　彼は10歳にして、天才だ。（強調年齡極端）
・安いにもかかわらず、おいしい。（反差）
　安さにして、この味はすごい。（強調條件極端）

◆ 錯誤陷阱
・不能亂接一般名詞（需極端條件）
・× 学生にして、アルバイトしている
・✓ 学生にして、会社を経営している（學生卻開公司）

◆ 變化補充
・用法分支
・AにしてB（強調極端）／Aにしては（對比）
・考試看到「年齡／條件異常高或低」優先選
```

## ○にあって（は／も）

```
◆ 語感例句結構
・特殊情境下
・名詞＋にあって（は／も）
・困難な状況にあっても、彼は冷静だった。（在困難情況下仍冷靜）
・ 戦時（せんじ）にあっては、情報が最も重要だ。（在戰時情境下資訊最重要）

◆ 混淆比較
・において／の中で
・において＝單純範圍／の中で＝一般情境／にあって＝特殊嚴峻情境
・会議において、重要な決定がなされた。（單純場合）
  戦時にあっては、判断が難しい。（特殊嚴峻情境）
・多くの人の中で、彼が一番だ。（一般範圍）

◆ 錯誤陷阱
・不能用在普通日常情境
・× 日常生活にあって、楽しいことが多い
・✓ 厳しい環境にあって、努力し続けた（嚴峻情境）

◆ 變化補充
・助詞差異
・は＝強調對比／も＝即使在那情況
・考試看到「困難／戰時／危機」優先選
```

## ◎間際に／間際の （まぎわ）

```
◆ 語感例句結構
・最後一刻
・動詞辭書形＋間際に／名詞＋の＋間際の
・出発する間際に、電話がかかってきた。（快出發時來電話）
・締め切り（しめきり）の間際の申請が多い。（截止前最後一刻很多申請）

◆ 混淆比較
・直前に（ちょくぜんに）／ところに
・直前に＝客觀時間點／ところに＝剛好發生／間際に＝最後一刻緊逼感
・出発直前に、確認をした。（單純之前） (かくにん)
　出発する間際に、電話が来た。（最後一刻）
・出かけようとしたところに、雨が降った。（剛好那瞬間）

◆ 錯誤陷阱
・不能接普通名詞（需の）
・× 会議間際に、資料を出す
・✓ 会議の間際に、資料を出す（會議前最後一刻）

◆ 變化補充
・語氣重點
・帶「時間逼近、來不及感」
・考試看到「差點來不及／最後瞬間」優先選
```

## ◎を～に控えて（ひかえて）

```
◆ 語感例句結構
・重要前夕
・名詞＋を～に控えて
・試験を目前に控えて、毎日勉強している。（考試在即每天讀書）
・結婚式を来週に控えて、準備が忙しい。（婚禮在即很忙）

◆ 混淆比較
・前に／に先立ち
・前に＝單純之前／に先立ち＝正式事前／を控えて＝重大事件前夕
・試験前に、復習する。（單純之前）
　試験を目前に控えて、緊張している。（前夕壓力）
・式に先立ち、説明が行われた。（正式流程）

◆ 錯誤陷阱
・不能接動詞（只接名詞）
・× 試験を受けるに控えて、勉強する
・✓ 試験を控えて、勉強する（考試在即）

◆ 變化補充
・固定搭配
・「目前に控えて」最常考
・看到「重大事件＋準備／緊張」優先選
```

## △や／やいなや

```
◆ 語感例句結構
・剛一就
・動詞辭書形＋や／やいなや
・彼は家に帰るや、すぐ寝た。（一回家就睡）
・ドアを開けるやいなや、猫が飛び出した。（一開門貓就衝出來）

◆ 混淆比較
・とたん（に）／たとたん（に）
・とたん＝瞬間自然發生／や＝書面快速連續／やいなや＝更強立即性
・立ち上がったとたん、めまいがした。（瞬間發生）
　立ち上がるや、外に出た。（動作連續）
・ドアを開けたとたん、風が入った（自然結果）
　ドアを開けるやいなや、猫が出た（強立即）

◆ 錯誤陷阱
・不能用過去形（接辭書形）
・× 家に帰ったや、寝た
・✓ 家に帰るや、寝た（回去就睡）

◆ 變化補充
・語氣差異
・や＝書面／やいなや＝更強烈立即
・考試看到「幾乎同時＋書面」優先選
```

## △がはやいか

```
◆ 語感例句結構
・瞬間連動
・動詞辭書形＋が早いか
・ベルが鳴るが早いか、学生は教室を飛び出した。（鈴一響學生就衝出去）
・彼は席に着くが早いか、すぐパソコンを開いた。（一坐下就開電腦）

◆ 混淆比較
・や／とたん（に）
・や＝書面連續／とたん＝自然發生／が早いか＝幾乎同時強瞬間
・立ち上がるや、外に出た。（連續動作）
　立ち上がるが早いか、走り出した。（幾乎同時）
・ドアを開けたとたん、風が入った。（自然結果）
　ドアを開けるが早いか、猫が飛び出した。（瞬間連動）

◆ 錯誤陷阱
・後項不能表意志（不可自己控制）
・× 家に帰るが早いか、勉強しよう
・✓ 家に帰るが早いか、寝てしまった（一回去就不自覺睡著）

◆ 變化補充
・限制特性
・多用於突發／非控制行為
・考試看到「幾乎同時＋不受控」優先選
```

## ○そばから

```
◆ 語感例句結構
・做了又壞
・動詞辭書形＋そばから
・片付けるそばから、子どもが散らかす（ちらかす）。（才收好又被弄亂）
・覚えるそばから、すぐ忘れてしまう。（才記住又忘）

◆ 混淆比較
・ても／たとたん（に）
・ても＝讓步結果／たとたん＝瞬間變化／そばから＝重複落空
・説明しても、理解してくれない。（做了也沒用）
  説明するそばから、忘れられる。（才講又忘）
・立ち上がったとたん、めまいがした。（瞬間發生）

◆ 錯誤陷阱
・不能用一次性事件（需反覆）
・× ドアを開けるそばから、風が入った
・✓ 注意するそばから、またミスする（反覆發生）

◆ 變化補充
・語氣重點
・帶無奈／控制不了
・考試看到「反覆＋剛做就失效」優先選
```

## △なり

```
◆ 語感例句結構
・一做就立刻
・動詞辭書形＋なり
・彼はドアを開けるなり、飛び出した。（一開門就衝出去）
・席に着くなり、すぐスマホを見始めた。（一坐下就看手機）

◆ 混淆比較
・が早いか／や
・が早いか＝非控制瞬間／や＝書面連續／なり＝主體立刻行動
・ベルが鳴るが早いか、飛び出した。（幾乎同時）
　ベルが鳴るなり、飛び出した。（自己立刻行動）
・立ち上がるや、外に出た。（書面連續）
　立ち上がるなり、歩き出した。（主動立即）

◆ 錯誤陷阱
・主語需一致（同一人動作）
・× 彼が帰るなり、私は寝た
・✓ 彼は帰るなり、寝た（他一回去就睡）

◆ 變化補充
・語氣重點
・多用於人主動行動
・考試看到「同一人＋立刻做下一步」優先選
```

## ◎この～というもの／ここ～というもの

```
◆ 語感例句結構
・最近一直
・この／ここ＋期間＋というもの
・ここ一週間というもの、ずっと忙しい。（這一週一直很忙）
・この数日（すうじつ）というもの、あまり寝ていない。（這幾天都沒怎麼睡）

◆ 混淆比較
・最近／このごろ
・最近＝單點印象／このごろ＝模糊近期／というもの＝這段期間持續
・最近、忙しい。（單純最近）
　ここ一週間というもの、忙しい。（這一整段都忙）
・このごろ、よく眠れない。（最近這陣子）
　この数日というもの、眠れていない。（持續狀態）

◆ 錯誤陷阱
・不能接未限定期間名詞
・× この時間というもの、忙しい
・✓ この数日というもの、忙しい（需具體期間）

◆ 變化補充
・語氣重點
・帶「整段期間持續同狀態」
・考試看到「這段期間一直…」優先選
```

## ○を皮切りに（して） （かわきり）

```
◆ 語感例句結構
・以此開端
・名詞＋を皮切りに（して）
・東京公演を皮切りに、全国ツアーが始まる。（以東京為起點展開巡演）
・新商品を皮切りに、次々(つぎつぎ)と新作が発売された。（以新品為開端陸續推出）

◆ 混淆比較
・をきっかけに／から
・きっかけ＝引發變化／から＝單純起點／皮切り＝連續展開起點
・留学をきっかけに、考え方が変わった。（契機改變）
　東京公演を皮切りに、ツアーが続く。（連續展開）
・朝から、仕事を始める。（單純開始）

◆ 錯誤陷阱
・後項需有「連續發展」
・× 試験を皮切りに、勉強した
・✓ 試験を皮切りに、イベントが続く（後面要接一連串）

◆ 變化補充
・語氣重點
・常搭「次々と／続く／始まる」
・考試看到「一開始→後面一連串」優先選
```

## ◎に至るまで  （にいたる）

```
◆ 語感例句結構
・範圍極限
・名詞＋に至るまで
・日常生活に至るまで、すべて管理されている。（連日常都被管理）
・細かい点に至るまで、しっかり確認した。（連細節都確認）

◆ 混淆比較
・まで／にわたって
・まで＝單純到…為止／にわたって＝長時間範圍／に至るまで＝連極端細節都包含
・夜まで働いた。（到晚上為止）
　細部に至るまで、チェックした。（連細節）
・数年間にわたって、研究した。（長期間）

◆ 錯誤陷阱
・不能用於普通範圍（需強調極端）
・× 駅に至るまで、歩いた
・✓ 細部に至るまで、確認した（強調連細節）

◆ 變化補充
・語氣重點
・帶「連…都不放過」
・考試看到「從大到小／連細節」優先選
```

## ○を限りに （かぎりに）

```
◆ 語感例句結構
・以此為止
・名詞＋を限りに
・今日を限りに、タバコをやめる。（從今天起戒菸）
・今月を限りに、このサービスは終了します。（本月起停止服務）

◆ 混淆比較
・まで／をもって
・まで＝單純期限／をもって＝正式截止／を限りに＝以此為分界改變
・今日まで働く。（做到今天為止）
　今日を限りに、辞める。（從今天起不再做）
・本日をもって、終了いたします。（正式公告）

◆ 錯誤陷阱
・不能接動詞（只接名詞時間）
・× 辞めるを限りに、会社を出た
・✓ 今日を限りに、会社を辞める（以今天為界）

◆ 變化補充
・語氣重點
・帶「之後完全改變／不再」
・考試看到「以今天為界→之後不同」優先選
```

# 同時・並行・連續

## ○ともなく／ともなしに

```
◆ 語感例句結構（必出）
・不自覺模糊做
・Vる＋ともなく／ともなしに
・テレビを見るともなく見ていたら、寝ちゃった。
　（無意識地看著電視，結果就睡著了）
    見るともなく = 不是刻意看／隨便看
    見ていたら = 正在看的時候（就發生了某事）
        動詞た形＋ら (做完就發現)
        テレビを見ていたら、寝ちゃった。（看著電視結果睡著了）
・音楽を聞くともなしに聞いていたら、昔のことを思い出した。
　（不知不覺聽著音樂，就想起以前的事）

◆ 混淆比較（必出｜選項核心）
・ながら／つつ／ていると
・有無意識差：ともなく＝無目的發生；ながら＝有意同時
・音楽を聞きながら勉強する。
　（邊聽音樂邊讀書）vs 音楽を聞くともなく聞いていた。
　（無意識地聽著音樂）
・スマホを見ていると眠くなる。
　（一直看手機就會想睡）vs スマホを見るともなく見ていた。
　（沒特別目的地滑手機）

◆ 錯誤陷阱（必出｜考試重點）
・接續錯：不能接意志動作（有目的）
・× 勉強するともなくする。
・✓ 勉強するともなくしていた。（無意識地在讀書）

◆ 變化補充（可省略）
・ともなく vs ともなしに
・兩者幾乎可互換（偏書面感差）
・考試不用糾結形式，重點在「無目的性」
```

## △かたがた

```
◆ 語感例句結構（必出）
・順便兼帶目的
・N＋かたがた
・散歩かたがた、コンビニに寄ってきた。
　（順便散步，也去了一下便利商店）
・お礼かたがた、ご挨拶に伺いました。（うかがいました）
　（順便道謝，也前來打招呼）

◆ 混淆比較（必出｜選項核心）
・ついでに／がてら
・正式度＋主目的：かたがた＝正式＋後項主目的
・買い物ついでに銀行に行った。
　（順便去銀行）vs お礼かたがた、ご挨拶に伺いました。
　（正式場合順便致意）
・散歩がてら写真を撮る。
　（邊散步邊拍照）vs 散歩かたがた、ご近所に挨拶した。
　（以打招呼為主，散步只是附帶）

◆ 錯誤陷阱（必出｜考試重點）
・接續錯：只能接名詞（多為漢語名詞）
・× 買い物するかたがた出かけた。
・✓ 買い物かたがた出かけた。（順便去買東西）

◆ 變化補充（可省略）
・語境限制
・多用於書信／正式拜訪場景
・看到敬語或正式語氣幾乎直接選
```

## ○かたわら

```
◆ 語感例句結構（必出）
・一邊主業一邊
・Vる／Nの＋かたわら
・彼は会社で働くかたわら、夜は日本語を勉強している。
　（他一邊上班，一邊晚上學日文）
・作家のかたわら、大学で講義もしている。
　（身為作家的同時，也在大學授課）

◆ 混淆比較（必出｜選項核心）
・ながら／つつ／合間に
・長期副線：かたわら＝主業＋副業並行；ながら＝同時動作
・音楽を聞きながら勉強する。
　（邊聽音樂邊讀書）vs 働くかたわら資格の勉強をしている。
　（工作之餘長期在準備證照）
・仕事の合間に運動する。
　（工作空檔運動）vs 仕事のかたわら副業をしている。
　（主業之外長期做副業）

◆ 錯誤陷阱（必出｜考試重點）
・短時間誤用：不能用於瞬間／短暫並行
・× ご飯を食べるかたわらテレビを見る。
・✓ ご飯を食べながらテレビを見る。（吃飯時看電視）

◆ 變化補充（可省略）
・語意限制
・多用於「職業／身分＋另一活動」
・看到長期並行（主＋副）優先選
```

## ○がてら

```
◆ 語感例句結構（必出）
・順路順便做
・Vる／N＋がてら
・散歩がてら、コーヒーを買ってきた。
　（散步順便買了咖啡）
・買い物がてら、銀行にも寄った。
　（去買東西順便也去了銀行）

◆ 混淆比較（必出｜選項核心）
・ついでに／かたがた
・日常順路：がてら＝輕鬆順便；ついでに＝純附帶；かたがた＝正式主目的
・スーパーに行くついでに薬局にも行く。
　（順便去藥局）vs 買い物がてら銀行に寄る。
　（出門買東西順路去銀行）
・お礼かたがた、ご挨拶に伺いました。
　（正式順便致意）vs 散歩がてら写真を撮る。
　（輕鬆順便拍照）

◆ 錯誤陷阱（必出｜考試重點）
・動作限制：不可接嚴肅／重大目的
・× 契約交渉がてら会社に行く。
・✓ 契約交渉のついでに会社に行く。（順便去公司）

◆ 變化補充（可省略）
・語境傾向
・多用於輕鬆日常行為（散步／購物）
・看到輕鬆＋順路幾乎直接選
```

# 原因・目的・結果

## **◎ べく**

```
◆ 語感例句結構（必出）
・語感：強意志目的
・結構：V辞書形＋べく（※多為「～べくして／～べく努力する」）
・彼は成功するべく、毎日めっちゃ努力している。（為了成功每天拼命努力）
・問題を解決すべく、みんなで話し合った。（為了解決問題大家一起討論）

◆ 混淆比較（必出｜選項核心）
・對手文法：ために／ように
・一句話差異：べく＝強意志控制／ために＝一般目的／ように＝非控制
・彼は合格するべく勉強した。（自己意志很強）
　彼は合格のために勉強した。（單純目的）
・忘れないようにメモした。（避免忘記，非意志控制）（「忘れない」是狀態，本身難以直接用意志控制）
　忘れないべくメモした。（不自然×）

◆ 錯誤陷阱（必出｜考試重點）
・NG用法說明：不能用在「非意志／自然結果」
・雨が降るべく出かけなかった。（×）
・雨が降りそうなので出かけなかった。（✓ 因為看起來會下雨）

◆ 變化補充（可省略）
・分類：古風書面
・影響答案的理由：日常幾乎不用，考試辨識用
・實戰建議：看到強烈「為了達成」＋自己控制 → 選べく
```

## **△ んがため（に）**

```
◆ 語感例句結構
・強目的執行
・Vない形＋んがため（に）
・成功せんがために、彼は毎日努力し続けた。／為了成功，他每天持續努力
・勝たんがために、手段を選ばなかった。／為了贏，他不擇手段
		文語（古典）
		成功せぬ
		勝たぬ

◆ 混淆比較
・んがため／ため（に）／ように
・んがため＝強烈目的＋意志強；ため＝一般目的；ように＝非控制目標
・合格せんがために、必死に勉強した。／為了考上拼命讀書（強意志）
・合格するために、勉強した。／為了考上而讀書（普通）

◆ 錯誤陷阱
・接續錯誤，不能用ます形或普通現在形
・× 成功しますんがために、努力した。
・✓ 成功せんがために、努力した。／為了成功而努力

◆ 變化補充
・語氣特徵
・非常書面、古風，多見於文章或演說
・考試看到「不擇手段也要達成」優先んがため
```

## **◎ ゆえ（に／の）**

```
◆ 語感例句結構（必出）
・語感：書面原因強調
・結構：普通形＋ゆえに／N＋の＋ゆえ（に）
・経験不足ゆえに、この結果になった。（因為經驗不足才變這樣）
・事故のゆえ、電車が遅れている。（因為事故電車延誤）

◆ 混淆比較（必出｜選項核心）
・對手文法：から／ので／ために
・一句話差異：ゆえに＝正式書面強原因／から＝主觀／ので＝客觀柔和
・時間がないから行かない。（主觀理由）
　時間がないゆえに行けない。（書面強調原因）
・雨なので中止です。（客觀說明）
　雨のゆえ中止となった。（正式書面）

◆ 錯誤陷阱（必出｜考試重點）
・NG用法說明：不能用於日常口語
・お腹空いたゆえにラーメン行こう。（×）
・お腹空いたからラーメン行こう。（✓ 因為餓了去吃拉麵）

◆ 變化補充（可省略）
・分類：硬書面語
・影響答案的理由：看到公告／法律文優先選
・實戰建議：口語直接排除，用於書面判斷
```

## **○ ことだし**

```
◆ 語感例句結構（必出）
・語感：順便理由提議
・結構：普通形＋ことだし（後接建議／打算）
・天気もいいことだし、ちょっと散歩行こう。（天氣也不錯，順便去走走吧）
・時間もあることだし、この店入ってみよう。（也有時間，就順便進這家店看看）

◆ 混淆比較（必出｜選項核心）
・對手文法：し／から
・一句話差異：ことだし＝拿現成理由順勢提議／し＝列舉理由／から＝單一原因
・雨も降ってるし、寒いし帰ろう。（列舉理由）
　雨も降ってることだし、帰ろう。（抓一個理由順勢提議）
・疲れたから帰る。（單一原因）
　疲れたことだし帰ろう。（順勢決定）

◆ 錯誤陷阱（必出｜考試重點）
・NG用法說明：不能接「純敘述結果」
・時間もあることだし、映画を見た。（×）
・時間もあることだし、映画を見よう。（✓ 因為有時間就去看吧）

◆ 變化補充（可省略）
・分類：口語偏多
・影響答案的理由：後句幾乎一定是「提議／意志」
・實戰建議：看到～よう／～う／～たら → 高機率
```

## **△ こととて**

```
◆ 語感例句結構
・理由客觀辯解
・普通形／N＋の＋こととて
・急なこととて、準備が間に合いませんでした。／因為太突然，來不及準備
・子供のしたこととて、許してあげてください。／畢竟是小孩做的，請原諒他吧

◆ 混淆比較
・こととて／ので／から
・こととて＝客觀理由＋帶辯解/請求；ので・から＝一般原因

◆ 錯誤陷阱
・多用於道歉／請求語境，不適合接強命令或直接斷定
・× 忙しいこととて、早くやれ。
・✓ 忙しいこととて、ご理解いただけますと幸いです。／因為很忙，還請理解

◆ 變化補充
・語氣特徵
・非常書面、禮貌緩和語氣
・考試看到「為了說明理由＋請求體諒」優先こととて
```

## **△ てまえ**

```
◆ 語感例句結構
・立場束縛行動
・Vた形／N＋の＋てまえ
・約束したてまえ、途中でやめるわけにはいかない。／既然答應了，就不能半途而廢
・上司の手前、文句は言えなかった。／在上司面前，不好抱怨

◆ 混淆比較
・てまえ／以上／からには
・てまえ＝顧及面子/立場；以上・からには＝責任必然
・約束したてまえ、断れない。／因為已經答應了（顧面子）所以不能拒絕
・約束した以上、断れない。／既然答應了（責任）就不能拒絕

◆ 錯誤陷阱
・不可用未發生或純假設行為，需是既成事實或既有立場
・× 約束するてまえ、頑張る。
・✓ 約束したてまえ、頑張る。／既然已經答應了，就會努力

◆ 變化補充
・語氣特徵
・常帶「不好意思／顧慮他人眼光」的壓力感
・考試看到「顧面子不得不…」優先てまえ
```

## **○ とあって**

```
◆ 語感例句結構（必出）
・語感：特殊原因結果
・結構：普通形＋とあって（後接自然結果）
・週末とあって、この店めっちゃ混んでいる。（因為是週末所以很擠）
・人気店とあって、開店前から並んでいる。（因為是人氣店早早就排隊）

◆ 混淆比較（必出｜選項核心）
・對手文法：ので／から
・一句話差異：とあって＝特殊狀況導致理所當然結果／ので＝一般原因
・週末なので混んでいる。（單純原因）
　週末とあって混んでいる。（因為是週末→理所當然很擠）
・雨だから中止した。（主觀判斷）

・NG用法說明：不能接「主觀意志／命令」
・休日とあって、早く帰ろう。（×）

◆ 變化補充（可省略）
・分類：偏書面描述
・影響答案的理由：後句多為「狀態結果」不是動作決定
・實戰建議：看到人多／熱門／特殊日子 → 優先選
```

## **△ ばこそ**

# 推測・判斷・可能性

## **◎ （よ）うにも～ない**

```
◆ 語感例句結構
・想做卻無法
・V意向形＋にも～ない
・行こうにも、電車が止まっていて行けない。／想去但電車停了去不了
・説明しようにも、時間が足りない。／想說明但時間不夠

◆ 混淆比較
・ようにも～ない／たくても～ない／られない
・ようにも～ない＝客觀條件卡住；たくても～ない＝主觀願望受阻；られない＝單純不能
・会いに行こうにも、仕事があって無理だ。／想去見但有工作卡住（客觀）
・会いたくても、行けない。／想見但去不了（主觀願望）

◆ 錯誤陷阱
・必須用意向形（～よう），不能用普通形或たい形
・× 行くにも、時間がない。
・✓ 行こうにも、時間がない。／想去但沒時間

◆ 變化補充
・語氣特徵
・常接「どうにも～ない」強化無力感
・考試看到「想做但卡住做不到」優先ようにも～ない
```

## **◎ にたえる／にたえない**

```
◆ 語感例句結構
・值得與不堪
・N／V辞書形＋にたえる／にたえない
・この映画は見るにたえる内容だ。／這部電影是值得一看的內容
・彼の態度は見るにたえない。／他的態度讓人看不下去

◆ 混淆比較
・にたえる／にたえない／がたい
・にたえる＝值得承受/評價；にたえない＝不堪忍受；がたい＝心理上難以接受
・彼のスピーチは聞くにたえる。／他的演講值得一聽
・彼のスピーチは聞きがたい。／他的演講讓人難以接受（心理抗拒）

◆ 錯誤陷阱
・多用固定形式「見るにたえない／聞くにたえない」，不能亂換動詞
・× 食べるにたえない。
・✓ 見るにたえない。／不堪入目

◆ 變化補充
・語氣特徵
・偏書面、評價強烈，常用於負面（にたえない）
・考試看到「不忍直視／不堪入耳」優先にたえない
```

## **○ （か）と思いきや**

```
◆ 語感例句結構
・以為結果相反
・普通形＋（か）と思いきや
・帰ったかと思いきや、また戻ってきた。／以為他回去了，結果又回來了
・簡単だと思いきや、めっちゃ難しかった。／以為很簡單，結果超難

◆ 混淆比較
・と思いきや／と思ったら／のに
・と思いきや＝預期→完全相反（落差強）；と思ったら＝立刻變化；のに＝單純逆接
・終わったと思いきや、まだ続きがあった。／以為結束了，結果還有後續（落差）
・終わったと思ったら、すぐ次が始まった。／剛以為結束，馬上開始下一個（時間接續）

◆ 錯誤陷阱
・前項多為過去或已判斷內容，不能用命令或未形成認知的句子
・× 行こうと思いきや、急げ。
・✓ 行けると思いきや、無理だった。／以為能去，結果不行

◆ 變化補充
・語氣特徵
・常帶驚訝或吐槽感，偏口語但考試常出
・看到「以為…結果完全相反」優先と思いきや
```

## **◎ とは**

```
◆ 語感例句結構
・竟然出乎意料
・普通形＋とは（思わなかった／驚いた）
・こんなところで会うとは、思わなかった。／竟然在這種地方遇到，沒想到
・彼が犯人だったとは、驚いた。／他竟然是犯人，太驚訝了

◆ 混淆比較
・とは／なんて／とはいえ
・とは＝客觀驚訝；なんて＝較主觀情緒強；とはいえ＝讓步轉折
・彼が来るとは、思わなかった。／沒想到他會來（客觀驚訝）
・彼が来るなんて、信じられない。／他竟然會來，難以置信（情緒強）

◆ 錯誤陷阱
・常接「思わなかった／知らなかった／驚いた」，不能接命令或請求
・× 彼が来るとは、待ってください。
・✓ 彼が来るとは、知らなかった。／不知道他會來

◆ 變化補充
・語氣特徵
・多用於感嘆句尾，帶「沒想到」語感

・日本はいうまでもなく、海外でも人気だ。（日本不用說，海外也受歡迎）

◆ 混淆比較
・はいうまでもなく／はもちろん／までもなく
・一句話差異：いうまでもなく＝理所當然強調／もちろん＝一般強調／までもなく＝不必到那程度
・彼はいうまでもなく有名だ。（強理所當然）
　彼はもちろん有名だ。（一般）
・言うまでもなく分かる。（不用做到）

◆ 錯誤陷阱
・常搭「も」形成擴展
・× 日本はいうまでもなく有名だ（略不自然）
・✓ 日本はいうまでもなく海外でも有名だ（對應）

◆ 變化補充
・書面語
・語氣偏強
・看到「不用說…連…也」優先選
```

## **◎ ならいざしらず**

```
◆ 語感例句結構
・語感：那種還好但
・結構：N／普通形＋ならいざしらず
・子どもならいざしらず、大人はダメだ。（小孩還好，大人就不行）
・昔ならいざしらず、今は通用しない。（以前還好，現在不行）

◆ 混淆比較
・ならいざしらず／はまだしも／どころか
・一句話差異：ならいざしらず＝A可接受→B不行（較書面）／はまだしも＝同義較口語／どころか＝反而更糟
・初心者ならいざしらず経験者はダメ。（書面）
　初心者はまだしも経験者はダメ。（口語）
・楽どころか大変だ。（反轉）

◆ 錯誤陷阱
・前項需為「較可接受條件」
・× 大人ならいざしらず子どもはダメ（邏輯反）
・✓ 子どもならいざしらず大人はダメ（順序）

◆ 變化補充
・書面語
・語氣較硬
・看到「還好…但…不行」優先選
```

# 補充

## **○ がい／かいがある**

```
◆ 語感例句結構
・語感：值得有回報
・結構：Vます形＋がいがある／Vた形＋かいがある
・この仕事はやりがいがある。（這工作很有價值）
・努力したかいがあった。（努力有回報）

◆ 混淆比較
・がいがある／だけのことはある／むだだ
・一句話差異：がいがある＝做這件事本身值得／だけのことはある＝結果證明值得／むだだ＝沒有價值
・やりがいがある仕事。（本身）
　努力しただけのことはあって成功した。（結果）
・やってもむだだ。（無價值）

◆ 錯誤陷阱
・「かいがある」多用過去結果
・× 頑張るかいがある
・✓ 頑張ったかいがある（結果）

◆ 變化補充
・やりがい（固定）
・常見名詞化
・看到「值得做／有回報」優先選
```

## **△ ぶり／っぷり**

```
◆ 語感例句結構
・語感：那種樣子
・結構：N／Vます形＋ぶり／っぷり
・彼の働きぶりは見事だ。（他的工作表現很出色）
・彼女の食べっぷりに驚いた。（她的吃相讓人驚訝）

◆ 混淆比較
・ぶり／っぷり／ようす
・一句話差異：ぶり＝客觀樣態／っぷり＝誇張生動（口語）／ようす＝中性描述
・話しぶりが落ち着いている。（客觀）
　話しっぷりが豪快だ。（誇張）
・様子は普通だ。（中性）

◆ 錯誤陷阱
・っぷり多接「動作強烈」
・× 静かっぷり
・✓ 飲みっぷり／食べっぷり（動作）

◆ 變化補充
・ぶり（時間）
・如：三年ぶり
・需分開記
```

## **◎ なり／なりの**

```
◆ 語感例句結構
・語感：照那立場
・結構：N＋なり／N＋なりの＋N
・子どもなりに考えている。（用小孩的方式在思考）
・子どもなりの理由がある。（有屬於小孩的理由）

◆ 混淆比較
・なり／なりの／なりに
・一句話差異：なり＝依該立場去做（副詞）／なりの＝該立場的性質（名詞修飾）／なりに＝依自己程度
・親なりに判断した。（立場）
　親なりの責任。（性質）
・自分なりにやる。（自己）

◆ 錯誤陷阱
・なり不可單獨修飾名詞
・× 子どもなり理由
・✓ 子どもなりの理由（需の）

◆ 變化補充
・語氣偏客觀
・常見評論句
・考試常考三者差
```

## **○ まま**

```
◆ 語感例句結構
・語感：保持原樣
・結構：Nの／イ形容詞い／ナ形容詞な＋まま／Vた形＋まま
・ドアを開けたまま出かけた。（門開著就出門了）
・昔のまま、何も変わっていない。（保持原樣完全沒變）

◆ 混淆比較
・まま／ままで／ままに
・一句話差異：まま＝狀態維持／ままで＝保持狀態去做／ままに＝順從不控制
・服を着たまま寝た。（狀態維持）
　そのままでいいよ。（維持狀態）
・流れのままに任せる。（順其自然）

◆ 錯誤陷阱
・動詞需用「た形」表已成狀態
・× 食べるまま寝た
・✓ 食べたまま寝た（狀態成立）

◆ 變化補充
・そのまま
・固定常用
・看到「就那樣」優先選
```
