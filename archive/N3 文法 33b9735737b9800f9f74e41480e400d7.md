# N3 文法

# 時間・時機

## て以来

```
◆ 語感例句結構（必出）
・語感：起點後持續
・結構：Vて＋以来（いらい）
・日本に来て以来、日本語を毎日使っている。（自從來日本後，一直每天用日文）
・あの事件があって以来、彼は外に出なくなった。（那件事之後，他就不出門了）

◆ 混淆比較（必出｜選項核心）
・對手文法：てから／後で
・一句話差異：「以来」＝持續變化，「てから」＝單純先後
・日本に来てから、日本語を勉強した。（來日本之後才開始學）
　vs 日本に来て以来、日本語を使い続けている。（來之後一直在用）
・卒業してから働いた。（畢業後開始工作）
　vs 卒業して以来、ずっと同じ会社で働いている。（畢業後一直在同一公司）

◆ 錯誤陷阱（必出｜考試重點）
・NG：後面不能接「單次動作」
・× 日本に来て以来、旅行した。
・✓ 日本に来て以来、旅行するようになった。（來之後變得會去旅行）

◆ 變化補充（可省略）
・分類：時間持續
・影響答案的理由：是否強調「一直持續到現在」
・實戰建議：看到「ずっと／今も」幾乎就是以来
```

## 最中（ さいちゅう ）に／最中だ

```
◆ 語感例句結構（必出）
・語感：正中進行中
・結構：Vている／名詞＋の＋最中に／最中だ
・今、会議の最中だ。あとでかけ直すね。（現在正在開會中，等下再打）
・シャワーを浴びている最中に、電話が鳴った。（正在洗澡時，電話響了）

◆ 混淆比較（必出｜選項核心）
・對手文法：ところに／間に
・一句話差異：「最中」＝強調事件正中間，「ところ」＝時間點瞬間
・ご飯を食べている最中に電話が来た。（吃到一半被打斷）
　vs ご飯を食べているところに電話が来た。（正好在那一刻來電）
・授業の最中に寝た。（上課進行中睡著）
　vs 授業の間に寝た。（整段時間內某時睡著）

◆ 錯誤陷阱（必出｜考試重點）
・NG：不能接「非進行狀態動詞」
・× 知っている最中に、電話が来た。
・✓ 勉強している最中に、電話が来た。（正在做某動作中）

◆ 變化補充（可省略）
・分類：進行強調
・影響答案的理由：是否要表「中途被打斷」
・實戰建議：出現「突然／電話／被打斷」優先選最中
```

## たとたん（に）

```
◆ 語感例句結構（必出）
・語感：瞬間立刻發生
・結構：Vた＋とたん（に）
・ドアを開けたとたん、猫が飛び出してきた。（一打開門，貓就衝出來）
・席に座ったとたん、急に眠くなった。（一坐下就突然想睡）

◆ 混淆比較（必出｜選項核心）
・對手文法：たら／てから
・一句話差異：「とたん」＝瞬間不可控，「たら」＝條件結果
・ドアを開けたとたん、風が吹き込んだ。（一開門瞬間風吹進來）
　vs ドアを開けたら、風が入ってきた。（開了之後才發現有風）
・席に座ったとたん、寝てしまった。（坐下瞬間就睡著）
　vs 席に座ってから、寝た。（坐下後過一段才睡）

◆ 錯誤陷阱（必出｜考試重點）
・NG：不能用於「可控制行為」
・× 家に帰ったとたん、勉強した。
・✓ 家に帰ったとたん、雨が降り出した。（回家瞬間就下雨）

◆ 變化補充（可省略）
・分類：瞬間連續
・影響答案的理由：是否為「意料外、自動發生」
・實戰建議：看到「突然／一下子」優先選とたん
```

## 際／際に（は）

```
◆ 語感例句結構（必出）
・語感：正式場合時
・結構：Vる／Vた／名詞＋の＋際（に／は）
・ご利用の際は、身分証をご提示ください。（使用時請出示證件）
・日本に行く際に、保険に入った。（去日本時辦了保險）

◆ 混淆比較（必出｜選項核心）
・對手文法：時に／とき
・一句話差異：「際」＝正式書面，「とき」＝一般日常
・申請の際は、この書類が必要です。（正式說明）
　vs 申請するとき、この書類が必要です。（日常描述）
・外出の際に、鍵を確認してください。（提醒／規範）
　vs 外出するとき、鍵を確認してね。（口語提醒）

◆ 錯誤陷阱（必出｜考試重點）
・NG：隨便口語情境使用會不自然
・× 友達に会う際、一緒にゲームした。
・✓ 友達に会うとき、一緒にゲームした。（日常應用用とき）

◆ 變化補充（可省略）
・分類：書面語
・影響答案的理由：句子是否偏「通知／規則／公告」
・實戰建議：看到「ご利用／申請／注意事項」優先選際
```

## ところ（に／へ／を）

```
◆ 語感例句結構（必出）
・語感：時點被碰到
・結構：Vている＋ところ（に／へ／を）
・出かけようとしているところに、友達が来た。（正要出門時被朋友來打斷）
・考えているところへ、いいアイデアが浮かんだ。（正在想時突然有好點子）

◆ 混淆比較（必出｜選項核心）
・對手文法：最中に／たとたん
・一句話差異：「ところ」＝抓時間點，「最中」＝過程中段，「とたん」＝瞬間結果
・ご飯を食べているところに電話が来た。（剛好那一刻）
　vs ご飯の最中に電話が来た。（吃到一半被打斷）
・席に座ったところへ、店員が来た。（剛坐下那刻）
　vs 席に座ったとたん、眠くなった。（坐下瞬間就變化）

◆ 錯誤陷阱（必出｜考試重點）
・NG：不能接「非動作狀態」
・× 知っているところに、電話が来た。
・✓ 考えているところに、電話が来た。（正在做動作中）

◆ 變化補充（可省略）
・分類：助詞差異
・影響答案的理由：「に」＝被影響、「へ」＝動作朝向、「を」＝經過場景
・實戰建議：被打斷選に，突發加入選へ，動作經過用を
```

## うちに

```
◆ 語感例句結構（必出）
・語感：趁變化前
・結構：Vる／Vている／い形／な形＋な＋うちに
・若いうちに、海外に行っておきたい。（趁年輕想先出國）
・温かいうちに食べてね。（趁還熱的時候吃）

◆ 混淆比較（必出｜選項核心）
・對手文法：間に
・一句話差異：「うちに」＝狀態會改變，「間に」＝單純期間
・日本にいるうちに、たくさん旅行したい。（趁還在日本）
　vs 日本にいる間に、旅行した。（在那段期間有做）
・暗くならないうちに帰ろう。（趁還沒變暗）
　vs 暗い間に帰った。（在黑的期間內回去）

◆ 錯誤陷阱（必出｜考試重點）
・NG：不能用於「已經固定不變」
・× 日本に住んでいるうちに、東京に住んでいる。
・✓ 日本に住んでいる間に、東京に住んでいる。（單純描述期間）

◆ 變化補充（可省略）
・分類：變化前時機
・影響答案的理由：是否含「之後會改變／來不及」
・實戰建議：看到「まだ／早く／今のうち」優先選うちに
```

## までに（は）

```
◆ 語感例句結構（必出）
・語感：期限前完成
・結構：時間／Vる＋までに（は）
・明日までに、このレポートを出してください。（請在明天之前交報告）
・30歳になるまでに、家を買いたい。（想在30歲前買房）

◆ 混淆比較（必出｜選項核心）
・對手文法：まで／うちに
・一句話差異：「までに」＝截止點前完成，「まで」＝持續到那一刻
・5時までに帰る。（5點前要完成回家）
　vs 5時まで働く。（工作到5點）
・雨が降るまでに帰ろう。（下雨前完成回家）
　vs 雨が降るまで待つ。（等到下雨為止）

◆ 錯誤陷阱（必出｜考試重點）
・NG：不能接「持續動作」
・× 5時までに働く。
・✓ 5時まで働く。（持續用まで）

◆ 變化補充（可省略）
・分類：截止期限
・影響答案的理由：是否強調「完成點」
・實戰建議：看到「提出／完成／準備好」優先選までに
```

## たび（に）

```
◆ 語感例句結構（必出）
・語感：每次必發生
・結構：Vる／名詞＋の＋たび（に）
・この曲を聞くたびに、あの人を思い出す。（每次聽這首歌都會想起那個人）
・出張のたびに、お土産を買ってくる。（每次出差都會買伴手禮）

◆ 混淆比較（必出｜選項核心）
・對手文法：とき／ごとに
・一句話差異：「たび」＝每次都伴隨情緒／變化，「ごとに」＝單純規律重複
・彼に会うたびに、ドキドキする。（每次見他都會心跳加速）
　vs 彼に会うとき、緊張する。（只是描述某時候）
・ページをめくるたびに、話が進む。（每翻一頁就推進）
　vs 10ページごとに、休憩する。（每10頁固定一次）

◆ 錯誤陷阱（必出｜考試重點）
・NG：不能接「單次或未重複事件」
・× 初めて会ったたびに、緊張した。
・✓ 会うたびに、緊張する。（每次都會）

◆ 變化補充（可省略）
・分類：反覆連動
・影響答案的理由：是否有「每次都會這樣」
・實戰建議：看到「いつも／毎回／～すると必ず」優先選たび
```

## から〜にかけて

```
◆ 語感例句結構（必出）
・語感：區間模糊延伸
・結構：時間／地點＋から〜にかけて
・昨日の夜から今朝にかけて、雨が降り続いた。（從昨晚到今天早上一直下雨）
・台北から新北にかけて、強い風が吹いている。（從台北到新北一帶有強風）

◆ 混淆比較（必出｜選項核心）
・對手文法：から〜まで／の間に
・一句話差異：「にかけて」＝範圍模糊，「まで」＝明確終點
・夜から朝にかけて、雪が降った。（大致那段時間）
　vs 夜から朝まで、雪が降った。（精確到早上為止）
・この地域から山にかけて、霧が出ている。（一帶範圍）
　vs この地域の間に、霧が出た。（期間內某時）

◆ 錯誤陷阱（必出｜考試重點）
・NG：不能用於「明確完成點」
・× 9時から10時にかけて会議を終わった。
・✓ 9時から10時まで会議をした。（明確結束用まで）

◆ 變化補充（可省略）
・分類：模糊範圍
・影響答案的理由：是否強調「一帶／大致期間」
・實戰建議：看到天氣／新聞描述幾乎選にかけて
```

# 原因・理由・結果

## おかげで（だ）

```
◆ 語感例句結構（必出）
・因好結果感謝
・Vた／Aい／Nの＋おかげで（だ）
・先生のおかげで、試験に合格できた。
　（多虧老師，我考試通過了）
・毎日練習したおかげで、上手になった。
　（因為每天練習，變得更厲害了）

◆ 混淆比較（必出｜選項核心）
・せいで／ため（に）
・結果正負：おかげで＝好結果；せいで＝壞結果；ために＝中性
・雨のせいで、試合が中止になった。
　（因為下雨導致取消）vs 雨のおかげで、涼しくなった。
　（因為下雨變涼快）
・事故のため、電車が遅れた。
　（中性原因）vs 先生のおかげで、理解できた。
　（正面結果）

◆ 錯誤陷阱（必出｜考試重點）
・語氣錯：不可用於壞結果（反諷除外）
・× 風邪をひいたおかげで、会社を休んだ。
・✓ 風邪をひいたせいで、会社を休んだ。（因為感冒請假）

◆ 變化補充（可省略）
・形式差異
・おかげだ＝句末斷定
・看到「感謝＋好結果」優先選
```

## せいで（だ）／せいか

```
◆ 語感例句結構（必出）
・因壞結果責怪
・Vた／Aい／Nの＋せいで（だ）／せいか
・雨のせいで、試合が中止になった。
　（因為下雨，比賽取消了）
・寝不足のせいか、今日はずっと頭がぼーっとする。
　（可能是睡眠不足，今天一直昏昏沉沉）

◆ 混淆比較（必出｜選項核心）
・おかげで／ため（に）
・語氣差：せいで＝負面原因；せいか＝不確定推測；おかげで＝好結果
・彼のせいで、計画が遅れた。
　（因為他導致延誤）vs 彼のおかげで、計画が進んだ。
　（多虧他順利進行）
・事故のため、電車が遅れた。
　（中性原因）vs 寝不足のせいか、集中できない。
　（可能因為睡不夠）

◆ 錯誤陷阱（必出｜考試重點）
・語氣誤用：せいで不可用於好結果
・× 先生のせいで、合格できた。
・✓ 先生のおかげで、合格できた。（多虧老師）

◆ 變化補充（可省略）
・細微差異
・せいか＝說話者不確定（體感推測）
・看到「不確定原因」優先せいか
```

## につき

```
◆ 語感例句結構（必出）
・公告理由限制
・N＋につき
・本日は定休日につき、休業いたします。
　（今天因為是公休日，暫停營業）
・満席につき、ご入場できません。
　（因為客滿，無法入場）

◆ 混淆比較（必出｜選項核心）
・ため（に）／ので／により
・語域差：につき＝公告書面限定；ため／ので＝一般原因
・雨のため、試合は中止です。
　（一般說法）vs 雨天につき、本日は中止といたします。
　（公告語氣）
・機械の故障により運休します。
　（正式書面）vs 故障につき、運休とさせていただきます。
　（公告常用）

◆ 錯誤陷阱（必出｜考試重點）
・口語誤用：不可用於日常對話
・× 雨につき、行かない。
・✓ 雨だから、行かない。（因為下雨不去）

◆ 變化補充（可省略）
・語境固定
・常見於通知／店內告示／公文
・看到「公告文」幾乎直接選
```

## によって（は）／により

```
◆ 語感例句結構（必出）
・依條件差異
・N＋によって（は）／により
・人によって、考え方は違う。
　（因人而異，想法不同）
・天候により、試合は中止となる場合があります。
　（依天氣情況，可能會取消比賽）

◆ 混淆比較（必出｜選項核心）
・で／によると／次第で
・功能差：によって＝依條件變化；で＝單純原因；によると＝資訊來源
・材料によって、味が変わる。
　（依材料不同味道會變）vs 材料で味が変わる。
　（單純原因）
・天気予報によると、明日は雨だ。
　（根據預報）vs 努力次第で結果が変わる。
　（取決於）

◆ 錯誤陷阱（必出｜考試重點）
・誤用來源：不可用於「聽說／根據」
・× ニュースによって、地震があった。
・✓ ニュースによると、地震があった。（根據新聞）

◆ 變化補充（可省略）
・形式差異
・によっては＝部分例外／により＝書面正式
・看到「公告書面」偏用により
```

## ものだから／もので

```
◆ 語感例句結構（必出）
・主觀理由辯解
・Vた／Aい／Nな＋ものだから／もので
・急いでいたものだから、連絡できなかった。
　（因為我很趕，所以沒聯絡）
・子どもなもので、よく分からないんです。
　（因為還是小孩，所以不太懂）

◆ 混淆比較（必出｜選項核心）
・から／ので／ため（に）
・語氣差：ものだから＝帶歉意辯解；ので＝客觀柔和；から＝直接理由
・遅れたので、すみません。
　（客觀說明）vs 遅れたものだから、すみません。
　（帶辯解）
・忙しいから行けない。
　（直接）vs 忙しいもので、行けないんです。
　（委婉解釋）

◆ 錯誤陷阱（必出｜考試重點）
・語境錯：不適合正式公文
・× 事故があったものだから、電車は遅延しております。
・✓ 事故があったため、電車は遅延しております。（正式用ため）

◆ 變化補充（可省略）
・形式差異
・もので＝較柔和口語
・看到「道歉／辯解」優先選
```

## わけだ

```
◆ 語感例句結構（必出）
・理解結論當然
・Vる／Vた／Aい／Nな＋わけだ
・毎日練習しているから、上手なわけだ。
　（每天都有練習，難怪會變厲害）
・彼は日本に10年住んでいる。日本語が上手なわけだ。
　（他住日本10年了，難怪日文很好）

◆ 混淆比較（必出｜選項核心）
・ということだ／はずだ／のだ
・推論確定：わけだ＝已有理由→理解結論；はずだ＝應該如此；ということだ＝轉述整理
・彼は10年日本にいる。上手なわけだ。
　（理解）vs 彼は上手なはずだ。
　（推測應該）
・彼は来ないということだ。
　（聽說）vs 来ないわけだ。
　（理解原因）

◆ 錯誤陷阱（必出｜考試重點）
・誤當推測：不可單獨表「應該」
・× 彼は来るわけだ。（想表應該會來）
・✓ 彼は来るはずだ。（應該會來）
```

# 程度・強調

## くらい（ぐらい）

```
◆ 語感例句結構
・程度下限
・N／V普通形＋くらい（ぐらい）
・この問題は子どもでもできるくらい簡単だ。

・昨日は立てないくらい疲れた。
　昨天累到站不起來

・くらい＝最低程度；ほど＝程度上升更強

　拼命工作（ほど更誇張）

◆ 錯誤陷阱
・誤把くらい當「精確數量」
・× 10人くらいちょうど来た
・✓ 10人くらい来た（約10人）／ちょうど10人来た（剛好10人）

◆ 變化補充
・ぐらい口語
・口語更自然，不影響意思
・日常會話優先用ぐらい
```

## ほど

```
◆ 語感例句結構
・程度上限
・N／V普通形＋ほど

　今天難過到想哭

◆ 混淆比較
・くらい／だけ
・ほど＝程度更強上限；くらい＝下限感

・不能接明確最小限制
・× 少しほど食べた
・✓ 少し食べた／少しだけ食べた（吃一點）

◆ 變化補充
```

## ば〜ほど

```
◆ 語感例句結構

◆ 變化補充
```

```
・気持ちをこめて話す vs 分かるために話す
　前者有感情，後者是目的

◆ 錯誤陷阱
・不能接無情感名詞
・× 石をこめて投げる
・✓ 力をこめて投げる
　（需可投入的「心/力」）

◆ 變化補充
・漢字：込めて
・意思相同
・考試不分寫法
```

# 義務・必要・限制

## ないと／なくちゃ

```
◆ 語感例句結構
・不做不行
・Vないと／Vなくちゃ
・もう行かないと。
　得走了
・早く寝なくちゃ。
　不早點睡不行

◆ 混淆比較
・なければならない／ないといけない
・ないと＝口語義務；なければならない＝正式強制
・行かないと vs 行かなければならない
　前者口語自然
・やらなくちゃ vs やらないといけない
　後者較完整

◆ 錯誤陷阱
・不能直接接名詞
・× 学生ないとだめ
・✓ 学生でないとだめ
　（名詞需でないと）

◆ 變化補充
・なくちゃ＝縮約
・更口語輕快
・會話最常用
```

## ないわけにはいかない

```
◆ 語感例句結構
・不得不做
・Vない＋わけにはいかない
・今日は休むわけにはいかない。
　今天不能請假（不得不去）
・約束だから行かないわけにはいかない。
　因為有約不能不去

◆ 混淆比較
・なければならない／ざるを得ない
・ないわけにはいかない＝情況壓力；ざるを得ない＝被迫無奈
・行かないわけにはいかない vs 行かざるを得ない
　後者更無奈
・やらなければならない vs やらないわけにはいかない
　前者規則義務，後者情境壓力

◆ 錯誤陷阱
・不能用於允許不做
・× 行かないわけにはいく
・✓ 行かないわけにはいかない
　（固定否定結構）

◆ 變化補充
・理由常在前
・多接「因為…」
・考試重點在語氣差
```

## わけにはいかない／わけにもいかない

```
◆ 語感例句結構
・情況限制
・V辞書形／ない形＋わけにはいかない／わけにもいかない
・今日は帰るわけにはいかない。
　今天不能回去（有理由）
・ここで諦めるわけにもいかない。
　在這裡也不能放棄

◆ 混淆比較
・ないわけにはいかない／てはいけない
・わけにはいかない＝情況限制；てはいけない＝規則禁止
・行くわけにはいかない vs 行ってはいけない
　前者因情況不行，後者規則不行
・やらないわけにはいかない vs やらなければならない
　前者情境壓力，後者義務

◆ 錯誤陷阱
・わけにもいかない需語氣轉折
・× 帰るわけにもいかない（無前文）
・✓ 忙しいので帰るわけにもいかない
　（需理由鋪墊）

◆ 變化補充
・にも＝更無奈
・強調兩難
・考試常考語氣差
```

## ほかない／より（ほか）ない

```
◆ 語感例句結構
・只能這樣
・V辞書形＋ほかない／N＋より（ほか）ない
・もう待つほかない。
　只能等了
・彼に頼るよりほかない。
　除了拜託他沒有別的辦法

◆ 混淆比較
・しかない／ざるを得ない
・ほかない＝無選擇；ざるを得ない＝被迫無奈
・やるほかない vs やらざるを得ない
　後者更無奈
・待つしかない vs 待つほかない
　意思近，但ほかない偏書面

◆ 錯誤陷阱
・不能接可能形
・× 行けるほかない
・✓ 行くほかない
　（需直接動詞原形）

◆ 變化補充
・よりほかない較正式
・強調排他唯一
・考試常見書面句
```

## から（に）は

```
◆ 語感例句結構
・既然就要
・V辞書形＋から（に）は
・やるからには最後までやる。
　既然要做就做到最後
・行くからには全力で楽しむ。
　既然要去就全力享受

◆ 混淆比較
・以上（は）／なら
・からには＝決心責任；なら＝單純條件
・やるからには勝つ vs やるなら勝つ
　前者帶覺悟
・行く以上は頑張る vs 行くからには頑張る
　意思近，但前者更書面

◆ 錯誤陷阱
・不能接已發生事實
・× 行ったからには楽しかった
・✓ 行くからには楽しむ
　（需未來決心）

◆ 變化補充
・からには強語氣
・常搭配意志形
・考試重點在「覺悟」
```

# 條件・假定

## さえ〜ば／さえ〜たら

```
◆ 語感例句結構
・只要就行
・N／Vます形＋さえ＋ば／Vたら
・お金さえあれば幸せだ。
　只要有錢就幸福
・君が来てくれたらさえいい。
　只要你來就好

◆ 混淆比較
・だけで／ば〜いい
・さえ〜ば＝最低條件；だけで＝單純條件
・名前さえ分かればいい vs 名前だけで分かる
　前者「只要知道名字就夠」
・合格しさえすればいい vs 合格すればいい
　前者更強調「只要」

◆ 錯誤陷阱
・さえ位置錯誤
・× あればお金さえいい
・✓ お金さえあればいい
　（さえ要接條件主體）

◆ 變化補充
・たら也可用
・語氣較口語
・考試兩者不分
```

## たとえ〜ても

```
◆ 語感例句結構
・就算也不變
・たとえ＋Vても／イ形ても／ナ形でも／Nでも
・たとえ雨でも行く。
　就算下雨也要去
・たとえ失敗しても諦めない。
　就算失敗也不放棄

◆ 混淆比較
・ても／としても
・たとえ〜ても＝強調假設；としても＝讓步推論
・雨でも行く vs たとえ雨でも行く
　後者更強烈
・失敗したとしても大丈夫 vs 失敗しても大丈夫
　前者較理性推論

◆ 錯誤陷阱
・不能接確定事實
・× たとえ昨日雨でも行った
・✓ 昨日雨でも行った／たとえ雨でも行く
　（たとえ需假設）

◆ 變化補充
・常省略たとえ
・口語多直接用ても
・考試常考強調有無
```

## たら／だったら／かったら

```
◆ 語感例句結構
・條件觸發
・Vたら／Nだったら／イ形かったら
・雨が降ったら中止する。
　如果下雨就中止
・時間がなかったら行かない。
　如果沒時間就不去

◆ 混淆比較
・と／ば
・たら＝結果後發；と＝必然結果
・ボタンを押したら音が出た vs 押すと音が出る
　前者一次結果，後者規則
・疲れたら休む vs 疲れると休む
　前者主觀判斷

◆ 錯誤陷阱
・不能用於已知必然
・× 春になったら桜が咲いた
・✓ 春になると桜が咲く
　（規律用と）

◆ 變化補充
・だったら口語強
・帶「那就…」語感
・考試常考語氣差
```

## とすれば／としたら／とする

```
◆ 語感例句結構
・假設前提
・V普通形／Nだ＋とすれば／としたら／とすると
・本当だとしたら大問題だ。
　如果是真的就很嚴重
・彼が来ないとすれば中止だ。
　如果他不來就取消

◆ 混淆比較
・たら／なら
・としたら＝低確定假設；たら＝具體條件
・雨だとしたら行かない vs 雨だったら行かない
　前者偏推測
・彼ならできる vs 彼だとすればできる
　前者判斷，後者假設推論

◆ 錯誤陷阱
・不能接確定事實
・× 彼が来たとしたら会った
・✓ 彼が来たら会う
　（としたら是未確定假設）

◆ 變化補充
・三者語氣差
・とすると較口語
・考試重點在假設程度
```

```jsx
**核心差別（直接判斷用）**

* とすれば → 理性推論（像在分析）
* としたら → 主觀假設（帶情緒）
* とすると → 順接判斷（那接下來）

---

**一組例句直接分辨**

* 彼が犯人だとすれば、動機が必要だ。
  　如果他是犯人，那就需要動機（在推理分析）

* 彼が犯人だとしたら、信じられない。
  　如果他是犯人，我無法相信（情緒反應）

* 彼が犯人だとすると、すぐ対策が必要だ。
  　如果他是犯人，那就得立刻對策（往下判斷）

```

# 規則・習慣・可能性

## ことになっている／こととなっている

```
◆ 語感例句結構
・規則決定
・V辞書形／ない形＋ことになっている／こととなっている
・この会社では私語は禁止することになっている。
　這家公司規定不能聊天

　規定明天7點集合

◆ 混淆比較
・ことにする／ことになる
・ことになっている＝既定規則；ことになる＝結果決定
・禁煙することになっている vs 禁煙することになった
　前者一直如此，後者剛決定
・早く寝ることにする vs 早く寝ることになっている
　前者自己決定

◆ 錯誤陷阱
・不能當臨時決定
・× 明日行くことになっている（剛決定）
・✓ 明日行くことになった（剛決定）
　（～ている是既定狀態）

◆ 變化補充
・となっている較正式
・公告/規定常用
・考試重點在語氣差
```

## ことにしている

```
◆ 語感例句結構

　我規定自己每天運動30分鐘

◆ 混淆比較
・ことにする／ことになっている
・ことにしている＝持續習慣；ことにする＝當下決定
・毎日勉強することにしている vs 今から勉強することにする
　前者是習慣
・禁煙することになっている vs 禁煙することにしている
　前者規則，後者自己決定

◆ 錯誤陷阱
・不能用於單次行為
・× 今日運動することにしている
・✓ 今日運動することにする
　（～ている是習慣）

◆ 變化補充
・主語多為自己
・帶自我約束感
・考試重點在「持續性」
```

## ようになっている

```
◆ 語感例句結構

　這扇門設計成會自動關

・ようになる／ことになっている
・ようになっている＝系統設計；ようになる＝自然變化
・使えるようになっている vs 使えるようになった
　前者設計，後者能力變化
・禁止することになっている vs 禁止するようになっている
　前者規則，後者機制

◆ 錯誤陷阱
・不能用於單純能力變化
・× 日本語が話せるようになっている
・✓ 日本語が話せるようになった
　（能力用ようになる）

◆ 變化補充
・多用於說明書
・描述系統/裝置
・考試重點在語感
```

## ようがない／ようもない

## について

```
◆ 語感例句結構
・主題內容
・N＋について
・この問題について説明する。
　說明這個問題
・日本文化について勉強している。
　正在學日本文化

◆ 混淆比較
・に関して／に対して
・について＝一般主題；に関して＝較正式
・計画について話す vs 計画に関して話す
　後者較書面
・彼に対して怒る vs 彼について話す
　前者對人動作，後者談論

◆ 錯誤陷阱
・不能用於動作對象
・× 彼について怒る
・✓ 彼に対して怒る
　（對象用に対して）

◆ 變化補充
・についての＋名詞
```
