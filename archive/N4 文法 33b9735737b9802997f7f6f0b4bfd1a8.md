# N4 文法

# 助詞・句尾

## までに

```
◆ 語感例句結構
・截止期限內完成
・N／V辞書形＋までに
・レポートは明日までに出してください。（報告請在明天前交）
・30歳までに結婚したい。（想在30歲前結婚）

◆ 混淆比較
・まで／うちに
・まで＝單純終點｜までに＝期限完成｜うちに＝趁變化前
・5時まで働く。（工作到5點）vs 5時までに終わらせる。（5點前完成）
・雨が降らないうちに帰る。（趁還沒下雨回去）vs 雨が降るまでに帰る（下雨前完成回去）

◆ 錯誤陷阱
・誤把までに當「持續到」
・× 5時までに働く
・✓ 5時まで働く（工作到5點）

◆ 變化補充
・動作性限制
・までに一定要「完成型動作」
・看到句子若是「做完」→選までに
```

## ばかり

```
◆ 語感例句結構（必出）
・剛做完
・動詞た形＋ばかり
・今帰ってきたばかりで、ちょっと疲れてる。
（我才剛回來，有點累）
・さっきご飯を食べたばかりなのに、もうお腹すいた。
（明明才剛吃完飯，又餓了）

◆ 混淆比較（必出｜選項核心）
・ところ／たばかり
・たばかり＝過了一點時間；ところ＝正好當下
・今来たばかり vs 今来たところ
（剛來沒多久 vs 正好剛來）
・さっき食べたばかり vs ちょうど食べるところ
（剛吃完 vs 正要吃）

◆ 錯誤陷阱（必出｜考試重點）
・不能用現在式
・× 食べるばかり
・✓ 食べたばかり（才剛吃完）

◆ 變化補充（可省略）
・時間感
・帶主觀「感覺很近」不一定真的剛剛
・看到「剛剛／才」優先選たばかり
```

```
◆ 語感例句結構（必出）
・老是在做
・動詞て形＋ばかりいる／名詞＋ばかり
・彼はスマホを見てばかりいる。
（他老是在看手機）
・最近甘いものばかり食べてる。
（最近幾乎都在吃甜食）

◆ 混淆比較（必出｜選項核心）
・だけ／しか
・ばかり＝偏向重複；だけ＝單純限定；しか＝否定限定
・肉ばかり食べる vs 肉だけ食べる
（幾乎都吃肉 vs 只吃肉）
・遊んでばかりいる vs 遊ぶしかない
（一直在玩 vs 只能玩）

◆ 錯誤陷阱（必出｜考試重點）
・不能接原形動詞
・× 食べるばかりいる
・✓ 食べてばかりいる（一直在吃）

◆ 變化補充（可省略）
・語氣
・常帶抱怨或負面評價
・看到「一直／老是」優先選てばかり
```

## でも

```
◆ 語感例句結構（必出）
・強逆接轉折
・普通形＋でも
・高いけど、でも買うよ。
（很貴，但我還是要買）
・疲れた。でもまだやる。
（很累，但還是要做）

◆ 混淆比較（必出｜選項核心）
・けど／しかし
・でも＝句首強轉折；けど＝句內弱轉折；しかし＝正式強轉折
・行きたい。でも時間がない
（想去，但沒時間）
・行きたいけど、時間がない
（想去但沒時間）

◆ 錯誤陷阱（必出｜考試重點）
・不能接在句中
・× 行きたいでも行けない
・✓ 行きたい。でも行けない（想去但去不了）

◆ 變化補充（可省略）
・口語
・常用在對話開頭強調情緒轉折
・考試看到「句子開頭轉折」優先選でも
```

## 疑問詞＋でも

```
◆ 語感例句結構（必出）
・語感：不管誰都
・結構：疑問詞＋でも
・誰でもできるよ。｜誰都可以做到喔
・何でも食べるタイプ。｜什麼都吃的類型

◆ 混淆比較（必出｜選項核心）
・對手文法：疑問詞＋か／疑問詞＋も
・一句話差異：でも＝全面肯定「全部都」，か＝不確定「某一個」，も＝強調數量多
・誰でもいい。vs 誰かいい？｜誰都可以 vs 有某個人可以嗎？
・何でも食べる。vs 何も食べない。｜什麼都吃 vs 什麼都不吃

◆ 錯誤陷阱（必出｜考試重點）
・NG：否定句不能用でも
・何でも食べない。（×）
・何も食べない。（✓ 什麼都不吃）

◆ 變化補充（可省略）
・分類：全稱肯定
・影響答案的理由：句子是肯定就選でも，是否定會變成も
・實戰建議：先判斷句子正負，再選でも／も
```

## 疑問詞＋か

```
◆ 語感例句結構（必出）
・語感：某個不確定
・結構：疑問詞＋か
・誰か来た？｜有誰來了嗎？
・何か食べる？｜要不要吃點什麼？

◆ 混淆比較（必出｜選項核心）
・對手文法：疑問詞＋でも／疑問詞＋も
・一句話差異：か＝不確定某一個，でも＝全部都可以，も＝全面否定或強調
・誰かいい人いる？vs 誰でもいい。｜有某個人嗎？vs 誰都可以
・何か食べる？vs 何も食べない。｜吃點什麼嗎？vs 什麼都不吃

◆ 錯誤陷阱（必出｜考試重點）
・NG：把か用在否定全無
・誰か来なかった。（×）
・誰も来なかった。（✓ 誰都沒來）

◆ 變化補充（可省略）
・分類：不特定存在
・影響答案的理由：句子在找「一個」就用か，不是全部或完全否定
・實戰建議：看到「有沒有某個」直接選か
```

## の（疑問）

```
◆ 語感例句結構（必出）
・語感：柔和確認感
・結構：普通形＋の？
・今日行くの？｜今天要去嗎？
・どうしたの？｜怎麼了？（帶關心）

◆ 混淆比較（必出｜選項核心）
・對手文法：か／のだ
・一句話差異：の？＝帶情緒確認，か＝中性疑問，のだ＝解釋說明
・行くの？vs 行きますか？｜要去嗎（柔）vs 要去嗎（正式）
・どうしたの？vs どうしたんだ？｜怎麼了（關心）vs 怎麼回事（說明）

◆ 錯誤陷阱（必出｜考試重點）
・NG：名詞直接接の？
・学生の？（×）
・学生なの？（✓ 是學生嗎？）

◆ 變化補充（可省略）
・分類：語氣差異
・影響答案的理由：題目在考「語氣」不是性別
・實戰建議：看到關心／輕柔語氣就選の？
```

## かい／だい

```
◆ 語感例句結構（必出）
・語感：男性輕問感
・結構：普通形＋かい／だい
・今日は行くかい？｜今天要去嗎？（輕問）
・どうしたんだい？｜怎麼了？（帶關心）

◆ 混淆比較（必出｜選項核心）
・對手文法：か／の？
・一句話差異：かい＝男性柔問，か＝中性疑問，の？＝柔和情緒
・行くかい？vs 行きますか？｜要去嗎（輕鬆）vs 要去嗎（正式）
・どうしたんだい？vs どうしたの？｜怎麼了（男性感）vs 怎麼了（柔）

◆ 錯誤陷阱（必出｜考試重點）
・NG：直接接名詞＋かい
・学生かい？（×）
・学生なんだい？（✓ 是學生嗎？）

◆ 變化補充（可省略）
・分類：口語男性
・影響答案的理由：帶輕鬆語氣但偏男性角色
・實戰建議：對話有男性角色語氣時優先考慮
```

# 指示・名詞化・口語縮約

## こんな／そんな／あんな

```
◆ 語感例句結構（必出）
・距離感評價指
・こんな／そんな／あんな＋N
・こんな店、初めて来た！
　（這樣的店，我第一次來）
・あんなこと、もう二度としない。
　（那樣的事，再也不會做）

◆ 混淆比較（必出｜選項核心）
・この／その／あの
・評價差：こんな＝帶情緒評價；この＝單純指示
・この店は静かだ。
　（這家店）vs こんな店、落ち着くね。
　（這種店真讓人放鬆）
・その人は誰？
　（那個人是誰）vs そんな人、信用できない。
　（那種人不可信）

◆ 錯誤陷阱（必出｜考試重點）
・誤用單純指物：不可只當「這個／那個」
・× こんな本をください。（想說這本）
・✓ この本をください。（這本書）

◆ 變化補充（可省略）
・距離差
・こんな＝近／そんな＝對方或已知／あんな＝遠
・看到「情緒評價」優先選
```

## そう／こう／ああ

```
◆ 語感例句結構（必出）
・方式狀態指示
・そう／こう／ああ＋V／言う／なる
・そう言えば、彼に会ったよ。
　（說到那個，我見過他）
・こうすれば、うまくいくと思う。
　（這樣做應該會順利）

◆ 混淆比較（必出｜選項核心）
・このように／そのように／あのように
・口語差：そう等＝口語直接；そのように＝較書面說明
・こうやって作ると簡単だよ。
　（這樣做很簡單）vs このように作ると簡単です。
　（書面說明）
・ああ言われたら、断れない。
　（被那樣說就難拒絕）vs あのように言われたら困る。
　（書面表達）

◆ 錯誤陷阱（必出｜考試重點）
・誤當名詞修飾：不可直接修飾名詞
・× そう人は嫌い。
・✓ そういう人は嫌い。（那種人不喜歡）

◆ 變化補充（可省略）
・距離差
・こう＝自己側／そう＝對方或已知／ああ＝遠或共同經驗
・看到「動作方式」優先選
```

## さ

```
◆ 語感例句結構（必出）
・形容詞名詞化
・Aい／Aな＋さ
・この部屋の広さに驚いた。
　（對這個房間的寬敞感到驚訝）
・彼の静かさが逆に怖い。
　（他的安靜反而讓人覺得可怕）

◆ 混淆比較（必出｜選項核心）
・み／こと
・客觀量感：さ＝可衡量程度；み＝主觀感受；こと＝抽象事情
・重さを量る。
　（測重量）vs 重みを感じる。
　（感受到沉重感）
・難しさを説明する。
　（說明難度）vs 難しいことを説明する。
　（說明困難的事）

◆ 錯誤陷阱（必出｜考試重點）
・接續錯：名詞不可直接接さ
・× 便利ささが重要だ。
・✓ 便利さが重要だ。（便利性很重要）

◆ 變化補充（可省略）
・語感差異
・さ＝偏客觀可比較
・看到「程度/數值感」優先選
```

## ～の（は／が／を）

```
◆ 語感例句結構（必出）
・動作名詞化焦點
・Vる＋の（は／が／を）
・日本語を勉強するのは楽しい。
　（學日文這件事很有趣）
・彼が来るのを待っている。
　（在等他來）

◆ 混淆比較（必出｜選項核心）
・こと／のが／のを
・具體感：の＝具體動作；こと＝抽象一般
・走ることは大切だ。
　（跑步這件事）vs 走るのは大変だ。
　（實際跑起來很累）
・彼が来ることを信じる。
　（相信這件事）vs 彼が来るのを待つ。
　（等他來這個動作）

◆ 錯誤陷阱（必出｜考試重點）
・助詞混用：主題／對象不可混
・× 彼が来るのは待っている。
・✓ 彼が来るのを待っている。（等待的對象）

◆ 變化補充（可省略）
・助詞差異
・のは＝主題／のが＝主語強調／のを＝受詞
・看到「動作當名詞」優先選
```

## こと

```
◆ 語感例句結構（必出）
・抽象事實概念
・Vる／Vた＋こと
・日本語を勉強することは大切だ。
　（學日文這件事很重要）
・彼が来たことを知っている。
　（知道他來過這件事）

◆ 混淆比較（必出｜選項核心）
・の／もの
・抽象差：こと＝一般事實；の＝具體動作；もの＝理由辯解
・走るのは大変だ。
　（實際跑起來累）vs 走ることは大切だ。
　（跑步這件事很重要）
・遅れたものだから、すみません。
　（辯解）vs 遅れたことを謝る。
　（對「遲到這件事」道歉）

◆ 錯誤陷阱（必出｜考試重點）
・誤用具體動作：臨場感場景不適用こと
・× 目の前で転ぶことを見た。
・✓ 目の前で転ぶのを見た。（看到他當場跌倒）

◆ 變化補充（可省略）
・語感差異
・こと＝偏抽象、規則、事實
・看到「一般論／規範」優先選
```

## ちゃ／ちゃう

```
◆ 語感例句結構（必出）
・口語完成遺憾
・Vて＋ちゃう／じゃう（＝てしまう）
・もう食べちゃった。
　（已經吃掉了）
・財布を忘れちゃった。
　（不小心忘記帶錢包了）

◆ 混淆比較（必出｜選項核心）
・てしまう／ている
・語氣差：ちゃう＝口語完成＋遺憾；てしまう＝正式；ている＝進行/狀態
・宿題をやってしまった。
　（正式完成）vs 宿題やっちゃった。
　（口語＋有點懊悔）
・今食べている。
　（正在吃）vs もう食べちゃった。
　（已經吃完）

◆ 錯誤陷阱（必出｜考試重點）
・誤用進行：不可表「正在」
・× 今ご飯食べちゃう。
・✓ 今ご飯食べている。（現在正在吃）

◆ 變化補充（可省略）
・音變
・て→ちゃ／で→じゃ（飲んじゃう）
・看到「口語＋已經／不小心」優先選
```

### 許可・禁止・義務・命令

## てもいい／てもかまわない

```
◆ 語感例句結構（必出）
・語感：允許放行
・結構：動詞て形＋もいい／てもかまわない
・ここで写真撮ってもいい？（可以在這裡拍照嗎？）
・遅れてもかまわないよ、ゆっくり来て。（晚到也沒關係，慢慢來）

◆ 混淆比較（必出｜選項核心）
・對手文法：てもいい vs なければならない / てはいけない
・一句話差異：てもいい＝可做／なければならない＝必須／てはいけない＝禁止
・入ってもいい。（允許）vs 入らなければならない。（必須進入）
　可以進去 vs 必須進去
・ここで食べてもいい。（允許）vs ここで食べてはいけない。（禁止）
　可以在這吃 vs 不可以在這吃

◆ 錯誤陷阱（必出｜考試重點）
・NG用法說明：把請求誤用成允許
・ドアを開けてもいい？（×想請對方開門）
・ドアを開けてくれる？（✓ 請你幫我開門）

◆ 變化補充（可省略）
・分類：語氣差
・影響答案的理由：かまわない較冷靜、不強調情感
・實戰建議：口語多用てもいい；正式或淡化情緒用てもかまわない
```

## なくてもいい／なくてもかまわない

```
◆ 語感例句結構（必出）
・語感：不做也可
・結構：動詞ない形＋なくてもいい／なくてもかまわない
・今日は来なくてもいいよ。（今天不來也可以）
・全部覚えなくてもかまわない。（不用全部記也沒關係）

◆ 混淆比較（必出｜選項核心）
・對手文法：なくてもいい vs なければならない / てはいけない
・一句話差異：なくてもいい＝不做OK／なければならない＝一定要做／てはいけない＝不能做
・行かなくてもいい。（不去也行）vs 行かなければならない。（一定要去）
　不去也可以 vs 一定要去
・食べなくてもいい。（不吃也可）vs 食べてはいけない。（不能吃）
　不吃也可以 vs 不可以吃

◆ 錯誤陷阱（必出｜考試重點）
・NG用法說明：把「禁止」誤用成「不需要」
・ここでタバコを吸わなくてもいい（×想表達不能抽）
・ここでタバコを吸ってはいけない（✓ 這裡不能抽菸）

◆ 變化補充（可省略）
・分類：語氣差
・影響答案的理由：かまわない較客觀冷靜
・實戰建議：口語用なくてもいい；正式或淡化語氣用なくてもかまわない
```

## てはいけない

```
◆ 語感例句結構（必出）
・語感：禁止行為
・結構：動詞て形＋はいけない
・ここでタバコを吸ってはいけない。（這裡不可以抽菸）
・授業中にスマホを使ってはいけないよ。（上課不能用手機）

◆ 混淆比較（必出｜選項核心）
・對手文法：てはいけない vs なくてもいい / なければならない
・一句話差異：てはいけない＝禁止／なくてもいい＝不做OK／なければならない＝必須
・入ってはいけない。（禁止進入）vs 入らなくてもいい。（不進去也可）
　不能進去 vs 不進去也可以
・提出してはいけない。（不能提交）vs 提出しなければならない。（必須提交）
　不能交 vs 一定要交

◆ 錯誤陷阱（必出｜考試重點）
・NG用法說明：把「義務」誤用成「禁止」
・早く寝てはいけない（×想表達一定要早睡）
・早く寝なければならない（✓ 一定要早睡）

◆ 變化補充（可省略）
・分類：口語縮約
・影響答案的理由：ちゃいけない常見於口語
・實戰建議：考試看到禁止語氣優先選てはいけない
```

## な（禁止）

```
◆ 語感例句結構（必出）
・語感：強硬禁止
・結構：動詞辭書形＋な
・ここに入るな！（不准進來！）
```

## だろう

```
◆ 語感例句結構

・結構：普通形＋だろう

◆ 錯誤陷阱
・NG用法說明：な形／名詞前要接だ
・× 静かだろう。（其實這句是正確，但很多人誤以為不能加だ）

◆ 變化補充
・分類：推測（普通形）
・影響答案的理由：句子非禮貌、內心判斷→用だろう
・實戰建議：日常心裡OS幾乎都用だろう
```

## （だろう）と思う／と思う

```
◆ 語感例句結構
・語感：推測 vs 意見
・結構：（普通形／だろう）＋と思う
・彼は来ると思う。／我覺得他會來。
・彼は来るだろうと思う。／我認為他應該會來。

◆ 混淆比較
・對手文法：～と思う vs ～だろうと思う
・一句話差異：と思う＝單純意見；だろうと思う＝推測＋保留
・行くと思う。vs 行くだろうと思う。／前者直接覺得，後者帶不確定
・大丈夫だと思う。vs 大丈夫だろうと思う。／後者更保守

◆ 錯誤陷阱

◆ 變化補充
```

## かもしれない


# 可能・能力・難易・程度・引用

## ことができる

```
◆ 語感例句結構
・語感：可以做到
・結構：動詞辞書形＋ことができる
・日本語を話すことができる。／可以說日文。
・このアプリで予約することができます。／可以用這個App預約。

◆ 混淆比較
・對手文法：可能形（話せる）／～てもいい
・一句話差異：ことができる＝能力/可能；可能形＝口語能力；てもいい＝允許
・話すことができる。vs 話せる。／意思一樣，後者更口語
・入ってもいい。vs 入ることができる。／前者允許，後者能力/條件可行

◆ 錯誤陷阱
・NG用法說明：不能再加を（會重複）
・× 日本語を話すことをできる。
・✓ 日本語を話すことができる。／可以說日文。

◆ 變化補充
・分類：能力／可能
・影響答案的理由：看到「可以做某事」→用ことができる
・實戰建議：正式或書面用這個，口語可用可能形
```

## （ら）れる（可能）

```
◆ 語感例句結構
・語感：能不能做
・結構：動詞可能形（五段：u→e＋る／一段：られる／する→できる／来る→来られる）
・日本語が話せる。／會說日文。
・寿司が食べられる。／可以吃壽司。

◆ 混淆比較
・對手文法：ことができる／受身（られる）
・一句話差異：可能形＝口語能力；ことができる＝正式；られる也可能是被動
・話せる。vs 話すことができる。／意思同，後者較正式
・食べられる。vs 褒められる。／前者可能，後者被動（被稱讚）

◆ 錯誤陷阱
・NG用法說明：可能形多用が，不用を
・× 日本語を話せる。
・✓ 日本語が話せる。／會說日文。

◆ 變化補充
・分類：動詞三類變化
・影響答案的理由：不同類型變形不同（五段→e＋る／一段→られる）
・實戰建議：先判斷動詞類型再變形
```

## 数量詞＋も

```
◆ 語感例句結構
・語感：多到誇張
・結構：数量詞＋も

◆ 混淆比較

・分類：強調數量
・影響答案的理由：看到「竟然這麼多」→用数量詞＋も
```

## ようにする

```
◆ 語感例句結構
・語感：刻意去做
・結構：動詞辞書形／ない形＋ようにする
・毎日運動するようにしている。／有在刻意每天運動。

◆ 混淆比較
```

## ていく／てくる

```
◆ 語感例句結構
・語感：變化方向感
・結構：動詞て形＋いく／くる
・これから暑くなっていく。／接下來會越來越熱。
・だんだん寒くなってきた。／已經慢慢變冷了。

◆ 混淆比較
・對手文法：～になる／～ている
・一句話差異：ていく＝往未來變；てくる＝從過去到現在變；なる＝單次變；ている＝狀態

・暑くなる。vs 暑くなってきた。／前者單純變，後者帶過程感

◆ 錯誤陷阱
・NG用法說明：不能只看字面「去/來」，要看時間方向
・× これから寒くなってきた。（未來卻用きた）
・✓ これから寒くなっていく。／接下來會變冷。

◆ 變化補充

・影響答案的理由：未來變化→ていく；已經開始變→てくる
```


## ～と～と、どちら

```

・× コーヒーとお茶とジュースと、どちら？

◆ 變化補充

```

# 理由・目的・並列

## し

```
◆ 語感例句結構
・語感：還有理由
・結構：普通形＋し、～
・この店は安いしおいしい。／這家店又便宜又好吃。
・今日は忙しいし、行けない。／今天很忙，而且不能去。

◆ 混淆比較
・對手文法：～て／～から
・一句話差異：し＝列舉多理由；て＝單純連接；から＝單一明確原因
・安くておいしい。vs 安いしおいしい。／前者並列，後者有「還有」
・疲れたから帰る。vs 疲れたし帰る。／前者因為，後者「又…所以」

◆ 錯誤陷阱
・NG用法說明：し通常不只一個理由（用一個會怪）
・× 疲れたし帰る。（只講一個理由）
・✓ 疲れたし時間もないし帰る。／又累又沒時間所以回去。

◆ 變化補充
・分類：理由列舉
・影響答案的理由：句子有「而且／還有」→用し
・實戰建議：後面常還會再補一個理由（就算省略也有那種感覺）
```

## ため（に）

```
◆ 語感例句結構
・語感：為了目的
・結構：動詞辞書形／名詞＋の＋ため（に）
・日本語を勉強するために日本へ行く。／為了學日文去日本。
・健康のために運動している。／為了健康在運動。

◆ 混淆比較
・對手文法：～ように／～から
・一句話差異：ために＝明確目的；ように＝達成狀態；から＝原因

◆ 錯誤陷阱
・NG用法說明：主語不同時容易不自然
・× 私は彼を助けるために彼は来た。
・✓ 彼は私を助けるために来た。／他為了幫我而來。

◆ 變化補充
・分類：目的
・影響答案的理由：句子有「為了～」→用ために
・實戰建議：先判斷是「原因」還是「目的」
```

```
・結構：名詞／動詞普通形＋とか～とか
・週末は映画を見るとか、ゲームするとかしてる。／週末會看看電影、玩玩遊戲之類的。
・コーヒーとかお茶とか飲む。／會喝咖啡、茶之類的。

◆ 混淆比較
・對手文法：～たり～たり／～や～など
・一句話差異：とか＝隨意舉例；たり＝動作列舉；や＝名詞列舉（較正式）
・食べたり寝たりする。vs 食べるとか寝るとかする。／前者較標準，後者較隨便口語
・本や雑誌など読む。vs 本とか雑誌とか読む。／前者正式，後者口語

◆ 錯誤陷阱
・NG用法說明：とか通常要成對出現
・× コーヒーとか飲む。（單個使用較不完整）
・✓ コーヒーとかお茶とか飲む。／喝咖啡或茶之類的。

◆ 變化補充
・分類：舉例（口語）
・影響答案的理由：句子有「之類的」→用とか
・實戰建議：口語想輕鬆列舉就用とか
```

# 條件・順接・逆接

## と

```
◆ 語感例句結構
・語感：一起或引用
・結構：名詞＋と／句子＋と
・友達と行く。／和朋友一起去。
・「行く」と言った。／說了「要去」。

◆ 混淆比較
・對手文法：～や／～か
・一句話差異：と＝完整列舉/一起/引用；や＝部分列舉；か＝選擇
・りんごとみかん。vs りんごやみかん。／前者全部列出，後者舉例
・コーヒーかお茶。vs コーヒーとお茶。／前者選一，後者兩個都有

◆ 錯誤陷阱
・NG用法說明：と不能用在「選擇」
・× コーヒーとお茶、どちら？
・✓ コーヒーかお茶、どちら？／咖啡還是茶？

◆ 變化補充
・分類：並列／共同／引用
・影響答案的理由：看到「一起／全部列出／引用」→用と
・實戰建議：先判斷是「一起還是選一」
```

## ば

```
◆ 語感例句結構
・語感：如果就會
・結構：動詞ば形（え段＋ば）
・早く行けば間に合う。／如果早點去就來得及。
・安ければ買う。／如果便宜就買。

◆ 混淆比較
・對手文法：～たら／～なら
・一句話差異：ば＝一般條件（結果自然發生）；たら＝時間/條件；なら＝前提話題
・雨が降れば行かない。vs 雨が降ったら行かない。／前者偏條件，後者也可時間
・行くなら連絡して。vs 行けば連絡して（×）／なら是前提，ば不自然

◆ 錯誤陷阱
・NG用法說明：ば句後面不能接「意志/命令」
・× 安ければ買おう。
・✓ 安かったら買おう。／如果便宜就買吧。

◆ 變化補充
・分類：條件（一般）
・影響答案的理由：自然結果→ば；人為決定→たら
・實戰建議：看到「就會那樣」優先用ば
```

## たら

```
◆ 語感例句結構
・語感：如果就做
・結構：動詞た形＋ら
・雨が降ったら行かない。／如果下雨就不去。
・家に帰ったら電話して。／回家之後打電話。

◆ 混淆比較
・對手文法：～ば／～なら
・一句話差異：たら＝條件＋可接意志/命令；ば＝自然結果；なら＝前提話題
・安かったら買おう。vs 安ければ買おう（×）／たら可接意志
・行くなら教えて。vs 行ったら教えて。／なら是前提，たら偏發生後

◆ 錯誤陷阱
・NG用法說明：たら有「先後順序」語感（發生後才做）
・× 起きたら寝る。（時間矛盾）
・✓ 起きたら顔を洗う。／起床後洗臉。

◆ 變化補充
・分類：條件／時間
・影響答案的理由：句子有「如果…就做（人為）」→用たら
・實戰建議：想接「打算/命令」幾乎選たら
```

## なら

```
◆ 語感例句結構
・語感：既然那就
・結構：普通形／名詞＋なら
・行くなら早く行こう。／既然要去就早點去。
・日本なら京都がいい。／如果是日本的話，京都比較好。

◆ 混淆比較
・對手文法：～たら／～ば
・一句話差異：なら＝承接前提（對方/情境）；たら＝發生後；ば＝自然條件
・行くなら連絡して。vs 行ったら連絡して。／前者前提，後者做了之後
・安ければ買う。vs 安いなら買う。／前者一般條件，後者基於某資訊

◆ 錯誤陷阱
・NG用法說明：なら通常有「前提已存在」
・× 明日なら雨が降る。（單純預測不自然）
・✓ 明日なら行ける。／如果是明天的話可以。

◆ 變化補充
・分類：條件（前提）
・影響答案的理由：句子在回應對方/已知條件→用なら
・實戰建議：看到對話感或「如果是這樣的話」就選なら
```

```jsx
### ■ 同一動作對比（超重要）

* 日本に行ったら写真を撮る
  👉 去了之後會拍照（發生→做）
```

## たところ

```
◆ 語感例句結構
・語感：剛做完
・結構：動詞た形＋ところ
・今帰ったところだ。／剛回來。
・ちょうど食べたところです。／剛好剛吃完。

◆ 混淆比較
・對手文法：～た／～ているところ
・一句話差異：たところ＝剛完成；た＝單純過去；ているところ＝正在
・帰ったところだ。vs 帰った。／前者剛回來，後者只是回來
・食べているところだ。vs 食べたところだ。／前者正在吃，後者剛吃完

◆ 錯誤陷阱
・NG用法說明：不能用來表示很久以前
・× 昨日帰ったところだ。
・✓ 今帰ったところだ。／剛回來。

◆ 變化補充
・分類：時間（剛完成）
・影響答案的理由：看到「剛剛」→用たところ
・實戰建議：搭配「今／ちょうど」最自然
```

## ても／でも

```
◆ 語感例句結構
・語感：就算也
・結構：動詞て形＋も／名詞＋でも
・雨が降っても行く。／就算下雨也要去。
・子どもでもできる。／連小孩也能做到。

◆ 混淆比較
・對手文法：～てもいい／～でも（舉例）
・一句話差異：ても＝讓步（即使）；でも＝名詞讓步/舉例；てもいい＝允許
・高くても買う。vs 高くてもいい。／前者即使貴也買，後者貴也可以
・コーヒーでも飲む。vs コーヒーを飲む。／前者隨便舉例，後者單純喝

◆ 錯誤陷阱
・NG用法說明：動詞不能接でも
・× 行くでもいい。
・✓ 行ってもいい。／去也可以。

◆ 變化補充
・分類：讓步／舉例
・影響答案的理由：看到「即使～也」→用ても／でも
・實戰建議：動詞用ても，名詞用でも
```

## けれど（も）／けど

```
◆ 語感例句結構
・語感：轉折但
・結構：普通形＋けれど（も）／けど
・行きたいけど、時間がない。／想去但沒時間。
・おいしいけれど、ちょっと高い。／好吃但有點貴。

◆ 混淆比較
・對手文法：～のに／～が
・一句話差異：けど＝一般轉折；のに＝不滿/反預期；が＝較正式轉折
・行きたいけど行けない。vs 行きたいのに行けない。／後者更有不甘
・いいけど…vs いいですが…／後者較正式

◆ 錯誤陷阱
・NG用法說明：名詞／な形前要加だ
・× 学生けど忙しい。
・✓ 学生だけど忙しい。／是學生但很忙。

◆ 變化補充
・分類：逆接（一般）
・影響答案的理由：普通轉折幾乎用けど
・實戰建議：口語用けど，正式用けれど（も）／が
```

## のに

```
◆ 語感例句結構
・語感：明明卻
・結構：普通形＋のに（名詞／な形：＋なのに）
・雨なのに出かけた。／明明下雨卻出門了。
・頑張ったのに失敗した。／明明很努力卻失敗了。

◆ 混淆比較
・對手文法：～けど／～から
・一句話差異：のに＝強烈逆接＋不滿；けど＝一般轉折；から＝原因
・行きたいのに行けない。vs 行きたいけど行けない。／前者更有不甘
・疲れたから寝る。vs 疲れたのに寝ない。／原因 vs 反預期

◆ 錯誤陷阱
・NG用法說明：名詞／な形前要加なのに
・× 学生のに忙しい。
・✓ 学生なのに忙しい。／明明是學生卻很忙。

◆ 變化補充
・分類：逆接（情緒）
・影響答案的理由：看到「明明～卻」→用のに
・實戰建議：帶抱怨/不滿語氣幾乎就是のに
```

# 授受表現

## あげる／てあげる／さしあげる／てさしあげる／やる／てやる

```
◆ 語感例句結構
・語感：向外給予
・結構：Nに Nをあげる／Vてあげる／（敬）さしあげる／Vてさしあげる／（對下位）やる／Vてやる
・妹にケーキをあげた。／我送了蛋糕給妹妹。
・道に迷っていた人に、駅まで案内してあげた。／我帶迷路的人走到車站。

◆ 混淆比較
・對手文法：くれる／てもらう／やる
・一句話差異：あげる系＝我方把東西／行為往外給；くれる＝別人給我方；もらう＝我方接受；やる只對動物、植物或明顯下位
・私は友だちに本をあげた。vs 友だちが私に本をくれた。／前者我給朋友，後者朋友給我。
・犬にえさをやった。vs 先生に本をさしあげた。／やる給狗可以，對老師要用さしあげる。

◆ 錯誤陷阱
・NG用法說明：不能把やる用在長輩／客戶；也不能把てあげる亂用在對方沒受惠感、會顯得施恩
・× 部長に資料をやりました。／彼に説明してあげる必要はないのに、してあげた。
・✓ 部長に資料をさしあげました。／彼に頼まれたので、説明してあげた。＋前者要敬語；後者要有「對方受惠」才自然

◆ 變化補充
・分類：物品授受 vs 行為授受
・影響答案的理由：Vてあげる／Vてさしあげる不是單純做動作，而是「為對方做」；考題常用受益方向和上下關係切選項
・實戰建議：先看「誰給誰」再看「地位高低」；對長輩用さしあげる，對動物或晚輩才考慮やる
```

## もらう／てもらう／いただく／ていただく

```
◆ 語感例句結構
・語感：向內接受
・結構：Nに Nをもらう／Vてもらう／（敬）いただく／Vていただく
・友だちにプレゼントをもらった。／我從朋友那裡收到禮物。
・先輩にレポートをチェックしてもらった。／我請前輩幫我檢查報告。

◆ 混淆比較
・對手文法：くれる／あげる／いただく
・一句話差異：もらう系＝我方「接受」；くれる＝對方主動給我；あげる＝我方給出去；いただく＝もらう的自謙（對上）
・先生に本をいただいた。vs 先生が本をくれた。／前者強調我謙虛接受，後者只是對方給我。
・私は彼に手伝ってもらった。vs 私は彼に手伝ってあげた。／前者我受幫助，後者我幫他。

◆ 錯誤陷阱
・NG用法說明：ていただく不能用在自己做的動作；也不能用在平輩或對下顯得過度敬語
・× 私はレポートを書いていただいた。／弟に宿題を見ていただいた。
・✓ 先生にレポートを書いていただいた。／弟に宿題を見てもらった。＋動作提供者必須是對方，敬語只對上使用

◆ 變化補充
・分類：物品授受 vs 行為授受
・影響答案的理由：Vてもらう／Vていただく強調「請別人幫我做」；考題常考主語是誰＋是否需要敬語
・實戰建議：先抓「誰做動作」→再判斷「我是不是受益者」→最後看是否需要用いただく
```

## くれる／てくれる／くださる／てくださる

```
◆ 語感例句結構
・語感：他人給我
・結構：Nが（私に）Nをくれる／Vてくれる／（敬）くださる／Vてくださる
・友だちが（私に）チョコをくれた。／朋友給了我巧克力。
・店員さんが丁寧に説明してくれた。／店員很親切地幫我說明。

◆ 混淆比較
・對手文法：もらう／あげる／くださる
・一句話差異：くれる系＝他人給我（或我方陣營）；もらう＝我接受（主語是我）；あげる＝我給出去；くださる＝くれる的尊敬（對方高位）
・友だちが本をくれた。vs 私は友だちに本をもらった。／同一事件，主語不同。
・先生がアドバイスをくださった。vs 私は先生にアドバイスをいただいた。／前者尊敬對方給我，後者我謙虛接受。

◆ 錯誤陷阱
・NG用法說明：くれる不能用在我給別人；くださる不可自用或對下位
・× 私は友だちにプレゼントをくれた。／私は資料をくださった。
・✓ 私は友だちにプレゼントをあげた。／先生が資料をくださった。＋主語要是「給的人」，且敬語只用於對方

◆ 變化補充
・分類：物品授受 vs 行為授受
・影響答案的理由：Vてくれる／Vてくださる＝對方「為我做」帶受惠感；考題常用主語＋尊敬判斷
・實戰建議：看到「對方幫我」先選くれる系；對上再升級成くださる
```

# 被動・使役・敬語

## （ら）れる（被動／尊敬）

```
◆ 語感例句結構
・語感：被動／尊敬
・結構：受身：Nが／に V（ら）れる｜尊敬：お／ご〜（ら）れる／V（ら）れる
・犬に手をかまれた。／被狗咬了手。
・先生はもう帰られました。／老師已經回去了。

◆ 混淆比較
・對手文法：〜てある／〜ている／お〜になる
・一句話差異：受身＝我被影響；てある＝結果狀態；尊敬（られる）＝抬高主語；お〜になる＝更明確敬語
・窓が割られた。vs 窓が割ってある。／前者被打破（被動），後者已被人弄好呈現狀態。
・先生は来られた。vs 先生がお越しになった。／前者普通尊敬，後者更正式敬語。

◆ 錯誤陷阱
・NG用法說明：尊敬用（ら）れる不能用在自己；受身不可亂加に造成錯誤主語
・× 私は帰られました。／私は先生に褒めた。
・✓ 先生は帰られました。／私は先生に褒められた。＋尊敬只對他人；受身需用られる

◆ 變化補充
・分類：受身 vs 尊敬同形
・影響答案的理由：考題常靠「主語地位」判斷是受身還是尊敬
・實戰建議：先看主語是不是上位者→是就優先判斷尊敬；否則多半是被動
```

## （さ）せる（使役）

```
◆ 語感例句結構
・語感：強制／放任
・結構：Nに V（さ）せる
・母は子どもに野菜を食べさせた。／媽媽讓孩子吃蔬菜（偏強制）。
・先生は学生に自由に話させた。／老師讓學生自由發言（偏放任）。

◆ 混淆比較
・對手文法：〜てもらう／〜てくれる／〜ように言う
・一句話差異：使役＝我控制對方做；てもらう＝我請對方幫我；くれる＝對方幫我；ように言う＝只是指示不保證執行
・私は彼に掃除させた。vs 私は彼に掃除してもらった。／前者我讓他做，後者我請他幫我做。
・先生は学生に早く来るように言った。vs 先生は学生に早く来させた。／前者只是叫，後者實際讓他來。

◆ 錯誤陷阱
・NG用法說明：助詞誤用（に／を）；自動詞用を表「讓其移動」時不可亂用に
・× 母は子どもを泣かせた（想表命令吃東西卻用錯動詞）／先生は学生を座らせた（想表坐下對象卻誤用を）
・✓ 母は子どもに野菜を食べさせた。／先生は学生に座らせた。＋動作執行者用に；移動型自動詞多用を（如子どもを歩かせた）

◆ 變化補充
・分類：強制（命令）vs 放任（允許）
・影響答案的理由：語境決定是「逼做」還是「讓做」，但形式相同；考題常用語氣或上下文判斷
・實戰建議：先判斷你有沒有控制權→有就用使役；再用語境分強制或放任
```

## （さ）せられる（使役被動）

```
◆ 語感例句結構
・語感：被迫做事
・結構：Nに V（さ）せられる
・上司に残業させられた。／被上司逼加班。
・母に野菜を食べさせられた。／被媽媽逼吃蔬菜。

◆ 混淆比較
・對手文法：受身（られる）／使役（させる）／〜てもらう
・一句話差異：使役被動＝我被迫做；受身＝單純被影響；使役＝我讓別人做；てもらう＝我請別人幫我
・私は先生に褒められた。vs 私は先生に発表させられた。／前者被稱讚，後者被逼發表。
・私は弟に掃除させた。vs 私は弟に掃除させられた。／前者我讓弟做，後者我被迫做。

◆ 錯誤陷阱
・NG用法說明：助詞錯（に／を）或把受身誤當使役被動
・× 上司を残業させられた。／私は発表された。
・✓ 上司に残業させられた。／私は発表させられた。＋動作施加者用に；要有「被迫做」才用使役被動

◆ 變化補充
・分類：不情願 vs 中性
・影響答案的理由：多半帶負面「被逼」語感，考題會用語境判斷是否需這層情緒
・實戰建議：看到「不得不」或不爽語氣→優先選使役被動
```

## 名詞＋でございます

```
◆ 語感例句結構
・語感：最高禮貌斷定
・結構：Nでございます
・こちらは本日のメニューでございます。／這是今天的菜單。
```

## （さ）せてください

```
◆ 語感例句結構
・語感：請求允許
・結構：V（さ）せてください
・今日は早く帰らせてください。／今天請讓我早點回去。
・ちょっと確認させてください。／請讓我稍微確認一下。

◆ 混淆比較
・對手文法：〜てもいいですか／〜ていただけますか／（さ）せていただく
・一句話差異：させてください＝請允許我做；てもいいですか＝詢問可不可以；ていただけますか＝請你幫我做；させていただく＝更低姿態（已獲允許）
・ここで写真を撮らせてください。vs ここで写真を撮ってもいいですか。／前者偏請求允許，後者單純詢問。
・説明させてください。vs 説明させていただきます。／前者請允許，後者帶已被允許＋更謙虛。

◆ 錯誤陷阱
・NG用法說明：對方沒有「允許權」時亂用；或已確定可做卻還用ください不自然
・× 友だちに「ちょっと帰らせてください」（對方無決定權）／会議で「発表させてください」（其實已安排）
・✓ 上司に「早退させてください」／会議で「発表させていただきます」＋有權者才用；已確定用いただく

◆ 變化補充
・分類：許可請求 vs 已獲許可
・影響答案的理由：是否「已被允許」會決定選ください還是いただく
・實戰建議：先判斷對方有沒有決定權→有才用させてください
```
