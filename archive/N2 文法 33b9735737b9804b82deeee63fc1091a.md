# N2 文法

---

- **◎：一定要精熟**
- **○：至少看到就知道意思、語感、使用場景**
- **△：只做辨識，不深背**
- **✕：真的放掉**

# 關係・無關・前提

## ○ にかかわって／にかかわり／にかかわる

```
◆ 語感例句結構（必出）
・語感：重大關聯影響
・結構：名詞＋にかかわって／にかかわり／にかかわる＋名詞
・命にかかわる事故が起きた。（發生了攸關性命的事故）
・この問題は会社の信用にかかわっている。（這問題關係到公司信用）

◆ 混淆比較（必出｜選項核心）
・對手文法：に関して／について
・一句話差異：「にかかわる」＝影響重大結果，「に関して」＝單純關於
・命にかかわるミスだ。（會影響生死的錯誤）
　vs 命に関しての説明。（只是關於生命的說明）
・会社の存続にかかわる問題だ。（關係到存亡）
　vs 会社について話す。（只是談論公司）

◆ 錯誤陷阱（必出｜考試重點）
・NG：誤用於「普通話題」
・× 映画にかかわる話をした。
・✓ 映画について話をした。（一般話題用について）

◆ 變化補充（可省略）
・分類：動詞變形
・影響答案的理由：句型位置不同（修飾名詞／句尾）
・實戰建議：看到「命／存亡／重大影響」優先選にかかわる
```

## ◎ にかかわらず

```
◆ 語感例句結構（必出）
・語感：條件無視通用
・結構：名詞／疑問詞＋にかかわらず
・年齢にかかわらず、参加できます。（不論年齡都能參加）
・雨かどうかにかかわらず、試合は行われる。（不管下不下雨都會舉行）

◆ 混淆比較（必出｜選項核心）
・對手文法：ても／に関係なく
・一句話差異：「にかかわらず」＝制度性無差別，「ても」＝讓步條件
・お金があるかどうかにかかわらず、応募できる。（制度上不看條件）
　vs お金がなくても、応募できる。（即使沒錢也可以）
・経験にかかわらず採用する。（不論經驗一律）
　vs 経験がなくても採用する。（讓步語氣）

◆ 錯誤陷阱（必出｜考試重點）
・NG：不能接「單一已確定事實」
・× 雨にかかわらず、今日は中止だ。
・✓ 雨でも、今日は中止だ。（已確定事實用ても）

◆ 變化補充（可省略）
・分類：無視條件
・影響答案的理由：是否表「一律適用／制度規則」
・實戰建議：看到「誰でも／すべて／一律」優先選にかかわらず
```

## ◎ にもかかわらず

```
◆ 語感例句結構
・預期反結果
・普通形＋にもかかわらず
・雨が降っているにもかかわらず、彼は出かけた。（明明下雨卻出門）
・彼は病気にもかかわらず、普通に働いている。（生病卻照常工作）

◆ 混淆比較
・のに／くせに
・のに＝客觀反差／にもかかわらず＝更強硬正式／くせに＝帶批評
・お金がないのに、旅行に行った。（沒錢卻去旅行）
　お金がないにもかかわらず、旅行に行った。（正式強調反差）
・知っているくせに、教えてくれない。（明明知道卻不說）
　知っているにもかかわらず、教えてくれない。（較客觀）

◆ 錯誤陷阱
・不能直接接名詞（需＋である）
・× 学生にもかかわらず、働いている
・✓ 学生であるにもかかわらず、働いている（是學生卻在工作）

◆ 變化補充
・語氣強度
・比「のに」更正式、書面常見
・考試看到強對比＋正式句，優先選這個
```

## ○ もかまわず

```
◆ 語感例句結構
・無視照做
・名詞＋も構わず
・雨も構わず、外で遊んでいる。（不顧下雨在外面玩）
・人の迷惑も構わず、大声で話している。（不顧別人困擾大聲講話）

◆ 混淆比較
・にもかかわらず／おかまいなしに
・にもかかわらず＝事實反差／も構わず＝主動無視／おかまいなしに＝更強烈無視
・雨にもかかわらず、試合は続いた。（客觀反差繼續）
　雨も構わず、試合を続けた。（無視雨硬要做）
・周りの目も構わず、泣き出した。（不顧旁人）
　周りの目もおかまいなしに、泣き出した。（更強烈不顧）

◆ 錯誤陷阱
・不能接句子（需名詞化）
・× 雨が降っているも構わず、出かけた
・✓ 雨も構わず、出かけた（不顧下雨出門）

◆ 變化補充
・語氣差異
・偏負面評價（自私／不顧他人）
・看到「不顧～硬要做」優先選
```

## ◎ を問わず／は問わず  （〜をとわず）

```
◆ 語感例句結構
・範圍不限
・名詞＋を問わず／は問わず
・年齢を問わず、誰でも参加できる。（不論年齡都能參加）
・男女は問わず、この仕事に応募できる。（不分男女可應徵）

◆ 混淆比較
・に関係なく／にかかわらず
・に関係なく＝一般不限／にかかわらず＝條件對立也成立／を問わず＝列舉範圍不限
・天気に関係なく、試合は行う。（不受天氣影響）
　天気にかかわらず、試合は行う。（即使影響也照做）
・年齢を問わず、応募可能。（不限定年齡範圍）
　年齢に関係なく、応募可能。（單純不相關）

◆ 錯誤陷阱
・不能接句子（只接名詞）
・× 若いかどうかを問わず、参加できる
・✓ 年齢を問わず、参加できる（不論年齡可參加）

◆ 變化補充
・助詞差異
・を／は差異不影響意思（は偏對比）
・考試兩個都可選，重點看名詞接續
```

## ◎ はともかく（として）

```
◆ 語感例句結構
・先放一邊
・名詞＋はともかく（として）
・結果はともかく、まずやってみよう。（結果先不管先做）
・細かいことはともかくとして、方向性は合っている。（細節先放一邊）

◆ 混淆比較
・はさておき／は別として
・はさておき＝話題切換／は別として（はべつとして）＝條件排除／はともかく＝先不討論
・冗談はさておき、本題に入ろう。（轉話題）
　冗談はともかく、本題に入ろう。（先不管）
・料金は別として、このホテルは良い。（排除條件）
　料金はともかく、このホテルは良い。（先不討論）

◆ 錯誤陷阱
・不能接動詞句（需名詞化）
・× 行くかどうかはともかく、準備しよう
・✓ 行くかどうかのことはともかく、準備しよう（去不去先放一邊）

◆ 變化補充
・語氣差異
・ともかく＝口語常用／ともかくとして＝稍正式
・考試看到「先不談這點」優先選
```

## ○ に先立ち／に先立つ／に先立って  （にさきだち）

```
◆ 語感例句結構
・正式先做
・名詞＋に先立ち／に先立って／に先立つ＋名詞
・会議に先立ち、資料を配布します。（會議前先發資料）
・出発に先立って、注意事項を説明した。（出發前先說明）

◆ 混淆比較
・前に／にあたって
・前に＝單純時間先後／にあたって＝面臨重要時刻／に先立ち＝正式事前準備
・出発前に、準備をする。（單純之前）
　出発に先立って、準備をする。（正式事前動作）
・試験にあたって、注意事項を確認する。（面臨考試）
　試験に先立ち、説明が行われた。（正式流程）

◆ 錯誤陷阱
・不能接動詞（只接名詞）
・× 始めるに先立ち、説明する
・✓ 開始に先立ち、説明する（開始前先說明）

◆ 變化補充
・形態差異
・に先立つ＋名詞（修飾用）
・考試看到「典禮／會議前正式流程」優先選
```

## ◎ をきっかけに（して）／をきっかけとして

```
◆ 語感例句結構
・契機轉變
・名詞＋をきっかけに（して）／をきっかけとして
・留学をきっかけに、日本語を本気で勉強し始めた。（因留學開始認真學日文）
・失敗をきっかけとして、考え方が変わった。（因失敗改變想法）

◆ 混淆比較
・を契機に／から
・を契機に＝更正式書面／から＝單純起點／きっかけ＝帶轉變契機
・この事件をきっかけに、法律が変わった。（契機帶變化）
　この事件を契機に、法律が変わった。（更正式）
・大学に入ってから、生活が変わった。（只是時間起點）

◆ 錯誤陷阱
・不能接句子（需名詞）
・× 彼に会ったをきっかけに、考えが変わった
・✓ 彼との出会いをきっかけに、考えが変わった（以相遇為契機）

◆ 變化補充
・語氣差異
・して可省略／として偏正式
・考試看到「某事件→之後改變」優先選
```

## △ を契機（けいき）に（して）／を契機として

```
◆ 語感例句結構
・重大契機
・名詞＋を契機に（して）／を契機として
・この事件を契機に、制度が大きく変わった。（因這起事件制度大改）
・独立を契機として、会社は成長し始めた。（以獨立為契機開始成長）

◆ 混淆比較
・をきっかけに／から
・きっかけ＝日常轉變／契機＝重大正式／から＝單純起點
・失敗をきっかけに、考え方が変わった。（日常契機）
　失敗を契機に、考え方が変わった。（較正式重大）
・社会人になってから、生活が変わった。（只是起點）

◆ 錯誤陷阱
・不能接句子（需名詞化）
・× 会社を辞めたを契機に、起業した
・✓ 会社を辞めたことを契機に、起業した（以離職為契機）

◆ 變化補充
・語氣差異
・して可省略／として更書面正式
・考試看到「制度變化／重大事件」優先選
```

# 時間・時機・變化

## ◎ おり（に／は／には／から）

```
◆ 語感例句結構（必出）
・某時機觸發
・Vる／Vた／Nの＋おりに／は／には／から
・日本に行ったおりに、彼に会いました。
　（去日本的時候，見了他）
・困ったおりには、遠慮なく連絡してください。
　（遇到困難時，請不要客氣聯絡我）

◆ 混淆比較（必出｜選項核心）
・とき（に）／際（に）
・正式＋契機：おり＝偏正式且強調時機；とき＝一般；際＝更正式書面
・日本に行ったとき、彼に会った。
　（去日本時見了他）vs 日本に行ったおりに、彼に会いました。
　（較正式：在那個時機見面）
・ご来店の際にお声がけください。
　（更正式）vs ご来店のおりにはお声がけください。
　（正式但較柔）

◆ 錯誤陷阱（必出｜考試重點）
・助詞誤用：から＝起點原因，不是單純「時」
・× 日本に行ったおりに、日本語が上手になった。
・✓ 日本に行ったおりから、日本語が上手になった。
　（從那時起變得更好）

◆ 變化補充（可省略）
・助詞差異
・に＝單次時點／には＝條件提醒／から＝起點變化
・看到「通知／提醒語氣」優先には
```

## ◎ にあたって／にあたり

```
◆ 語感例句結構（必出）
・重要時機準備
・N／Vる＋にあたって／にあたり
・新しいプロジェクトを始めるにあたって、計画を見直した。
　（在開始新專案之際，重新檢討了計畫）
・留学にあたり、必要な書類を全部準備した。
　（在留學前，準備好所有必要文件）

◆ 混淆比較（必出｜選項核心）
・際に／前に／に際して
・是否準備：にあたって＝事前準備；前に＝單純時間；際に＝當下時點
・出発前に荷物を確認する。
　（出發前確認行李）vs 出発にあたって荷物を確認する。
　（為了出發而事前確認）
・ご来店の際にお声がけください。
　（來店時）vs 開店にあたって、スタッフを募集する。
　（為開店做準備）

◆ 錯誤陷阱（必出｜考試重點）
・小事誤用：不可用於日常瑣事
・× 昼ご飯を食べるにあたって手を洗う。
・✓ 昼ご飯を食べる前に手を洗う。（吃飯前洗手）

◆ 變化補充（可省略）
・形式差異
・にあたり＝較書面簡潔
・考試不影響選項，視語氣即可
```

## ◎ に際して／に際し／に際しての

```
◆ 語感例句結構（必出）
・正式關鍵時點
・N／Vる＋に際して／に際し／に際しての＋N
・入社に際して、必要な書類を提出してください。（ていしゅつしてください）
　（在入職之際，請提交必要文件）
・試験に際し、身分証の提示が必要です。
　（在考試之際，需要出示身分證）

◆ 混淆比較（必出｜選項核心）
・にあたって／ときに／前に
・正式＋當下時點：に際して＝關鍵時刻；にあたって＝事前準備
・開店にあたって、スタッフを募集する。
　（為開店做準備）vs 開店に際して、挨拶を行う。
　（在開店當下進行致詞）
・日本に行ったとき、彼に会った。
　（一般時間）vs 日本に行くに際して、注意事項を確認した。
　（重要時點前後需注意）

◆ 錯誤陷阱（必出｜考試重點）
・日常誤用：不可用於輕鬆小事
・× ご飯を食べるに際して手を洗う。
・✓ ご飯を食べる前に手を洗う。（吃飯前洗手）

◆ 變化補充（可省略）
・形式差異
・に際し＝更書面簡潔／に際しての＋N＝修飾名詞
・看到公告／規則／正式文幾乎直接選
```

## △ にて／でもって

```
◆ 語感例句結構（必出）
・手段場所正式
・N＋にて／でもって
・会議は三階の会議室にて行われます。
　（會議將在三樓會議室舉行）
・本日をもちまして、このサービスは終了とさせていただきます。
　（今日起，本服務將結束）
  本日にて、このサービスは終了とさせていただきます。
		にて → 正式的「で」
		をもちまして → 商業公告固定句型（結束、截止）

◆ 混淆比較（必出｜選項核心）
・で／によって／をもって
・正式替代：にて＝で（場所/手段）；でもって＝をもって（手段/期限）
・会議は会議室で行われる。
　（一般）vs 会議は会議室にて行われます。
　（正式）
・本日をもって終了します。
　（以今日為界）vs 本日でもって終了します。
　（口語但同義）

◆ 錯誤陷阱（必出｜考試重點）
・誤當一般口語使用
・× コンビニにて買ったよ。
・✓ コンビニで買ったよ。（日常用で）

◆ 變化補充（可省略）
・語域差異
・にて＝書面/公告；でもって＝口語帶正式感
・看到公告語氣優先にて／期限句優先（を）もって
```

## △ か～ないかのうちに

```
◆ 語感例句結構（必出）
・剛發生立刻
・Vた＋か～ないかのうちに
・ドアが閉まったか閉まらないかのうちに、彼はもう出ていった。
　（門才剛關上，他就已經離開了）
・ベルが鳴ったか鳴らないかのうちに、学生たちは教室に入ってきた。
　（鈴聲才剛響，學生就湧進教室）

◆ 混淆比較（必出｜選項核心）
・とたん（に）／やいなや／すぐに
・瞬間差：か～ないか＝幾乎同時；とたん＝前後明顯；やいなや＝書面急速
・ドアを開けたとたん、猫が飛び出した。
　（剛一開門就）vs ドアが開いたか開かないかのうちに、猫が飛び出した。
　（幾乎同時瞬間）
・席に着くやいなや、電話が鳴った。
　（書面急速）vs 席に着いたか着かないかのうちに、電話が鳴った。
　（口語自然瞬間）

◆ 錯誤陷阱（必出｜考試重點）
・動詞限制：不可用於持續／狀態動詞
・× 雨が降っているか降っていないかのうちに出かけた。
・✓ 雨が降ったか降らないかのうちに出かけた。（剛要下就出門）

◆ 變化補充（可省略）
・語氣差異
・偏口語敘述瞬間連續動作
・看到「幾乎同時」優先選
```

## ◎ ないうちに

```
◆ 語感例句結構（必出）
・趁未發生前
・Vない＋うちに／Aくない＋うちに／Nでない＋うちに
・忘れないうちに、メモしておこう。
　（趁還沒忘之前，先記下來）
・雨が降らないうちに、帰ったほうがいい。
　（趁還沒下雨前，最好先回去）

◆ 混淆比較（必出｜選項核心）
・うちに（肯定）／前に
・是否變化：ないうちに＝避免變化；うちに＝自然變化
・暗くならないうちに帰ろう。
　（趁還沒變暗前回家）vs 暗いうちに帰ろう。
　（趁還是暗的時候回家）
・寝る前に歯を磨く。
　（單純順序）vs 忘れないうちにメモする。
　（避免忘記）

◆ 錯誤陷阱（必出｜考試重點）
・誤用肯定：不可用於已發生或自然變化
・× 桜が咲くうちに見に行こう。（想表「還沒開前」卻用錯）
・✓ 桜が咲かないうちに見に行こう。（趁還沒開之前）

◆ 變化補充（可省略）
・語意重點
・常搭配意志形（～よう／～ておく）
・看到「避免發生」幾乎直接選
```

## ◎ しだい

```
◆ 語感例句結構（必出）
・一完成立刻
・Vます＋しだい／Nしだい
・準備ができしだい、出発します。
　（準備一完成就出發）
・詳細（しょうさい）が分かりしだい、ご連絡いたします。
　（詳細一知道就聯絡您）

◆ 混淆比較（必出｜選項核心）
・たら／と／次第で
・立即性：しだい＝完成瞬間立刻；たら＝條件後；次第で＝取決於
・準備ができたら出発する。
　（準備好再出發）vs 準備ができしだい出発する。
　（剛好就立刻出發）
・結果は努力次第で変わる。
　（取決於努力）vs 詳細が分かりしだい連絡する。
　（一知道就立刻）

◆ 錯誤陷阱（必出｜考試重點）
・接續錯：不可用辭書形（る形）
・× 準備ができるしだい出発します。
・✓ 準備ができしだい出発します。（完成後立刻）

◆ 變化補充（可省略）
・名詞用法
・Nしだい＝視情況而定（別義）
・看到「一完成就」優先選（注意別和次第で混）
```

# 原因・理由・結果

## **◎ あまり（に）**

```
◆ 語感例句結構（必出）
・語感：過頭導致結果
・結構：あまり（に）＋普通形（多接結果句）
・嬉しさのあまり、泣いてしまった。（太開心結果哭了）
・緊張のあまり、何も話せなかった。（太緊張結果說不出話）

◆ 混淆比較（必出｜選項核心）
・對手文法：すぎて／ので
・一句話差異：あまり（に）＝情緒或狀態過頭→自然結果／すぎて＝程度過高（較客觀）／ので＝一般原因
・疲れすぎて寝た。（太累就睡）
　疲れのあまり寝てしまった。（累過頭不由自主）
・雨なので中止。（單純原因）
　驚きのあまり声が出ない。（情緒過頭）

◆ 錯誤陷阱（必出｜考試重點）
・NG用法說明：不適合「可控制行動」
・嬉しさのあまり、プレゼントを買おう。（×）
・嬉しくてプレゼントを買った。（✓ 自己決定行動）

◆ 變化補充（可省略）
・分類：原因結果
・影響答案的理由：多接「非自願結果」
・實戰建議：看到「太…結果不由自主」→選
```

## **◎ 以上（は）**

```
◆ 語感例句結構（必出）
・語感：既然到這程度
・結構：V辞書形＋以上（は）
・約束した以上は、守らないといけない。（既然答應了就得遵守）
・やると決めた以上、最後までやる。（既然決定做就做到最後）

◆ 混淆比較（必出｜選項核心）
・對手文法：からには／以上（數量）
・一句話差異：以上（は）＝已成事實→責任／からには＝強主觀決心／以上＝數量比較
・学生である以上、ルールを守るべきだ。（基於立場）
　学生なんだから、守るべきだ。（語氣較主觀）

◆ 錯誤陷阱（必出｜考試重點）
・NG用法說明：不能接「未確定事項」

◆ 變化補充（可省略）

・實戰建議：看到「既然已經…就必須」→選
```


## **◎ からといって**

```
◆ 語感例句結構（必出）

・結構：普通形＋からといって（後接否定／反駁）

◆ 變化補充（可省略）
```

## **◎ しだいです**

```
◆ 語感例句結構（必出）
・語感：結果原因說明
・結構：普通形＋しだいです（多為正式說明）
・本日の会議は中止となりました。天候が悪化したしだいです。（因天氣惡化而中止）

◆ 混淆比較（必出｜選項核心）

◆ 錯誤陷阱（必出｜考試重點）
・NG用法說明：不能用於口語日常
・昨日は疲れたしだいです。（×）
・昨日は疲れたから休んだ。（✓ 自然口語）

◆ 變化補充（可省略）

```

```
◆ 混淆比較
・ないことには／なければ／なくては
・ないことには＝「前件不成立，後面根本無法談」；なければ・なくては＝單純必要條件
・予約がないことには、入れません。／沒有預約就根本進不去
・予約がなければ、入れません。／如果沒有預約，就不能進去

◆ 錯誤陷阱
・後面通常接「わからない・できない・始まらない」這種不能成立的結果，不能亂接命令
・× 早く行かないことには、急いでください。
・✓ 早く行かないことには、間に合いません。／不早點去的話，就趕不上了

◆ 變化補充
・常見固定感
・這個文法帶有「先決條件卡住」的語氣，比單純條件更重
・考試看到後項是「無法判斷／無法進行／無法成立」，優先想到ないことには
```

## **○ を～として／を～とする／を～とした**

```
◆ 語感例句結構
・定義當作用
・N＋を～として／とする／とした＋N
・彼をリーダーとして、チームをまとめた。／以他為隊長，統整了團隊
・これは冗談とした発言だった。／這是當作玩笑說的發言

◆ 混淆比較
・として／とする／とした
・として＝身分/立場；とする＝主觀設定；とした＝修飾名詞用（已設定）
・先生として話します。／以老師的立場說
・彼を犯人とする説がある。／有把他當犯人的說法

◆ 錯誤陷阱
・とした後面一定要接名詞，不能直接當句尾用
・× 彼をリーダーとした。
・✓ 彼をリーダーとしたチームを作った。／組了一個以他為隊長的隊伍

◆ 變化補充
・語氣差異
・とする偏書面、帶「認定／假設」感；として偏日常
・考試看到「說法／立場／身分」，優先判斷三者語感
```

## **△ も～なら～も／も～ば～も**

```
◆ 語感例句結構
・雙項並列評價
・N／形容詞＋も～なら～も／も～ば～も
・この店は味もいいなら、値段も安い。／這家店不只好吃，價格也便宜
・彼は仕事もできれば、気配りもできる。／他不但工作能力強，也很會顧慮他人

◆ 混淆比較
・も～なら～も／も～し～も
・も～なら～も＝偏評價、帶對比或意外；も～し～も＝單純列舉理由
・あの人は頭もいいなら、運動もできる。／那人不只聰明，連運動也強（帶評價）
・あの人は頭もいいし、運動もできる。／那人聰明，而且會運動（單純列舉）

◆ 錯誤陷阱
・ば形要接在動詞或イ形容詞，名詞不能直接用「なら」亂接
・× 彼は学生もなら、社会人もだ。
・✓ 彼は学生でもあれば、社会人でもある。／他既是學生，也是社會人士

◆ 變化補充
・なら vs ば
・なら多口語；ば稍正式，但語感差不影響答案
・考試重點在「評價並列」vs「單純列舉」
```

## **○ ものなら**

```
◆ 語感例句結構
・假設難成強警告
・V意向形＋ものなら
・そんなこと言おうものなら、絶対怒られる。／要是敢說那種話，一定會被罵
・今サボろうものなら、あとでめっちゃ後悔するよ。／現在敢偷懶，之後一定超後悔

◆ 混淆比較
・ものなら／たら／なら
・ものなら＝「幾乎不會發生＋一發生就慘」；たら・なら＝普通假設
・そんなことしたら、怒られるよ。／那樣做的話會被罵（普通提醒）
・そんなことしようものなら、即クビだ。／要是敢那樣做，立刻被開除（強警告）

◆ 錯誤陷阱
・一定要接「意向形（～よう）」不能用普通形
・× 行くものなら、危ない。
・✓ 行こうものなら、危ない。／要是敢去就危險

◆ 變化補充
・語氣特徵
・常搭配「絶対／すぐ／とんでもないことになる」強化後果
・考試看到「敢…就完了」這種語感，優先選ものなら
```

## **◎ ながら（も）**

```
◆ 語感例句結構
・逆接同時矛盾
・Vます形／形容詞／名詞＋ながら（も）
・知っていながら、何も言わなかった。／明明知道，卻什麼都沒說
・残念ながら、今回は参加できません。／很遺憾，這次無法參加

◆ 混淆比較
・ながら（も）／のに／けど
・ながら（も）＝同一主體「內在矛盾」；のに・けど＝單純逆接
・彼は学生ながら、会社を経営している。／他雖是學生，卻在經營公司（身份反差）
・彼は学生なのに、会社を経営している。／他明明是學生，卻在經營公司（事實反差）

◆ 錯誤陷阱
・主語需一致，不能前後換人
・× 彼は知りながら、私は何も言わなかった。
・✓ 彼は知りながら、何も言わなかった。／他明明知道卻沒說

◆ 變化補充
・固定用法
・「残念ながら／不思議ながら」常當句首副詞
・考試看到「明明…卻…（同一人）」優先ながら
```

## **◎ ものの**

```
◆ 語感例句結構
・讓步已成事實
・普通形＋ものの
・やってみたものの、うまくいかなかった。／雖然試了，但沒有成功
・彼は来ると言ったものの、まだ来ていない。／他說會來，但還沒到

◆ 混淆比較
・ものの／けど／とはいえ
・ものの＝承認前項「已發生」但結果不如預期；けど＝一般轉折；とはいえ＝承認＋部分讓步評價
・勉強したものの、点数は伸びなかった。／雖然讀了，但分數沒提升（結果落差）
・勉強したけど、点数は伸びなかった。／讀了，但分數沒提升（普通轉折）

◆ 錯誤陷阱
・後項多接負面或未達預期，不能接正面順結果
・× 努力したものの、合格できた。
・✓ 努力したものの、合格できなかった。／雖然努力了，但沒考上

◆ 變化補充
・語氣特徵
・偏書面、敘述事實＋落差，不帶強情緒
・考試看到「做了但沒達到預期」優先ものの
```

## **△ やら～やら**

```
◆ 語感例句結構
・雜亂多重抱怨
・N／普通形＋やら～やら
・仕事やら家事やらで、今日は全然休めなかった。／工作啦家事啦一堆，今天完全沒休息
・嬉しいやら恥ずかしいやらで、何を言えばいいかわからない。／又開心又害羞，不知道該說什麼

◆ 混淆比較
・やら～やら／とか～とか
・やら～やら＝情緒混亂或雜事一堆；とか～とか＝單純列舉
・雨やら風やらで、外に出られない。／又下雨又颳風，出不了門（帶困擾）
・雨とか風とかで、外に出られない。／雨啊風啊，出不了門（普通列舉）

◆ 錯誤陷阱
・不可只列單一項目，必須至少兩個以上
・× 仕事やらで忙しい。
・✓ 仕事やら会議やらで忙しい。／工作啦會議啦很多事，很忙

◆ 變化補充
・語氣特徵
・常帶負面或混亂情緒，但也可用於情緒交錯（喜悅＋害羞）
・考試看到「一堆…搞得…」或「又…又…心情混亂」優先やら～やら
```

## **○ にしろ**

```
◆ 語感例句結構
・讓步皆成立
・N／普通形＋にしろ（～にしろ）
・行くにしろ、行かないにしろ、早く決めて。／去也好不去也好，快點決定
・本当だにしろ、嘘だにしろ、確認は必要だ。／不管是真的還是假的，都需要確認

◆ 混淆比較
・にしろ／にせよ／ても
・にしろ＝列兩種情況都成立；にせよ偏書面；ても＝單一讓步
・行くにしろ行かないにしろ、連絡して。／不管去不去，都要聯絡
・行っても、連絡して。／就算去了，也要聯絡（單一條件）

◆ 錯誤陷阱
・常用成對（AにしろBにしろ），不能只用單邊卻沒有對應項
・× 行くにしろ、連絡して。
・✓ 行くにしろ行かないにしろ、連絡して。／不管去或不去，都要聯絡

◆ 變化補充
・語氣差異
・にせよ較正式，但考試多可互換
・看到「不管A還是B都一樣」優先にしろ
```

## **◎ にせよ／にもせよ**

```
◆ 語感例句結構
・讓步假設成立
・普通形＋にせよ／にもせよ
・本当だにせよ、確認は必要だ。／就算是真的，也需要確認
・行くにもせよ、時間をちゃんと決めよう。／就算要去，也要把時間訂好

◆ 混淆比較
・にせよ／にしろ／ても
・にせよ＝假設讓步（偏書面）；にしろ＝列舉兩種都成立；ても＝一般讓步
・本当だにせよ、慎重に判断すべきだ。／即使是真的，也該慎重判斷
・本当だにしろ嘘だにしろ、確認が必要だ。／不管真或假，都需要確認

◆ 錯誤陷阱
・にもせよ多用在「要做某事」前，不能亂接已完成結果
・× 行ったにもせよ、問題は解決した。
・✓ 行くにもせよ、準備が必要だ。／就算要去，也需要準備

◆ 變化補充
・語氣差異
・にもせよ＝帶「即使如此也要做某事」的前提感
・考試看到「就算…也要先…」優先にもせよ
```

# 附帶・追加・排除・變化

## **△ こと（も）なく**

```
◆ 語感例句結構
・語感：完全沒做
・結構：V辞書形＋こと（も）なく
・彼は謝ることもなく、そのまま帰った。（他完全沒道歉就直接回去了）
・彼女は一度も休むこともなく、毎日来ている。（她一次也沒請假，每天都來）

◆ 混淆比較
・ことなく／ないで／ずに
・一句話差異：ことなく＝完全沒發生（強調0次）／ないで・ずに＝單次沒做就做後項
・彼は何も言うことなく出ていった。（完全沒說任何話就離開）
　彼は何も言わないで出ていった。（沒說話就離開，但不強調0次）
・彼女は一度もミスすることなく終えた。（完全0失誤）
　彼女はミスせずに終えた。（沒犯錯完成，但語氣較中性）

◆ 錯誤陷阱
・不能接「た形」或已發生次數語境
・× 彼は謝ったこともなく帰った
・✓ 彼は謝ることもなく帰った（強調完全沒做）

◆ 變化補充
・こともなく（強調）
・加入「も」→加強「連一次都沒有」
・考試看到「一度も～ない」語境優先選
```

## **○ をぬきにして（は／も）／はぬきにして**

```
◆ 語感例句結構
・語感：先排除再談
・結構：N＋をぬきにして（は／も）／N＋はぬきにして
・細かい話はぬきにして、本題に入ろう。（先不談細節，進入正題吧）
・冗談をぬきにしても、それはちょっと危ないよ。（就算不開玩笑，那還是有點危險）

◆ 混淆比較
・抜きで／抜きにして／なしで
・一句話差異：ぬきにして＝刻意排除再談（多用轉話題）／抜きで・なしで＝單純沒有
・前置きはぬきにして、結論だけ言うね。（刻意略過開場白）
　前置きなしで結論だけ言うね。（只是沒有開場白）
・感情をぬきにして考えよう。（排除情緒來討論）
　感情抜きで考えよう。（不帶情緒思考）

◆ 錯誤陷阱
・不能接動詞句（限定名詞）
・× 説明するをぬきにして始める
・✓ 説明はぬきにして始める（先不說明就開始）

◆ 變化補充
・をぬきにしても
・「も」→就算排除也成立（讓步）
・看到「即使不考慮～也…」優先選
```

## **◎ ぬきで／ぬきに／ぬきの**

```
◆ 語感例句結構
・語感：單純沒加
・結構：N＋ぬきで／ぬきに／ぬきの＋N
・今日は砂糖ぬきでコーヒー飲む。（今天咖啡不加糖）
・冗談ぬきの話だけど、ほんとに大事なことだよ。（不是開玩笑，是真的很重要）

◆ 混淆比較
・ぬきで／ぬきにして／なしで
・一句話差異：ぬきで＝單純沒有（狀態）／ぬきにして＝刻意排除來討論／なしで＝客觀缺少
・今日はお酒ぬきでいこう。（單純不喝酒）
　今日はお酒をぬきにして話そう。（先不談喝酒，來談別的）
・予約なしで入れますか。（沒有預約）
　予約ぬきで入れますか。（語氣較主觀＝不帶預約）

◆ 錯誤陷阱
・ぬきに不可直接接動詞句
・× 冗談ぬきに話す
・✓ 冗談ぬきで話す（以「不開玩笑」的狀態說）

◆ 變化補充
・ぬきの＋名詞
・修飾後項→變成名詞限定
・看到名詞後接→優先選ぬきの
```

## **◎ 上（に）**

```
◆ 語感例句結構
・語感：再加一層
・結構：普通形＋上（に）
・この店は安い上に、めっちゃ美味しい。（這家店不只便宜，還很好吃）
・彼は仕事が速い上、ミスも少ない。（他不只做事快，錯誤也少）

◆ 混淆比較
・上に／し／うえで
・一句話差異：上に＝正面累加（加分）／し＝並列列舉（較口語）／うえで＝前提後再做
・彼は優しい上に、面白い。（加分疊加）
　彼は優しいし、面白い。（單純列舉）
・考えた上で決めた。（先思考再決定）
　考えた上に決めた。（×錯）

◆ 錯誤陷阱
・不可接負面累加語境
・× 遅刻した上に、怒られた（語感不自然）
・✓ 遅刻して、さらに怒られた（單純結果）

◆ 變化補充
・上（うえ）
・「に」可省略（書面）
・考試多數仍選「上に」
```

## **◎ ないかぎり**

```
◆ 語感例句結構
・語感：不發生就不變
・結構：Vない形＋かぎり
・雨が降らないかぎり、試合は行われる。（只要不下雨，比賽就會進行）
・特に問題がないかぎり、このままでいいよ。（沒有問題的話，就照這樣）

◆ 混淆比較
・ないかぎり／なければ／ない限りは
・一句話差異：ないかぎり＝條件維持（不變）／なければ＝單純假設／ない限りは＝強調條件界線

・許可がない限りは入れません。（界線強）
　許可がなければ入れません。（普通）

◆ 錯誤陷阱
・不可用於一次性已發生事件
・× 昨日雨が降らないかぎり試合した
・✓ 雨が降らなかったので試合した（已發生用因果）

◆ 變化補充
・ない限り（は）
・語氣較強
・規則／限制句常考
```

## **○ に相違ない**

```
◆ 語感例句結構
```

```
◆ 混淆比較
・ことには／にかけては／については
・一句話差異：ことには＝對某點特別敏感／在意／にかけては＝該領域最強／については＝單純主題
・時間のことには厳しい。（特別在意）
　時間にかけては誰にも負けない。（最強）
・時間について話す。（主題）

◆ 錯誤陷阱
・多接「性格傾向」而非客觀事實
・× 天気のことには晴れだ
・✓ 健康のことには気を使う（在意）

◆ 變化補充
・は（強調）
・常見「ことにはうるさい」
・看到「對某點特別在意」優先選
```

## **◎ はまだしも／ならまだしも**

```
◆ 語感例句結構
・語感：A還好B不行
・結構：N／普通形＋はまだしも／ならまだしも
・一人ならまだしも、二人は無理だ。（一個人還行，兩個就不行）
・少し高いのはまだしも、こんなに高いのは無理。（稍微貴還好，這麼貴不行）

◆ 混淆比較
・はまだしも／どころか／ならまだしも
・一句話差異：はまだしも＝A可接受B不可／どころか＝反而更…／ならまだしも＝條件限定對比
・子どもはまだしも、大人はダメだ。（A可B不可）
　安くなるどころか高くなった。（反而）
・一回ならまだしも何回もは無理。（條件限定）

◆ 錯誤陷阱
・需有「程度差」或「條件差」
・× りんごはまだしも食べる（無對比）
・✓ 安いのはまだしも高いのは無理（對比）

◆ 變化補充
・ならまだしも（條件）
・語氣更明確
・看到「還好…但…不行」優先選
```

## **◎ にわたって／にわたり**

```
◆ 語感例句結構
・語感：長時間範圍
・結構：N＋にわたって／にわたり／にわたる＋N
・三日間にわたって会議が行われた。（會議持續三天）

◆ 混淆比較
・にわたって／にかけて／を通じて
・一句話差異：にわたって＝整段期間/範圍全涵蓋／にかけて＝從A到B之間／を通じて＝透過媒介
・一週間にわたって雨が降った。（整段）
　東京から大阪にかけて雨。（範圍區間）
・SNSを通じて広まった。（透過）

◆ 錯誤陷阱
・多接「長期間／廣範圍」
・× 一瞬にわたって光った
・✓ 数年間にわたって研究した（需長）

◆ 變化補充
・にわたる＋名詞
・名詞修飾
・看到「長達／遍及」優先選
```

## **◎ に沿って／に沿い／に沿う**

```
◆ 語感例句結構
・語感：照方針走
・結構：N＋に沿って／に沿い／に沿う＋N
・計画に沿って進めてください。（請照計畫進行）
・川に沿う道を歩いた。（走在沿著河的路上）

◆ 混淆比較
・に沿って／に従って／に応じて
・一句話差異：に沿って＝依方針/路線貼著走／に従って＝遵守規則／に応じて＝依情況調整
・方針に沿って進める。（貼著方針）
　ルールに従って行動する。（遵守）
・状況に応じて変える。（調整）

◆ 錯誤陷阱
・需有「基準／路線」概念
・× 気分に沿って行く
・✓ ガイドラインに沿って実施する（有基準）

◆ 變化補充
・に沿う＋名詞
・名詞修飾
・考試多考接續
```

## **◎ によると／によれば**

```
◆ 語感例句結構
・語感：根據消息
・結構：N＋によると／によれば
・ニュースによると、明日は雨らしい。（根據新聞，明天好像會下雨）
・先生によれば、この問題は重要だそうだ。（老師說這題很重要）

◆ 混淆比較
・によると／では／によって
・一句話差異：によると＝情報來源＋傳聞／では＝說話者立場／によって＝原因或依據
・新聞によると事故があった。（來源）
　私では無理だ。（立場）
・人によって違う。（依人）

◆ 錯誤陷阱
・後項需接傳聞（らしい／そうだ等）
・× ニュースによると雨だ
・✓ ニュースによると雨らしい（需傳聞）

◆ 變化補充
・によれば（較書面）
・語意相同
・考試可互換
```

## **◎ というより**

```
◆ 語感例句結構
・語感：與其說不如說
・結構：普通形＋というより（は）
・彼は優しいというより、ただおとなしいだけだ。（與其說溫柔，不如說只是安靜）
・これは仕事というより趣味だ。（與其說是工作，不如說是興趣）

◆ 混淆比較
・というより／むしろ／というか
・一句話差異：というより＝修正前述表達／むしろ＝反轉偏向／というか＝口語補充修正
・疲れたというより眠い。（修正）
　疲れた。むしろ限界だ。（反轉）
・彼は優しいというか、甘い。（口語補充）

◆ 錯誤陷阱
・需有「前項→修正」關係
・× これは本だというより読む（結構錯）
・✓ 本というより雑誌だ（修正分類）

◆ 變化補充
・というよりは
・語氣較完整
・看到「不如說」優先選
```
