# 動詞・形容詞・名詞的變化規則＋敬語變化

---

### **一類動詞**的活用變化

| 活用形 | 變化規則 | 例：書く（かく） | 例：飲む（のむ） |
| --- | --- | --- | --- |
| **受身形（被～）** | 「a」段 + れる | 書か + れる = **書かれる**（被寫） | 飲ま + れる = **飲まれる**（被喝） |
| **使役形（讓～做）** | 「a」段 + せる | 書か + せる = **書かせる**（讓寫） | 飲ま + せる = **飲ませる**（讓喝） |
| **使役受身形（被迫做）** | a段 + せる（使役形）→ 去る + られる | 書かせ + られる = **書かせられる**（被迫寫） | 飲ませ + られる = **飲ませられる**（被迫喝） |
| **未然形（ない形）** | 「a」段 + ない | 書か + ない = **書かない**（不寫） | 飲ま + ない = **飲まない**（不喝） |
| **連用形（ます形）** | 「い」段 + ます | 書き + ます = **書きます**（寫） | 飲み + ます = **飲みます**（喝） |
| **終止形（原形）** | 原形不變 | **書く**（寫） | **飲む**（喝） |
| **否定意志形（不要～）** | 終止形 + まい
※ 現代日語中較少使用，多見於書面語、小說或正式語氣 | 書く + まい = **書くまい**（不要寫） | 飲む + まい = **飲むまい**（不要喝） |
| **連體形（名詞修飾）** | 原形不變 | **書く** 本（寫書用的本／用來寫的本） | **飲む** 水（要喝的水） |
| **可能形（能夠～）** | 「え」段 + る | 書け + る = **書ける**（能寫） | 飲め + る = **飲める**（能喝） |
| **假定形（條件形）** | 「え」段 + ば | 書け + ば = **書けば**（如果寫） | 飲め + ば = **飲めば**（如果喝） |
| **命令形（命令句）** | 「え」段（書け・飲め） | 書け！= **書け！**（寫吧！） | 飲め！= **飲め！**（喝吧！） |
| **意志形（～よう）** | 「お」段 + う | 書こ + う = **書こう**（來寫吧） | 飲も + う = **飲もう**（來喝吧） |
| **て形（進行式、請求）** | 依據不同結尾變化（見下表） | 書い + て = **書いて**（正在寫） | 飲ん + で = **飲んで**（正在喝） |
| **た形（過去式）** | 依據不同結尾變化（見下表） | 書い + た = **書いた**（寫了） | 飲ん + だ = **飲んだ**（喝了） |

**た形 / て形**

| **結尾類型** | **例詞**（動詞） | **て形**（進行式、請求） | **た形**（過去式） |
| --- | --- | --- | --- |
| **く
→ いて** | 書く（かく，寫） | 書いて | 書いた |
| **ぐ
→ いで** | 泳ぐ（およぐ，游泳） | 泳いで | 泳いだ |
| **す
→ して** | 話す（はなす，說話） | 話して | 話した |
| **ぬ、ぶ、む
→ んで** | 飲む（のむ，喝） | 飲んで | 飲んだ |
|  | 遊ぶ（あそぶ，玩） | 遊んで | 遊んだ |
|  | 死ぬ（しぬ，死） | 死んで | 死んだ |
| **る、う、つ
→ って** | 帰る（かえる，回家） | 帰って | 帰った |
|  | 立つ（たつ，站立） | 立って | 立った |
|  | 待つ（まつ，等待） | 待って | 待った |
- く → **きます**
- ぐ → **ぎます**
- す → **します**
- ぬ・ぶ・む → **に／び／み ます**
- る・う・つ → **り／い／ち ます**

### **二類動詞**的活用變化

| **活用形** | **變化規則** | **例：食べる（たべる）** | **例：見る（みる）** |
| --- | --- | --- | --- |
| **使役形（讓～做）** | 去「る」+ させる | **食べさせる**（讓吃） | **見させる**（讓看） |
| **受身形（被～）** | 去「る」+ られる | **食べられる**（被吃） | **見られる**（被看） |
| **使役受身形（被迫做）** | 去「る」+ させられる | **食べさせられる**（被迫吃） | **見させられる**（被迫看） |
| **未然形（ない形）** | 去「る」+ ない | **食べない**（不吃） | **見ない**（不看） |
| **連用形（ます形）** | 去「る」+ ます | **食べます**（吃） | **見ます**（看） |
| **終止形（原形）** | 保持不變 | **食べる**（吃） | **見る**（看） |
| **否定意志形（不要～）** | 原形 + まい | **食べまい**（不要吃） | **見まい**（不要看） |
| **連體形（名詞修飾）** | 保持不變 | **食べる** もの（要吃的東西） | **見る** 映画（要看的電影） |
| **可能形（能夠～）** | 去「る」+ られる。             ※ 口語中常省略「ら」 | **食べられる**（能吃） | **見られる**（能看） |
| **假定形（條件形）** | 去「る」+ れば | **食べれば**（如果吃） | **見れば**（如果看） |
| **命令形（命令句）** | 去「る」+ ろ | **食べろ！**（吃吧！） | **見ろ！**（看吧！） |
| **意志形（～よう）** | 去「る」+ よう | **食べよう**（來吃吧） | **見よう**（來看吧） |
| **て形（進行式、請求）** | 去「る」+ て | **食べて**（正在吃） | **見て**（正在看） |
| **た形（過去式）** | 去「る」+ た | **食べた**（吃了） | **見た**（看了） |

た形 / て形

- **て形 → 去掉「る」，加上「て」**
- **た形 → 去掉「る」，加上「た」**

| **動詞** | **て形（請求、進行式）** | **た形（過去式）** |
| --- | --- | --- |
| 食べる（たべる，吃） | 食べて | 食べた |
| 見る（みる，看） | 見て | 見た |
| 起きる（おきる，起床） | 起きて | 起きた |
| 寝る（ねる，睡覺） | 寝て | 寝た |
| 決める（きめる，決定） | 決めて | 決めた |

### **三類動詞**「する / くる」變化

| **活用形** | **する（做）** | **くる（來）** |
| --- | --- | --- |
| **未然形（否定）** | しない | こない |
| **使役形（讓～做）** | させる | こさせる |
| **受身形（被～）** | される | こられる |
| **使役受身形（被迫做）** | させられる | こさせられる |
| **連用形（ます形）** | します | きます |
| **終止形（原形）** | する | くる |
| **否定意志形（不要～）** | するまい | くるまい |
| **連體形（名詞修飾）** | する | くる |
| **可能形（能夠～）** | **できる**（特殊變化） | **こられる** |
| **假定形（條件形）** | すれば | くれば |
| **命令形（命令句）** | しろ／せよ | こい |
| **意志形（～よう）** | しよう | こよう |
| **て形（請求、進行）** | して | きて |
| **た形（過去式）** | した | きた |

**※ くる 的「受身形」與「可能形」形式相同，依語境判斷**

### **い形容詞**（以「い」結尾）

| **活用形** | **變化規則** | **例：高い（たかい，高的）** |
| --- | --- | --- |
| **肯定（現在）** | **原形** | **高い**（高的） |
| **否定（現在）** | 去「い」+ くない | **高くない**（不高的） |
| **肯定（過去）** | 去「い」+ かった | **高かった**（曾經高的） |
| **否定（過去）** | 去「い」+ くなかった | **高くなかった**（以前不高的） |
| **副詞形** | 去「い」+ く | **高く** 話す（說得大聲） |
| **條件形（假設）** | 去「い」+ ければ | **高ければ**（如果高的話） |

---

### **な形容詞（接「だ／な」的形容動詞）**

| **活用形** | **變化規則** | **例：元気（げんき，健康的）** |
| --- | --- | --- |
| **肯定（現在）** | **+ だ（普通形）** /
+ です（禮貌形） | **元気だ**（健康的） |
| **否定（現在）** | + ではない（普通形） / ではありません（禮貌形） | **元気ではない**（不健康） |
| **肯定（過去）** | + だった | **元気だった**（曾經健康） |
| **否定（過去）** | + ではなかった | **元気ではなかった**（以前不健康） |
| **副詞形** | だ形 → に（元気だ→元気に） | **元気に** 遊ぶ（健康地玩耍） |
| **條件形（假設）** | + なら | **元気なら**（如果健康的話） |

### **名詞**

| **活用形** | **變化規則** | **例：学生（がくせい，學生）** |
| --- | --- | --- |
| **肯定（現在）** | + だ（普通形） / です（禮貌形） | **学生だ**（是學生） |
| **否定（現在）** | + ではない（普通形） / ではありません（禮貌形） | **学生ではない**（不是學生） |
| **肯定（過去）** | + だった | **学生だった**（曾經是學生） |
| **否定（過去）** | + ではなかった | **学生ではなかった**（以前不是學生） |
| **條件形（假設）** | + なら | **学生なら**（如果是學生） |

### **尊敬語**（そんけいご） → 用於尊敬對方（對別人的動作）

用於提升對方的行為或身份，表達尊敬。

- 一般用 **「お + 動詞ます形 + になる」** 或 **「れる / られる」**
    - **（※ 多用於和語動詞，外來語與部分動詞不適用）**
- 一些動詞有特殊變化（例：行く → いらっしゃる）

| **普通動詞** | **尊敬語** |
| --- | --- |
| 言う（說） | **おっしゃる** |
| する（做） | **なさる** |
| 行く / 来る / いる（去 / 來 / 在） | **いらっしゃる / おいでになる** |
| いる（在） | **いらっしゃる** |
| 見る（看） | **ご覧になる（ごらんになる）** |
| 食べる / 飲む（吃 / 喝） | **召し上がる（めしあがる）** |
| 知っている（知道） | **ご存じだ（ごぞんじだ）** |
| くれる（給我） | **くださる** |
| 読む（讀） | **お読みになる** |
| 書く（寫） | **お書きになる** |
| 会う（見面） | **お会いになる** |
| 帰る（回家） | **お帰りになる** |

---

### **謙讓語**（けんじょうご） → 用於降低自己（對自己的動作）

用於降低自己，表達謙遜，強調對方的尊貴。

- 一般用 **「お + 動詞ます形 + する」** 或 **「させていただく」**
- 一些動詞有特殊變化（例：行く → 参る）

| **普通動詞** | **謙讓語** |
| --- | --- |
| 言う（說） | **申す（もうす）/ 申し上げる（もうしあげる）** |
| する（做） | **いたす** |
| 行く / 来る（去 / 來） | **参る（まいる）** |
| いる（在） | **おります** |
| 見る（看） | **拝見する（はいけんする）** |
| 食べる / 飲む（吃 / 喝） | **いただく** |
| 知っている（知道） | **存じている（ぞんじている）** |
| もらう（接受） | **いただく** |
| あげる（給） | **差し上げる（さしあげる）** |
| 読む（讀） | **お読みする
拝読する** |
| 書く（寫） | **お書きする
拝書する（較少）／書かせていただく** |
| 会う（見面） | **お目にかかる（おめにかかる）** |

---

### **丁寧語**（ていねいご） → 一般的禮貌語（用「です、ます」說話）

用於一般的禮貌表達，與尊敬語和謙讓語不同，沒有階級差異。

- 動詞用 **「ます形」**，名詞和形容詞用 **「です / ございます」**
- 用於日常生活，如與客戶、上司、長輩交談

| **普通語** | **丁寧語** |
| --- | --- |
| する（做） | **します** |
| 行く / 来る / いる（去 / 來 / 在） | **行きます / 来ます / います** |
| 見る（看） | **見ます** |
| 食べる / 飲む（吃 / 喝） | **食べます / 飲みます** |
| ある（有） | **ございます** |
| ない（沒有） | **ございません** |
| いい（好） | **よろしい** |
| そうです（是那樣） | **左様でございます（さようでございます）** |
