# 日文核心文法系統 N1

分層原則

| 層級 | 目標 | 學習方式 |
| --- | --- | --- |
| A 高頻閱讀核心 | 新聞、評論、正式文章反覆出現 | 能拆長句、判斷作者立場與語氣強弱 |
| B 書面辨識核心 | 硬書面、公告、論述、專欄與社會議題 | 以閱讀理解為主，少量正式輸出 |
| C N1+ 語感辨識 | 文語、固定句、低頻硬表現 | 先認得，不當日常聊天模板 |

標籤

日常高頻：日常或工作口語也可自然使用，但 N1 中比例不高。

書面高頻：文章、評論、報告、正式通知常見。

新聞常見：新聞、社會文章、評論常見。

偏低頻／閱讀辨識：N1 可見，但主動輸出優先度低。

N1 易混淆重點

やいなや / が早いか / なり / そばから：前三者都是剛…就…；そばから 多為剛做完又反覆被抵消。

かたがた / がてら / かたわら：かたがた 正式順便，がてら 日常順便，かたわら 長期並行。本檔未列為主軸時，仍需在主表辨識。

をよそに / をものともせず / を尻目に：をよそに 無視他人關心，をものともせず 不畏困難，を尻目に 不理會旁人反應而做。

にたえる / にたえない / に足る：にたえる 值得或經得起，にたえない 可表不忍或不勝，に足る 是值得、足以。

ゆえに / こととて / とあって：ゆえに 書面原因，こととて 常帶歉意，とあって 表特殊情況造成結果。

ないまでも / までもない / べくもない：ないまでも 是即使不到…，までもない 是不必…，べくもない 是不可能。

をおいて / ならでは / にとどまらず：をおいて 是非…莫屬，ならでは 是只有…才有的特色，にとどまらず 是不止於某範圍。

いかんにかかわらず / いかんでは / いかんによっては：前者是不管結果如何；後兩者是視情況可能改變。

たるもの / ともあろうものが / におかれましては：都和身份有關；たるもの 講責任，ともあろうものが 批評身份不相稱，におかれましては 是敬語稱呼。

ごとし / べし / べからず / まじき：文語或標語感表現，N1 要能讀懂即可，不建議日常大量輸出。
